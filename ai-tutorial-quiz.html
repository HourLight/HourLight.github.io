用 AI 做互動小工具
讓客人在你的網頁玩到不想走

你的 IG 作品很多人按讚，但沒人預約。
因為按讚只是路過，互動才是停留。

馥靈之鑰美業 AI 學苑｜王逸君

為什麼需要互動工具？

你的 IG 每張圖都有一兩百讚，但轉換成預約的有幾個？可能一個月就那麼兩三個。問題與其說你的作品不好，不如說客人滑過去就忘了。

互動工具的意思是，讓客人在你的網頁上「做一件事」。選一個顏色看適不適合自己、回答幾個問題看自己適合什麼風格的美甲、輸入膚色看哪個色號最顯白。做完之後，她會覺得「這個美甲師好懂我」，然後自然就點預約了。

這跟心理測驗的原理一樣。人只要開始回答問題，就很難中途離開。而且做完之後會想分享結果，你的曝光就是這樣來的。

場景：美甲師 Ami 的選色器

Ami 做了一個「你的手適合什麼顏色」的小工具放在網頁上。客人選膚色色調、選平常穿衣風格、選場合，最後會配出三個推薦色。每天大概有二十幾個人玩，其中五六個會直接點預約連結。比她在 IG 發限動的轉換率高了三倍。

工具一：美甲風格配對器

這是最容易上手的互動工具。客人回答 4 到 5 個問題，系統根據答案配出最適合她的美甲風格。問題要有趣，不要像問卷調查，要像朋友在聊天。

Prompt 模板：美甲風格配對器
幫我做一個「你適合什麼風格的美甲」互動工具，用純 HTML + CSS + JavaScript，一個檔案。 規則： - 共 5 題選擇題，每題 4 個選項 - 題目風格輕鬆有趣，像朋友聊天 - 每個選項對應一個風格分數（日系清新 / 韓系簡約 / 歐美個性 / 經典法式） - 最後統計分數，顯示最適合的風格 + 風格描述 + 推薦款式 - 結果頁要有「預約體驗」按鈕，連結到：（你的預約連結） - 結果頁要有「分享結果」按鈕（複製結果文字） 題目方向： 1. 你今天的心情像什麼天氣？（晴天/微風/小雨/打雷） 2. 挑一個週末活動（咖啡廳看書/逛市集/夜店/野餐） 3. 你的手機桌布是什麼風格？（花草/極簡/偶像/風景） 4. 朋友形容你最常用的詞？（可愛/帥氣/優雅/隨性） 5. 選一個旅行目的地（京都/首爾/紐約/巴黎） 設計要求： - 手機優先，米白底 + 大地色系 - 一次只顯示一題，選了自動跳下一題 - 有進度條 - 結果頁要有視覺衝擊，配色要好看
進階技巧

題目選項不要寫得太明顯。像「你喜歡日式美甲嗎？」這種一看就知道答案的問題沒有意義。要用間接問法，讓客人透過生活喜好去對應風格，這樣結果出來她才會覺得「好準」。

工具二：膚色選色器

很多客人不知道自己適合什麼顏色。冷色調的手塗暖色指甲油，做完覺得怪怪的但說不出來哪裡不對。你可以做一個選色工具，讓她選自己的膚色色調，系統推薦最搭的指甲油顏色。

Prompt 模板：膚色選色器
幫我做一個「什麼指甲油顏色最適合你」的選色工具，用純 HTML + CSS + JavaScript，一個檔案。 步驟流程： Step 1：選膚色色調（暖黃調 / 冷粉調 / 中性調 / 深膚色）用色塊讓她對比手腕內側選 Step 2：選場合（日常通勤 / 約會 / 重要場合 / 度假放鬆） Step 3：選偏好感覺（低調氣質 / 活潑亮眼 / 個性前衛 / 優雅經典） 結果： - 推薦 3 個指甲油色號（用色塊顯示）+ 每個色號的名稱和適合場景 - 加一句風格建議（例如：「你的膚色配上莓果色系會特別顯白，約會的時候試試看」） - 底部有「預約體驗這個顏色」按鈕 設計要求： - 手機優先，極簡風 - 色塊要夠大，方便在手機上點選 - 結果頁的推薦色塊要大且突出 - 米白底 + 大地色系 + 推薦色塊用實際顏色

工具三：膚質自檢工具

這個特別適合做臉部護膚的美業人。客人自己在家用手機做膚質自檢，系統根據她的回答判斷膚質類型，推薦適合的療程。重點是：她做完之後就已經被教育了，到店裡不用再從頭解釋。

Prompt 模板：膚質自檢工具
幫我做一個「一分鐘膚質自檢」互動工具，用純 HTML + CSS + JavaScript，一個檔案。 共 6 題： 1. 早上洗完臉不擦任何東西，兩小時後你的T字部位？（出油明顯 / 微微出油 / 沒什麼感覺 / 感覺緊繃） 2. 你的兩頰摸起來？（粗糙有顆粒 / 偶爾乾癢脫皮 / 還算滑順 / 容易泛紅） 3. 換季的時候皮膚會？（瘋狂出油長痘 / 乾燥脫皮 / 沒什麼變化 / 又乾又油） 4. 你的毛孔在鼻頭和臉頰看起來？（很明顯 / 鼻頭明顯臉頰還好 / 不太明顯 / 不太確定） 5. 用新的保養品皮膚容易？（長痘 / 過敏泛紅 / 沒什麼反應 / 看產品而定） 6. 你覺得自己最大的皮膚困擾是？（出油和痘痘 / 乾燥和細紋 / 暗沉不均勻 / 敏感不穩定） 結果分類：油性肌 / 乾性肌 / 混合肌 / 敏感肌 每個結果包含： - 膚質說明（50字內，白話文） - 日常保養三步驟建議 - 推薦療程名稱（接你的服務項目） - 「立即預約膚質諮詢」按鈕 設計：手機優先，米白底，一次一題自動跳轉，進度條

工具四：今日美甲靈感產生器

有些客人與其說不想做指甲，不如說不知道做什麼。她們會在 IG 滑半小時還是選不出來。你做一個「今日美甲靈感」產生器，她按一下按鈕就隨機跳出一個風格建議，附上你的作品參考。輕鬆好玩，而且會讓她一直點。

Prompt 模板：美甲靈感產生器
幫我做一個「今日美甲靈感」隨機產生器，用純 HTML + CSS + JavaScript，一個檔案。 功能： - 點擊按鈕，隨機顯示一個美甲風格建議 - 每個建議包含：風格名稱、風格描述（30字內）、推薦配色（用色塊呈現3個顏色）、適合場合 - 準備 20 組不重複的風格建議 - 按鈕文字：「換一個靈感」 - 底部固定一行：「喜歡這個風格？→ 預約體驗」（連結到預約頁） 20 組風格範例方向： 櫻花漸層 / 焦糖奶茶 / 極光鏡面 / 法式微笑線 / 莓果紅酒 / 大理石紋 / 復古格紋 / 透明果凍 / 金箔點綴 / 乾燥花押花 / 貓眼極光 / 暈染水彩 / 幾何色塊 / 珍珠貝殼 / 啞光裸色 / 彩虹糖果 / 銀河星空 / 蕾絲雕花 / 動物紋路 / 季節限定（當季主題） 設計： - 手機優先 - 卡片式設計，翻牌動畫效果 - 米白底 + 大地色系 - 色塊要圓形，排成一排

怎麼放到你的網頁上

AI 生成的程式碼就是一個完整的 HTML 檔案。你有兩個選擇。

第一個，如果你已經有了個人網頁（按照上一篇教學做的），就在 GitHub 裡新增一個檔案，例如 style-match.html，把程式碼貼進去。然後在你的首頁加一個連結指向它就好。

第二個，如果你還沒有網頁，直接把這個互動工具當成你的第一個頁面也可以。先做一個有趣的工具讓客人玩，比一個漂亮但靜態的頁面更有用。

轉換率的關鍵

互動工具做好只是第一步，真正的轉換來自結果頁的設計。結果頁要讓客人覺得「被懂了」，然後自然看到預約按鈕。不要在結果還沒出來之前就急著推銷，那會讓人反感。

怎麼讓更多人來玩

工具做好之後，重點是讓人知道它的存在。

在 IG 限動分享：「我做了一個小工具，測你適合什麼風格的美甲，連結在主頁」。這比你發作品照的互動率高得多，因為人天生喜歡跟自己有關的東西。

在 LINE 自動回覆裡放連結。新加你 LINE 的客人，第一條訊息就附上這個互動工具的連結。她玩完之後自然就知道你做什麼、擅長什麼。

結果頁一定要有分享按鈕。客人做完測驗之後想分享給朋友，你的工具就這樣被傳出去了。免費的口碑行銷。
