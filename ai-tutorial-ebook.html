用 AI 寫你的第一本電子書
美業人的知識變現

你做了十幾年美業，腦袋裡的東西夠寫三本書了。
問題不是沒東西寫，而是不知道怎麼開始。

馥靈之鑰美業 AI 學苑｜王逸君

為什麼美業人應該出一本書

你可能覺得出書是作家的事、是名人的事。但你想想，你每天在跟客人聊的那些，皮膚怎麼保養、指甲怎麼養護、肩頸為什麼痠、哪個精油適合什麼體質，這些全都是知識。而且是你花了好幾年摸出來的真知識，不是網路上複製貼上的那種。

一本電子書可以做到三件事：讓你的專業被看見、讓你睡覺也在賺錢、讓新客人在認識你之前就信任你。

以前出書要找出版社、要自己排版、要印刷。現在用 AI 寫稿、用 Canva 排版、上架到 Readmoo 或 Amazon KDP，從頭到尾不用花一毛錢，只花時間。

了解美業合作方案

Step 1：選一個你最會講的主題

不要想著寫一本「什麼都有」的書。最好的電子書只講一件事，把那件事講透。

選主題有個簡單的判斷方式：你平常被客人問最多的那個問題，就是你的書。如果你是美甲師，客人老問「凝膠會不會傷甲」，那就寫一本《美甲新手的 20 個真心話》。如果你是芳療師，客人常問「精油怎麼選」，那就寫《給忙碌女生的精油入門》。

1 用 AI 幫你釐清主題方向

如果你有好幾個想法但不知道選哪個，把它們全部丟給 AI，讓它幫你分析哪個最有市場。

Prompt 模板：主題評估
我是一位在台灣做了【N】年的【美甲師/美髮師/芳療師/SPA 美容師】。我想寫一本電子書，目前有以下幾個主題在考慮： 1. 【主題 A，例：美甲新手最常犯的錯】 2. 【主題 B，例：居家精油保養完全指南】 3. 【主題 C，例：美業創業第一年生存手冊】 我的目標讀者是：【例：25-40 歲想學美甲保養的女生 / 想創業的美業新手】 請幫我分析： 1. 每個主題的市場需求（誰會買？有多大？） 2. 每個主題的競品狀況（市面上已經有類似的書嗎？） 3. 我寫哪個最有優勢（根據我的背景） 4. 最終推薦一個主題，說明理由 5. 給這個主題想 5 個書名（要吸引人但不浮誇）

Step 2：列出完整大綱

大綱就是這本書的骨架。骨架穩了，後面填肉就快了。

一本好的電子書大概 8-12 章，每章 2,000-3,000 字，整本大約 2-3 萬字。不用太長，讀者在手機上看的，太厚會放棄。

2 用 AI 產出書籍大綱
Prompt 模板：大綱產生
我要寫一本電子書，主題是「【你選定的主題】」。 目標讀者：【描述】 書的定位：【實用工具書 / 經驗分享 / 入門指南 / 觀念翻轉】 預計字數：約 2-3 萬字（8-12 章） 語氣風格：口語化、像朋友在聊天、不學術、有個人觀點 請幫我列出完整大綱，包含： 1. 書名（3 個選項） 2. 副標題 3. 各章標題 + 每章 3-5 個重點（用一句話描述） 4. 前言的核心訊息（為什麼寫這本書） 5. 結語的收尾方向 大綱要有節奏感：開頭吸引人、中間給乾貨、後段給行動力。不要每章都是「什麼是XX」「XX的好處」這種教科書結構。
大綱確認要點

AI 產出的大綱一定要自己看過一遍。有些章節可能太理論，有些可能跟你的經驗不符。大膽刪、大膽改。這是你的書，不是 AI 的書。

Step 3：逐章寫作

拿到大綱之後，一章一章寫。不要想著一次寫完整本，每天寫一章，10 天就寫完了。

寫作的關鍵是：你提供經驗和觀點，AI 負責組織和潤飾。你說「我有個客人怎樣怎樣」，AI 幫你把故事寫順。你說「這個技術的重點是什麼什麼」，AI 幫你編排成讀者容易吸收的結構。

3 逐章寫作 Prompt
Prompt 模板：單章寫作
現在幫我寫這本書的第【N】章。 章節標題：「【標題】」 這章要涵蓋的重點： 1. 【重點 A】 2. 【重點 B】 3. 【重點 C】 我的個人經驗和素材（請融入寫作中）： - 【你的真實故事、客人案例、自己踩過的坑】 - 【你對這個主題的獨特看法】 寫作要求： - 字數約 2,500 字 - 語氣像跟讀者面對面聊天 - 每隔 3-4 段插入一個小標題，方便掃讀 - 長短句交替，不要每句都很長 - 可以有一兩句幽默或自嘲，但不要刻意搞笑 - 結尾給讀者一個可以馬上做的小行動 注意：不要用「首先」「其次」「最後」「總之」這類過渡詞。
Prompt 模板：改寫潤稿
以下是我寫的初稿，請幫我改寫。 【貼上你的初稿】 改寫方向： 1. 保留我的觀點和故事，不要改掉核心意思 2. 讓句子更順暢，去掉贅字 3. 如果有太學術的地方，換成口語化的說法 4. 加入 2-3 個生活化的比喻 5. 段落太長的拆開，太短的合併 6. 確保讀起來有呼吸感（長短句交替） 請給我改完的全文，不要只給修改建議。

Step 4：前言、結語和書籍簡介

前言是讀者打開書看的第一段文字，決定他要不要繼續看下去。結語是他看完之後記住的最後一句話。書籍簡介是他在書店頁面決定要不要買的關鍵。這三個東西，花的時間應該跟寫正文一樣多。

4 前言與書籍簡介
Prompt 模板：前言撰寫
幫我寫這本書的前言，800-1,000 字。 書名：「【書名】」 作者背景：我做【美甲/美髮/芳療】【N】年，【簡述你的經歷和轉折點】 寫這本書的動機：【為什麼想寫？是因為看到什麼問題？想傳達什麼？】 前言要做到： 1. 第一段就抓住讀者（用一個場景或問題開頭，不要「大家好我是XXX」） 2. 讓讀者覺得「這個人懂我」 3. 誠實說出這本書能給什麼、不能給什麼 4. 結尾讓人想翻到下一頁 語氣：溫暖但不煽情，有自信但不自大。像是一個前輩在跟你說「這條路我走過，我告訴你我看到了什麼」。
Prompt 模板：書籍簡介（上架用）
幫我寫這本書的書籍簡介，用在 Readmoo/Amazon KDP 上架頁面。 書名：「【書名】」 全書章節：【列出章節標題】 目標讀者：【描述】 請寫兩個版本： 版本 A（200 字精簡版）：用在書店頁面的主要描述 版本 B（400 字完整版）：用在詳細介紹區 要求： - 讓沒聽過你的人 10 秒內知道這本書在講什麼 - 列出 3-5 個「讀完你會得到」的具體收穫 - 不要用「本書旨在」「筆者認為」這種正式用語 - 結尾用一句話製造購買衝動

Step 5：排版與封面設計

文字寫完了，接下來是讓它看起來像一本書。電子書的排版不用太複雜，重點是乾淨、好讀、在手機上看起來舒服。

排版工具選擇

Canva 是最簡單的選擇。它有免費的電子書模板，你只要把文字貼進去、換個顏色、選個字型就好了。如果你想更專業一點，可以用 Google Docs 排好之後匯出 PDF 或 EPUB。

5 封面設計 Prompt
Prompt 模板：封面設計指導
我要用 Canva 設計電子書封面。 書名：「【書名】」 副標題：「【副標題】」 作者名：【你的名字】 書的調性：【專業溫暖 / 清新活潑 / 高端質感 / 實用工具感】 目標讀者偏好的視覺風格：【簡約 / 花朵植物 / 幾何圖形 / 手繪感】 主色調偏好：【暖色系 / 冷色系 / 大地色 / 莫蘭迪色】 請給我： 1. 3 個封面設計方案的文字描述（包含構圖、配色、字型建議） 2. 在 Canva 搜尋模板時可以用的英文關鍵字（5 個） 3. 封面設計的 3 個注意事項（避免常見錯誤） 4. 書名在封面上的排列建議（字級大小、位置）

Step 6：上架到平台

書寫完了、排版好了，最後一步是讓全世界都能買到它。兩個主要平台：台灣讀者用 Readmoo，海外讀者用 Amazon KDP。兩個都可以同時上架，不衝突。

Readmoo 上架流程

Readmoo 是台灣最大的電子書平台。到 Readmoo 的「自助出版」頁面申請帳號，上傳 EPUB 格式的書稿，填寫書籍資訊，設定價格，送審。審核通常 3-5 個工作天。分潤比例是你拿 70%，平台拿 30%。

Amazon KDP 上架流程

Amazon KDP（Kindle Direct Publishing）面向全球讀者。到 kdp.amazon.com 註冊帳號，上傳書稿（支援 EPUB、PDF、DOCX），設定價格和銷售地區。定價在 USD 2.99-9.99 之間可以拿 70% 分潤，其他價格帶是 35%。

6 上架資料準備
Prompt 模板：上架資料
我的電子書要上架到 Readmoo 和 Amazon KDP，幫我準備以下資料。 書名：「【書名】」 副標題：「【副標題】」 作者：【名字】 書籍分類：【例：商業經營 / 生活風格 / 美容保養】 目標讀者：【描述】 全書字數：約【N】字 定價考量：目標讀者可接受的價格帶 請幫我準備： Readmoo 用： 1. 書籍簡介（繁體中文，200 字 + 400 字各一版） 2. 分類建議（Readmoo 的分類選擇） 3. 關鍵字標籤（5-7 個） 4. 建議定價（TWD） Amazon KDP 用： 1. 書籍簡介（英文版 + 繁體中文版） 2. 分類建議（Amazon BISAC 分類 2 個） 3. 搜尋關鍵字（7 個，英文） 4. 建議定價（USD） 5. 銷售地區建議

Step 7：上架之後的行銷

書上架了不代表就有人會買。你需要讓你的客人、追蹤者、朋友都知道你出書了。

7 新書推廣 Prompt
Prompt 模板：新書推廣文案
我的電子書「【書名】」剛上架了。幫我寫推廣文案。 上架平台：【Readmoo / Amazon KDP / 兩者都有】 購買連結：【連結】 定價：NT$【金額】 請幫我寫以下版本： 1. IG 限時動態文字（3 張，每張 1-2 行） 2. IG 貼文（150 字以內 + 5 個 hashtag） 3. Facebook 長文（300 字，講出書背後的故事） 4. LINE 群發訊息（簡短版，100 字以內） 5. 個人朋友圈/限動分享（自然不推銷的版本） 語氣：不要太銷售感。我不是在賣書，我是在分享一個我覺得有用的東西。像朋友跟你說「欸我最近做了一個東西你看看」那種感覺。
定價建議

台灣市場的電子書甜蜜點大約是 NT$199-399。如果你的書偏實用工具型（有大量模板或方法論），可以定到 NT$349-499。如果是輕鬆的經驗分享型，NT$199-249 比較容易讓人按下購買。

可以在上架初期設定限時優惠價，例如第一週 NT$149 衝銷量和評論，之後恢復原價。
