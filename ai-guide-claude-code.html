Claude Code 是什麼？小白也能用嗎｜一個美業老闆的真心話

作者：王逸君（馥靈之鑰創辦人｜美業 34 年資歷）｜2026.04.11

先講結論：Claude Code 就像你請了一個住在電腦裡的工程師。你跟他說你要什麼，他就幫你做。不是給你建議而已，是真的動手幫你做出來。

我身邊有不少美業朋友開始用 AI 工具了。寫文案、做圖、回客人訊息，這些大家比較熟。但很多人不知道，AI 現在已經可以幫你做「技術活」了，像是改網站、建系統、批次處理資料，這些以前一定要請工程師做的事。

這篇文章是寫給不是技術背景、但想知道 Claude Code 到底能幹嘛的人看的。

Claude Code 是什麼

Claude 是 Anthropic 這家公司做的 AI。Claude Code 是他們推出的一個工具，讓 Claude 可以直接操作你電腦上的檔案。

用比喻來說：Claude.ai（就是你在網頁上跟 AI 聊天那個）像是一個很聰明的顧問。你問他問題，他給你答案和建議。但他不會幫你動手。

Claude Code 像是你請了一個工程師坐在你旁邊。你說「幫我把這個網頁的標題改成紅色」，他就直接幫你改。你說「幫我建一個預約系統」，他就真的寫出程式碼，建出一個可以用的系統。

差別在於：一個是用嘴巴幫你，一個是用手幫你。

跟 Claude.ai 到底差在哪裡

這是很多人搞不清楚的地方。讓我用最簡單的方式講。

Claude.ai 適合的事情：聊天、問問題、寫文案、想點子、分析資料、翻譯。你丟一段文字給它，它回一段文字給你。像傳 LINE 一樣。

Claude Code 適合的事情：改網站、寫程式、建工具、批次處理檔案、整理資料庫。你告訴它你要什麼結果，它直接去動你電腦上的檔案。

舉個例子。假設你的網站有幾十頁，你想把所有出現「某個舊名稱」的地方統一改成新名稱。如果用 Claude.ai，你要一頁一頁貼給它看，它告訴你哪裡要改，你自己手動去改。幾十頁可能要改好幾天。

用 Claude Code，你說一句「幫我把所有 HTML 檔案裡的舊名稱改成新名稱」。它幾分鐘搜完全部檔案、列出要改的地方、一次改完、檢查有沒有改壞東西、然後自動上傳到網站。

三分鐘。

什麼時候用 Claude Code

如果你有以下任何一種需求，Claude Code 可能幫得上你。

你想改自己的網站，但不會寫程式。你想做一個線上工具（計算機、測驗、預約系統），但請人太貴。你有很多檔案要批次處理（改名、整理、轉檔）。你想把某個重複的工作自動化。

具體來說，Claude Code 能做的事包括：從零開始建一個完整的品牌官網、做各種計算工具（把不同系統整合在一起）、做互動測驗系統（有計分邏輯的那種）、串接線上金流（讓客人可以直接付錢）、做課程投影片（HTML 格式的教學簡報）、建預約系統、做會員管理後台。

這些事如果請外面的工程師做，保守估計幾十萬起跳。

什麼時候用 Claude.ai 就好

如果你的需求是：想一個社群文案、分析一段文字、翻譯一篇文章、討論一個商業點子、寫一份報告。用 Claude.ai 就夠了。

Claude.ai 的優勢是快、方便、不需要設定。打開網頁就能用。你不需要安裝任何東西。

我自己的使用習慣是：想東西的時候開 Claude.ai，做東西的時候開 Claude Code。想的時候需要一個對話的對象，做的時候需要一個動手的夥伴。

不是工程師也能用嗎

可以。但要講實話：有一點學習曲線。

Claude Code 的操作介面是終端機（就是那個看起來很駭客的黑色視窗）。你不需要打很技術的指令，大部分時候就是用中文告訴它你要什麼。但你需要會基本的檔案操作：知道你的檔案放在哪裡、知道怎麼打開終端機、知道怎麼切換資料夾。

這些東西大概花一兩天就能學會。YouTube 上搜「終端機基礎教學」就有很多。

我媽今年 75 歲了，她會用 ChatGPT 幫她寫 LINE 回覆。她不會用 Claude Code，但她證明了一件事：AI 的使用門檻，比你以為的低很多。

Claude Code 的門檻比 ChatGPT 高一點，但比學程式低非常多。你不需要會寫程式，你只需要會描述你要什麼。「幫我做一個頁面，上面有一個表格，列出我們家的服務和價格」。這種程度的描述就夠了。

多少錢

Claude Code 需要訂閱 Anthropic 的 Max 方案。目前有兩種：每月 100 美元和每月 200 美元。

100 美元的額度對一般使用來說夠用。如果你是重度使用（像我一樣每天都在改網站），200 美元比較保險。

值不值得？看你的時間值多少。

如果你原本每個月花 5,000 到 10,000 元請人改網站、做設計、處理技術問題，那 100 美元（大約 3,200 元台幣）的 Claude Code 訂閱費，可能比你原本的花費還少。而且它 24 小時在線，不會已讀不回，不會拖工期。

如果你目前沒有任何技術需求，只是想聊天和寫文案，那 Claude.ai 的免費版或 Pro 版（每月 20 美元）就夠了，不需要上 Claude Code。

我的真心話

用了 Claude Code 之後，我最大的感受不是「效率變高了」。是「我能做的事變多了」。

以前有很多想法，但卡在「我不會做」。想做一個心理測驗？要請工程師。想在網站上加一個新功能？要請工程師。想改一下版面？還是要請工程師。每一個「想做」都要先翻譯成工程語言，然後等人家排期，然後來回溝通好幾次，最後做出來的可能跟你想的不一樣。

現在是：我想到什麼，直接跟 Claude Code 講。它做出來的東西如果不對，我當場就說「不是，我要的是這樣」。三分鐘後就改好了。

這種「想到就能做到」的感覺，對一個創業者來說，比什麼都值錢。

當然，Claude Code 不是萬能的。它有時候會做錯，有時候會理解偏差，有時候改完一個地方會不小心弄壞另一個地方。你需要學會怎麼跟它溝通、怎麼檢查它的產出、怎麼在它出錯的時候導正。

但跟請真人工程師比起來，這些都是小事。至少它不會突然消失、不會談到一半說要加價、不會在你最急的時候跟你說他在忙別的案子。

常見問題

Claude Code 跟 Claude.ai 有什麼不同？

Claude.ai 是聊天介面，你問問題它回答。Claude Code 可以直接操作你電腦上的檔案，幫你改程式、建網站、批次處理資料。一個用嘴巴幫你，一個用手幫你。

不是工程師也能用 Claude Code 嗎？

可以，但需要學基本的終端機操作（大約一兩天）。你不需要會寫程式，只需要會用中文描述你要什麼。「幫我做一個表格列出服務和價格」這種程度就夠了。

Claude Code 要多少錢？

需要 Claude Max 方案，月費 100 美元或 200 美元。100 美元方案的額度足夠一般使用。如果你原本每月花超過這個數字在技術外包上，Claude Code 可能更划算。

「工具不會取代你，但會放大你。」
