妳現在正從桌面捷徑開啟，可以直接開始使用。

📱 iPhone / iPad
3 步驟放到主畫面
  1. 先確定你是用 Safari 瀏覽器打開（不是 LINE / Chrome 內建瀏覽器，會裝不了）。
    看底下有沒有一個正方形往上的箭頭 ⬆️，點它。
  2. 跳出來的選單往下滑一點，找「加入主畫面」 ➕，點下去。
  3. 右上角點「加入」就好了。
    手機主畫面就會出現馥靈之鑰的圖示，以後直接點開就能用。
💡 為什麼不能「點一下就裝」？
iOS Safari 不允許網站自己觸發安裝（Apple 限制），一定要用「分享 → 加入主畫面」。這不是 bug，是規定。裝好之後和正常 App 一樣全螢幕，沒有網址列。
🤖 Android
一鍵安裝到桌面
如果上面按鈕沒反應，照下面步驟手動裝：
  1. 點右上角「⋮」選單按鈕（三個點）
  2. 找「安裝應用程式」或「加到主畫面」，點下去
  3. 跳出來點「安裝」就好了
💻 電腦
用手機掃這個 QR Code
用 iPhone 相機 / Android 掃描器掃一下
會自動開啟這個頁面，依照手機指示安裝
電腦版 Chrome / Edge 也可以把馥靈之鑰裝成桌面應用程式。網址列右邊如果有『⊕ 安裝』圖示點一下就能裝。
