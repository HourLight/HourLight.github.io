## 前言：33大命理系統在馥靈之鑰的角色

你有沒有過那種感覺？Google Maps 明明顯示你在這條路上，但你就是覺得哪裡怪怪的。

一套命理系統，就是一張地圖。八字看你的時間座標，占星看你的空間座標，人類圖看你的能量座標。一張地圖可能帶錯路，但 33 張地圖同時指向同一個方向的時候，那個方向大概就不會錯太遠。

這就是馥靈之鑰做 33 大系統整合的原因。不是要讓人覺得很厲害，是因為單一系統的準確度有天花板。當東方的八字說你今年適合動、西方的占星也說你今年木星過境事業宮、連瑪雅曆都顯示你進入白風波符的啟動年，三個完全不同文化背景的系統交叉驗證出同一個訊號，那個訊號的可信度就不一樣了。

覺察師在使用這些工具時，記住三件事：

► 每套系統是一個「不同角度的鏡子」，不是標準答案
► 交叉比對才有意義，單一系統別說得太絕對
► 提問者聽完要能「做點什麼」，不是只有「哇好準」

覺察師要記住第四件事，也是最重要的一件：這些工具是平台的流量入口，每一個工具都要能接住提問者，然後帶到下一步。免費工具讓人體驗 → 體驗完覺得準 → 引導到AI深度解讀或一對一服務 → 形成服務鏈。沒有下一步的解讀，就只是免費算命而已。

## 一、東方命理

### 工具：八字命理計算器

📌 一句話：用你出生那一刻的天干地支，看你這輩子拿到什麼牌。

🎯 為什麼要用？八字是東方命理的基底語言。就像學英文要先學字母，學東方命理要先會看八字。四柱八個字，藏了你的性格傾向、運勢節奏、關係模式。

💡 適合情境：
► 提問者想知道「我適合做什麼工作」
► 想了解今年整體運勢走向
► 想看兩個人的關係合不合

📚 理論基礎：源自中國古代天文曆法，以年月日時四柱、天干地支、五行生剋為核心架構。至少有千年以上的實證累積。

🔮 H.O.U.R. 對應：H（身心校準）→ 從五行看體質傾向；R（行動進化）→ 從大運流年找行動時機。

🌿 精油搭配：五行缺什麼補什麼。缺木用尤加利、缺火用肉桂、缺土用岩蘭草、缺金用茶樹、缺水用薰衣草。

📖 操作：輸入國曆生日與出生時辰 → 系統自動轉換農曆 → 排出四柱、藏干、十神、納音 → 展開大運流年。頁面同時提供稱骨算命。

🗣️ 解讀要訣：先看日主強弱，再看用神忌神，接著看大運流年。跟提問者講的時候，不要說「你命中缺火」，要說「你的能量結構裡火的比例偏低，所以容易在需要衝勁的時候覺得使不上力」。

❓ 常見問答：
► 「不知道出生時辰怎麼辦？」→ 可以先算不含時柱的三柱，準確度約七成
► 「八字跟紫微哪個準？」→ 不是誰準的問題，是看的維度不同。八字看節奏，紫微看格局

📎 頁面：bazi-calculator.html

💰 變現應用：
► 諮詢切入：「幫你看八字五行結構＋今年大運走向」，搭配馥靈初探 NT$5,400
► 交叉銷售：八字看完建議加看紫微（格局）和占星（人際），形成「東西方三角驗證」套餐
► 服務鏈：八字結果 → 抽牌（看當下行動建議）→ 心理測驗（驗證性格傾向）→ 深度覺醒 NT$6,800
► 話術：「八字告訴你五行缺什麼，精油可以幫你補。要不要看看你的完整33套命盤？一次算完交叉比對更準。」

### 工具：紫微斗數命盤

📌 一句話：用 115 顆星耀在十二宮的排列，畫出你人生的衛星導航圖。

🎯 為什麼要用？如果八字是看你拿到什麼牌，紫微就是看你怎麼打這副牌。十二宮各管一個人生領域，星耀的亮度和組合決定了你在每個領域的天賦和挑戰。

💡 適合情境：
► 想做完整的人生全面盤點
► 想了解感情模式為什麼一直重複
► 想知道適合獨立創業還是當幕僚

📚 理論基礎：相傳源自唐代，以北斗南斗為主星系統，結合十二地支宮位。是東方命理中結構最精密的系統之一。

🔮 H.O.U.R. 對應：O（智慧辨識）→ 從命宮主星看核心決策模式；U（潛能解鎖）→ 從遷移宮和官祿宮找天賦施展方向。

🌿 精油搭配：依命宮主星選。紫微星用乳香（王者氣度）、天機星用迷迭香（思路清晰）、太陽星用甜橙（溫暖開朗）、太陰星用茉莉（細膩直覺）。

📖 操作：輸入國曆生日（系統自動轉農曆）→ 選擇性別 → 排出十二宮完整命盤 → 含主星、輔星、煞星、四化、大限走向。

🗣️ 解讀要訣：先看命宮主星組合定調，再看三方四正有沒有煞星沖破。跟提問者講的時候用生活化的比喻：「你的命宮是天機太陰，就像一個感受力超強的偵測器，好處是什麼都察覺得到，代價是容易想太多。」

❓ 常見問答：
► 「命宮沒有主星是不是很差？」→ 不是，借對宮星耀，反而適應力更強
► 「四化飛星要怎麼看？」→ 進階內容，先把十二宮主星看熟再學四化

📎 頁面：ziwei-calculator.html

💰 變現應用：
► 諮詢切入：「十二宮全面盤點＋大限走向」，這是最適合做完整報告的系統，搭配深度覺醒 NT$6,800

► 服務鏈：紫微命盤 → 感情宮有疑問就推合盤引擎 → 事業宮有疑問就推BG5 → 一對一陪跑
► 話術：「你的命盤裡夫妻宮化忌，不是感情不好的意思，是你對感情的標準比較高。要不要用合盤引擎看看你跟對方的33套比對？」

### 工具：七政四餘命盤

📌 一句話：中國原生的占星術，用日月五星加四虛星排出你的東方星盤。

🎯 為什麼要用？很多人以為占星是西方的東西。其實中國在漢代就有完整的星命系統。七政四餘用的是二十八宿而不是十二星座，座標系統完全不同，看到的面向也不一樣。

💡 適合情境：
► 對中國天文學有興趣的提問者
► 想用東方星盤交叉驗證西洋占星的結果
► 想了解自己在宿度系統裡的定位

📚 理論基礎：七政指日月金木水火土，四餘指羅睺計都月孛紫氣。以二十八宿為背景座標，十二宮為人生領域劃分。

🔮 H.O.U.R. 對應：O（智慧辨識）→ 從命宮宿度看先天智慧型態。

🌿 精油搭配：依化曜屬性。太陽化忌用佛手柑、月亮化忌用羅馬洋甘菊。

📖 操作：輸入出生資料 → 系統計算七政四餘行星位置、二十八宿宿度、十二宮宮度 → 顯示化曜十神與生剋關係。

🗣️ 解讀要訣：這套系統比較冷門，大部分提問者第一次接觸。先解釋「這是中國自己的占星術」，再帶入宿度解讀。

❓ 常見問答：
► 「跟西洋占星差在哪？」→ 座標系統不同，西洋用回歸黃道（春分點），七政四餘用恆星黃道

📎 頁面：qizheng-calculator.html

💰 變現應用：
► 諮詢切入：搭配西洋占星做「東西方星盤雙重解讀」，提升服務獨特性
► 交叉銷售：「你的西洋星盤看完了，要不要看看中國的星盤怎麼說？兩張星圖重疊的部分就是你最確定的座標。」
► 服務鏈：七政四餘 → 西洋占星 → 吠陀占星 → 三套星盤交叉＝高端服務，搭配三次轉化 NT$12,800

🌿 精油搭配：同八字五行對應。

🌿 精油搭配：依落宮五行。
