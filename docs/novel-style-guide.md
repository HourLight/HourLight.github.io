## 三、組織與陣營

### 聞香堂（正派）

**核心信念**：嗅覺與記憶、情緒、命運有深層連結，這是人的本能而非迷信

**組織特徵**
- 無總部、無會長、無入會儀式
- 成員靠血親或師徒關係相連
- 每年特定時節秘密見面一次
- 地點與方式每次都變
- 分散居住、分開生活

**起源**
- 南宋紹興年間，祖師「觀月法師」發現嗅覺系統的秘密
- 第二代：林采（三個徒弟分散京城、嶺南、江南）
- **第十八代最慘**：8 人遭害（含林嫣）、12 人失聯、只剩 7 人存活
- **第十九代**：目前只有趙承硯 + 趙馥靈（念兒候補）

**十大禁忌**（第 33 章趙承硯給的冊子）
1. 不可以氣味害人
2. 不可謀取私利超過己身所需
3. 不可傳承於無德者
4. 不可公開身份於無信之人
5. **不可對無味堂有任何善意**
6. 不可記錄本代所有成員真名於一處
7. 不可在無緊急之事時互相聯絡
8. 不可教授外人完整系統（只能教片段）
9. 不可挑戰無味堂於他們設定的戰場
10. **不可遺忘本堂從未追求『被看見』**

**目前活存的第 18-19 代成員**
1. 江雪娘（領導）
2. 趙承硯（執筆）
3. 周蓮（宮內）
4. 王姑（山東）
5. 林老丈（汴京）
6. 沈先生（江陵）
7. 蔡婆（廣州）
8. 趙馥靈（第 19 代新成員）
9. 念兒（候補，8 歲）

### 無味堂（反派）

- **核心信念**：消除所有跟氣味、感覺、直覺相關的知識，只留純理性
- **視聞香堂為「混亂的起源」**
- 清除策略：漸進式暗殺（每次殺 1-2 個，不大屠殺，避免團結）

## 七、寫作風格規範（最重要的部分）

### 🖋 句式特徵
- **對話佔比 40-50%**
- **短句為主**，知識密集段快節奏，情緒高漲時短句疊堆
- **長句用在**背景設定、複雜心理
- **古白話 + 現代文流暢混用**（不違和）

### 🌿 場景描寫優先順序
1. **氣味**（絕對第一優先）— 每個重要場景都有「她聞到了什麼」
2. **身體語言**（微表情、呼吸、手部、步態）
3. **動作**（比心理獨白多）
4. 光線、色彩等放最後

### 🤖 小馥對話的固定格式

**最基本呼叫：**
```
「小馥。」
「在。」
「[問題]」
「[分析回答]」
```

**分析報告：**
- 以「根據...」開頭
- 用「...可能...」「高度...」「建議...」結尾
- 精確到百分比、心跳、瞳孔數據

### 類型市場分析

穿越商業爽文的讀者核心需求（市場研究整合）：

1. **智商碾壓是核心爽點** — 讀者要在女主亮出知識的那一刻，感受到「這個時代的人根本想不到」的震撼。不是「她說了個合理的建議」，是「她拿出的是這個時代整整超前幾百年的東西」。《聞香識命》目前的知識植入已有這個基礎（GC-MS、三假消息測試、零知識切割），但每次亮出的時機要更有儀式感，讓讀者看見「啊，她在用2030年的方法」這個清醒的瞬間。

2. **打臉要可見、要完整** — 質疑過女主的人，讀者記得每一個。這些人要在後續章節裡被明確回饋：謠言說她妖術的人、懷疑她能力的人、以為她只是個運氣好的小商人的人，都需要一個看得見的翻轉。不必讓女主親手扇，但讀者要「眼睜睜看著那個人意識到自己錯了」。

3. **商業勝利要有具體數字** — 「生意越來越好」不夠。讀者要感受到差距的規模：這批乳香比市場價低9%，省了多少錢；觀香閣的商業聲名在半年內從零到讓顧家主動上門。數字不用每次都精確，但要讓人感受到「她領先別人到底領先多少」。

4. **金手指要有清醒感** — 小馥和空間實驗室是這部小說的核心金手指。每次女主運用2030年的知識，要讓讀者明確感受到「她來自未來」的身份。目前小馥的分析有數據有論文引用，這個方向是對的，但可以更刻意地讓古代人物的反應來放大這個差距：「他從未見過有人這樣分析問題」「這個方法他想了三十年也沒想出來」。

5. **配角要有記憶點** — 讀者記角色靠的是一個「標誌性特質」，不是完整設定。林老丈現在有「六十年老人的尊嚴」，盧先生有「廣藿香浸透的布料」，雪娘目前還是一個名字。每個重要配角需要有一個讀者不會搞混的標誌。

6. **節奏：每段弧線三步結構** — 設置威脅（1章）→ 升溫危機（1-2章）→ 女主碾壓解決（1章）→ 小勝收尾（0.5章）。目前第51-60章的情報洩漏弧線走得很穩（發現縫隙→三假消息設計→第一個回聲→排除確認→汴京修復→商業回收），是全書目前最完整的弧線節奏，可以作為後續章節的標準模板。

7. **感情線要有進度條** — 讀者不需要言情，但需要一個讓人想磕的張力。趙馥靈和鄭君珩目前是「靈魂層同盟」，這個設定本身就是最好的CP基礎。每10章要有一個讓讀者感受到兩人距離縮短的時刻：不需要表白，可以是「他記住了她說的某句話」「她意識到他理解她的方式是這個時代沒有人能做到的」。

### 《聞香識命》當前定位確認

**真正在運作的優勢（不能丟）：**

- 智商爽點已建立：三假消息測試（51-54章）是全書目前智力密度最高的弧線，讀者看完之後知道女主是真的在用2030年的方法，不是靠直覺蒙對的。
- 情感層有厚度：不告訴林老丈測試真相、繞一千里保全老人臉面，這個選擇讓女主不是一個「贏就夠了」的主角，是一個「怎麼贏也在意」的人。這個層次在商業爽文裡是稀缺的，要繼續保留。
- 氣味作為敘事工具是真正的差異化：每個人物都有氣味特徵，每個情緒節點都有嗅覺錨定，這是這部小說在同類中無法被複製的獨特性。
- 小馥的毒舌設定已站穩：「謝謝誇獎」「您在假裝平靜」這類台詞讓AI助手有人格，不是工具，是夥伴，讀者會喜歡這個關係。

**目前需要補強的地方：**

- 鄭君珩（七皇子）的存在感：第36-60章他幾乎缺席，讀者對他的記憶還停在第11-30章的宮廷線。商業弧線發展的同時，需要定期讓他出現，維持CP感和宮廷線的溫度。
- 無味堂的具體形象：目前只知道「他們殺人有耐心、不大屠殺」，但沒有一個讀者記得住的無味堂人物。反派需要一個臉。
- 趙馥蘭的弧線：她在第10章之後開始鬆動，但第36-60章幾乎消失。一個有成長空間的嫡出大姐是很好用的人物，不要讓她閒置太久。

### 核心差異化優勢（不能丟的靈魂）

這部小說在穿越商業文裡真正稀缺的三件事：

1. **嗅覺敘事體系** — 沒有第二部穿越文的主角是芳療師，沒有第二部穿越文的人物設定是靠氣味辨識身份、靠嗅覺感知情緒。這不是一個「設定」，是整個敘事邏輯的基礎：因為聞香堂、因為林嫣、因為母親的筆記、因為鄭君珩的MCS，所有主線都跟氣味有關。這條線只要繼續認真寫，任何同類作品都複製不了。

2. **AI助手作為思維工具而非萬能金手指** — 小馥不解決問題，小馥分析問題。最終判斷和行動永遠是女主自己做的。這讓智商爽點是屬於女主的，不是屬於工具的。讀者感受到的是「她在用工具」，不是「工具在替她解決」。這個設計讓主角不會讓人覺得開掛開得理所當然。

3. **商業勝利有情感質地** — 林老丈那一章是最好的例子：她贏了，但她的贏法是讓另一個人也保住了。這種「怎麼贏也在意」的質地讓商業線不只是商業線，是主角性格的延伸。這是把商業爽文寫成文學的關鍵差距。

### 未來章節爽點設計建議（第61章後）

1. **顧老爺談判正式開場（61-62章）** — 顧老爺進場後的正式商談，讓讀者看到女主在一個長安最老的商業家族面前毫不怯場。爽點設計：顧老爺在談話中問到「你的消息從哪裡來」，女主的回答（「做香料的人靠的是鼻子」）讓他無法追問，卻讓他更確定她值得合作。讓讀者感受到：她沒有說謊，但她讓一個活了六十年的老商人聽到了他想聽的答案。

2. **林老丈的汴京回信揭曉（第61章或62章）** — 「三行字和一個看不懂的附圖」。這個懸念留了很久，讀者需要一個讓人驚喜的揭曉。建議：那個附圖是一種聞香堂傳統的確認符號，只有第18代以上的人才認識，而馥靈認識，因為她在母親的筆記裡見過。這個認識的瞬間是爽點，也是把林老丈的地位更鞏固的機會。

3. **蔡婆的情報來源揭曉（63-65章）** — 她怎麼在密信未寄出前就知道計畫。建議答案：柳氏提前傳訊（已知聞香堂成員之間有自己的通訊方式）。揭曉時讓馥靈意識到：她以為自己在推進計畫，但這個網絡在她出發之前就已經在為她鋪路了。這個爽點的性質是「自己比自己以為的更被信任」。

4. **趙馥蘭的回歸（65-70章）** — 迷迭香的伏筆已埋了幾十章，讓它有一個爽的收線：趙馥蘭其實已經在默默為馥靈做某件事，那件事在某個關鍵時刻暴露出來。建議：迷迭香的來源是趙馥蘭托人從某個特殊管道弄來的，那個管道和聞香堂有關係，趙馥蘭不一定知道全貌，但她在用她知道的一點點保護妹妹。讀者對嫡庶矛盾的預期被徹底打破的瞬間是爽點。

5. **無味堂第一個真正的臉（70-75章）** — 目前無味堂只有「周嬤嬤害了林嫣」，但讀者見不到活著的無味堂人。建議讓一個無味堂的人物以「看起來完全無害的形象」出現，可以是顧家的某個人、可以是一個從北邊來的商人，然後在某個細節上讓馥靈聞出「沒有氣味」。那個「他的氣味是故意清除過的」的發現，就是懸念。

6. **七皇子重新出場（75-80章）** — 商業線穩住之後，讓宮廷線重新被啟動。建議觸發點：顧家的合作讓觀香閣進入了長安上層商業圈，而這個圈子跟皇子的黨爭有交集。七皇子的出現不是愛情線，是「他們發現彼此的勢力都在成長，是時候確認同盟還在」的重新校準。讓讀者看到：他記得她說過的某句話，她也發現他這段時間在做的事跟她的網絡有隱性的連結。

7. **張叔同倒台的過程（80-85章）** — 第35章預告一個月後被彈劾，但這個收尾目前沒有展開。建議讓女主在某個意外的情況下看見張叔同倒台的過程，而且是遠遠看著、什麼都不用做的那種，強調「他的帳本已經沒有她了、毒酒被掉包的事他到死都不知道是誰做的」。這個爽點的性質是：最乾淨的勝利是讓對方連自己輸給了誰都不知道。

8. **觀香閣商號正式對外揭開身份（90-100章）** — 目前馥靈在長安商圈以「觀香閣掌事」存在，但她的真實身份（將軍府庶出小姐）是隱藏的。讓這個身份被揭開的那一刻成為一個大爽點：所有那些以為觀香閣背後有什麼大人物的人，發現掌事就是一個十三歲的庶出小姐。讓這個揭曉的時機是她自己選擇的，不是被逼的，因為那個時候她已經不需要隱藏了。
