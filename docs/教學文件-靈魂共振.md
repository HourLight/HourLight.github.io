# 靈魂同頻 教學文件

## 這是什麼？（跟小孩解釋版）

想像兩個收音機。如果它們調到同一個頻道，就能聽到同樣的歌。這就是「同頻」。

靈魂同頻就是在看：兩個人的頻道有多像？

不是說頻道不同就不好。有時候一個人聽古典樂、一個人聽搖滾，放在一起反而很特別。重點不是一不一樣，是知道差在哪裡。

## 運作原理（一步一步）

1. 輸入兩個人的生日
2. 分別計算每個人的：生命靈數、八字日主、馥靈 H/O/U/R 四個數
3. 比較生命靈數的合盤分數（同數字、互補數字、相生相剋等）
4. 比較八字日主的五行互動（相生、相剋、比和等）
5. 檢查天干合化（特殊的化學反應組合）
6. 比較 H.O.U.R. 四個維度的分數
7. 加權計算總分
8. 交叉驗證所有系統

## H.O.U.R. 四維怎麼看？

► H（身心校準）：兩人在情緒和身體層面的同步度。分數高代表你們對「舒服」的定義很像。
► O（智慧辨識）：兩人在思考和判斷上的同步度。分數高代表你們的邏輯方式接近。
► U（潛能解鎖）：兩人在成長和突破上的同步度。分數高代表你們想要變好的方向一致。
► R（行動進化）：兩人在行動和執行上的同步度。分數高代表你們的步調接近。

四個維度不需要全部高。有些組合是 H 高 R 低（情感很合但做事步調不同），有些是 O 高 H 低（想法很合但情緒節奏不同）。每種組合都有它的功課和禮物。

## 天干合化是什麼？

八字裡有五組特殊的天干組合，碰在一起會產生化學反應：

► 甲 + 己 → 合化土（穩定的結合）
► 乙 + 庚 → 合化金（精煉的結合）
► 丙 + 辛 → 合化水（流動的結合）
► 丁 + 壬 → 合化木（成長的結合）
► 戊 + 癸 → 合化火（熱情的結合）

如果兩個人的日主剛好是合化組合，代表天生有一種吸引力。很多長久的關係都有這個特徵。

## 常見問題

Q: 分數低代表不適合在一起嗎？
A: 不代表。分數反映的是「多系統共識度」，不是合不合。有些最好的關係是互補型，各系統看法不同反而代表關係有豐富的面向。好的關係不一定同頻，但一定有意識地經營。

Q: 可以測跟老闆或同事的合盤嗎？
A: 當然可以。任何兩人之間都有能量互動。事業合盤看的是工作上的互補性和潛在衝突點，很實用。

Q: 兩個人都是同一天生日怎麼看？
A: H.O.U.R. 四維會完全同頻（90分），靈數也一樣（80分）。同天生日的人確實有很強的共鳴，但也可能卡在同樣的盲點裡。互相提醒盲區是最重要的功課。

## 三個解讀範例

範例一：總分 88，天干合化金
兩人有天然的化學反應（乙庚合化金）。五行是「生我」關係，B 在滋養 A。H.O.U.R. 四維中 H 和 O 都很高（90, 85），代表情感和思維都同步。這是一組高度共振的關係。

範例二：總分 62，無合化
五行是「剋我」關係，有壓力和挑戰。但 U 維度很高（85），代表兩人在成長方向上一致。這種關係不輕鬆，但很有深度。雙方都在這段關係裡快速成長。

範例三：總分 73，天干合化水
有化學反應但五行是「比和」（同類）。兩人很像，容易互相理解，但也可能缺少互補的刺激。R 維度偏低（55），代表做事步調不同。需要在行動層面多溝通和協調。
