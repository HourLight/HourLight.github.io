## marvel 或 Zastrology 板

**標題：[分享] 免費的33合1命理計算器，全部線上算**

最近找到一個蠻特別的命理網站，跟版友分享一下。

它的功能是一次輸入生日時辰，同時跑 33 套命理系統的計算。
完全免費，不用註冊，計算結果直接顯示在頁面上。

因為我自己平常就有在研究命理，八字紫微占星都有在看，
所以看到這個工具的時候蠻好奇的，實際測了幾次，
把計算結果跟我自己手動排的盤做比對，準確度還行。

簡單列一下它有哪些系統，
我把比較多人會用到的標出來：

→ 八字：年月日時四柱，含十神、大運流年
→ 紫微斗數：命宮、身宮、十二宮完整排盤
→ 西洋占星：太陽月亮上升，含相位
→ 人類圖：類型、策略、權威、通道
→ 瑪雅曆：KIN、圖騰、調性
→ 生命靈數：生命路徑數、卓越數
→ 七政四餘：中國古典占星
→ 卡巴拉：生命之樹路徑
→ 吠陀占星：Nakshatra、Dasha

以上是我比較熟悉的，其他還有大六壬、九星氣學、
梅花易數、奇門遁甲、河洛理數等等。
完整清單 33 套，有些系統我自己也是第一次看到。

比較有意思的是，當多套系統同時指向同一個方向的時候，
那個資訊的可信度會比只看單一系統高很多。
概念有點像做研究的交叉驗證。

它另外還有一個合盤功能，兩個人的資料丟進去，
33 套系統同時跑相容度分析，這個我覺得蠻實用的。

3 張牌 NT$199、5 張 NT$399、7 張 NT$599。
