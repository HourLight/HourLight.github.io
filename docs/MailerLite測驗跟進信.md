## 技術整合備註

目前 hl-email.js 訂閱模組需要新增功能：
- 測驗完成時自動加標籤 `quiz_complete` 到 MailerLite
- 可透過 MailerLite API 在測驗結果頁觸發：`POST /api/subscribers/{id}/tags`
- 或透過 hl-tracker.js 追蹤事件後，用 MailerLite webhook 同步標籤

建議 D06 技術工程部評估最佳整合方式。
