# 時空尋跡 教學文件

## 這是什麼？（跟小孩解釋版）

想像你的人生是一條河。

上游是你帶來的東西（過去的原型）。
河面是你現在站的地方（今天的節氣和牌卡）。
下游是你要去的方向（未來三個月的趨勢）。

時空尋跡就是站在河邊，同時看見上游、腳下和下游。

如果上游、河面、下游的水都是同一種顏色，那就是你的「生命主旋律」。代表你的過去、現在和未來，正在說同一件事。那個訊號特別強。

## 運作原理（一步一步）

### 過去：馥靈 H 數原型
1. 用農曆生日計算馥靈 H 數（農曆月份 + 農曆日期，縮減到一位數）
2. H 數 1-9 對應九種原型：開拓者、調停者、表達者、建造者、流浪者、守護者、求真者、掌權者、大愛者
3. 同時看出生時的月相（農曆幾號 → 新月/上弦/滿月/下弦）

### 現在：節氣 + 牌卡 + H.O.U.R. 階段
1. 偵測當下的二十四節氣和對應的五行
2. 根據生日和今天的日期，用固定種子抽出一張牌卡主題
3. 你選的 H.O.U.R. 階段反映當下的心理狀態

### 未來：個人年月數
1. 計算個人年數（生命靈數 + 今年年份數字和，縮減到一位數）
2. 個人月數 = 個人年數 + 月份，縮減到一位數
3. 算出未來三個月的個人月數和對應主題

### 生命主旋律
如果過去、現在、未來有三個以上的時段指向同一個五行，那就是生命主旋律。

## 月相怎麼看？

農曆日期直接對應月亮的盈虧：

► 初一：新月（水）。黑暗中的開始。帶著重啟的力量。
► 初二到初七：上弦漸盈（木）。月亮在長大，帶著成長的能量。
► 初八：上弦月（木）。半月，行動和決策的力量。
► 初九到十四：漸滿（火）。接近圓滿，展現和表達。
► 十五：滿月（火）。最大的光明，完成和被看見。
► 十六到二十二：下弦漸虧（金）。開始收斂，整理和反思。
► 二十三：下弦月（金）。放手的力量。
► 二十四到二十九：殘月（水）。沉澱和轉化。
► 三十：晦月（土）。最後的靜默，醞釀下一次開始。

## 常見問題

Q: H.O.U.R. 階段怎麼選？
A: 直覺選。不用想太久。H 是身心校準（覺得累就選這個），O 是智慧辨識（想看清楚就選這個），U 是潛能解鎖（想突破就選這個），R 是行動進化（準備好要動就選這個）。

Q: 未來三個月的預測準嗎？
A: 這不是預測，是趨勢。就像天氣預報告訴你接下來是雨季，不代表每天都下雨，但你會帶傘。個人月數告訴你的是「能量的底色」，不是「會發生什麼事」。

Q: 沒有出現生命主旋律代表什麼？
A: 代表你的過去、現在和未來正在經歷不同的主題。這不是壞事，是豐富。每一段都有它自己的功課。沒有主旋律的人，通常是多面向發展的人。

## 三個解讀範例

範例一：H 數 3（表達者），節氣立春（木），未來三月皆木/火
過去帶來表達的能量，現在節氣也在催動成長（木），未來三個月也是成長和表達的主題。生命主旋律是「木」。這個人這段時間就是要「說出來」和「長出來」。任何壓抑和停滯都會感覺特別不舒服。

範例二：H 數 7（求真者），節氣霜降（土），未來三月金/水/木
過去帶著追問真相的模式，現在節氣說要簡化（土），未來三個月分別是收斂、流動、成長。沒有明顯的主旋律。這個人正在經歷多元的變化期，不同月份有不同的節奏。建議不要用同一套策略應對所有月份。

範例三：H 數 1（開拓者），選了 R 階段，個人年數 8
過去是開拓者，現在選了行動進化，個人年數是力量和決策的 8。三條線全部指向「動起來」。如果這個人還在猶豫某件事，所有系統都在說：不要再想了，去做。
