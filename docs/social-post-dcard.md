## Post #1｜心理測驗板

**標題：我做了101個心理測驗之後發現的事**

我知道這標題聽起來很誇張，但真的是 101 個。

事情是這樣的，我前陣子發現一個網站上面有超多心理測驗，不是那種「選一個顏色看你今天運勢」的那種喔，是真的有在跑計分邏輯的。

一開始只是無聊點進去做 MBTI，結果做完它推薦我做依附型態，做完依附型態又跳出愛之語，然後九型人格、大五人格、DISC，一個接一個，我就這樣掉進去了。

花了大概兩個禮拜（對，我很閒），全部做完。

做到後面開始出現一件很有趣的事：不同測驗的結果開始互相印證。

比如說我 MBTI 測出來是 INFP，然後依附型態是焦慮型，愛之語是肯定的言語。把這三個放在一起看，突然就懂了為什麼我每次吵架都會先道歉，即使錯的不是我。

不是因為我沒骨氣。是因為我的人格特質讓我把「關係和諧」看得比「誰對誰錯」重要太多了。

我挑三個我覺得最準、最值得做的：

→ 依附型態測驗：做完你會秒懂自己在感情裡為什麼老是重複同一個模式。超痛但超值得。

→ 富命覺醒測驗：這個比較特別，不是測你有沒有錢，是測你對「值得擁有」這件事的信念。我測完才發現我潛意識裡覺得自己不配過好日子，嚇到。

→ 關係羅盤：不只測愛情，是測你在所有關係裡的定位。家人、朋友、同事那些全部涵蓋。

101 個測驗做下來，最大的收穫不是知道自己是什麼型。是終於看懂那些重複出現的模式是怎麼來的。

## Post #2｜星座板

**標題：把八字紫微占星人類圖一次算完是什麼體驗**

好，我承認我是那種什麼都要算的人。

星座看了、八字排了、紫微也去廟裡問過、人類圖朋友幫我跑的、生命靈數 IG 上面算的。然後每次結果都不太一樣，我就很困惑到底要聽誰的。

前幾天滑到一個網站，它可以一次跑 33 套命理系統。你沒看錯，三十三套。

我第一個反應是：蛤，是不是唬爛的。

然後我就去試了。輸入生日時辰，它真的一次全部跑出來。八字、紫微斗數、西洋占星、人類圖、瑪雅曆、七政四餘、生命靈數、卡巴拉、吠陀占星，還有一些我根本沒聽過的系統。

說一下我的感受。

八字說我今年偏財運不錯但要注意健康，紫微說我天機化忌容易想太多，占星說我土星正在回歸，人類圖說我是 2/4 投射者。

分開看的話，就是各說各話。但放在一起看的時候，出現了一個很明顯的交集：今年的主題就是「停下來想清楚再動」。四五套系統都在講同一件事。

這個感覺很不一樣。以前看單一系統總覺得「也許吧」，但當好幾套都指向同一個方向的時候，那個「也許」就變成「好吧確實是這樣」了。

它還有一個合盤功能，可以把兩個人的資料丟進去，33 套系統同時跑相容度。我跟我男友試了一下，結果真的蠻準的，那些我們老是吵架的點，系統全部標出來了哈哈哈。
