## 🗝️ 技術對接流程

### 1. 產生「每人一組」的 HRC7 代碼

走 admin-dashboard 新功能：**「沉睡名單批次產碼」**

- 自動列出 Firestore `users` 裡 30 天沒 `lastLogin` 的會員
- 按「批次產碼」→ 每位會員配一組 HRC7 代碼（7 張牌，NT$599）
- 匯出 CSV：`email, displayName, unlock_code`

### 2. 匯入 MailerLite

- MailerLite 後台 → Subscribers → Import CSV
- 自訂欄位對映：`unlock_code`（新欄位，type: text）
- 或直接建一個 group：「Awaken-20260419」

### 3. 建立並群發 email

- Campaigns → New → Regular
- Template：貼上上面那段 HTML
- MailerLite 會自動把 `{{fields.unlock_code}}` 替換成每人專屬代碼
- From name：王逸君（馥靈之鑰）
- Subject 建議：「我是誰？」或「逸君最近有點沮喪」（真實不推銷）

### 4. 追蹤

- UTM：5 個熱搜 CTA 都帶 `?src=wake`
- GA4 事件：`session_source=wake` 追回流轉化
- MailerLite 後台看開信率 / 點擊率 / bounce
