## 整套劇本的語感重點

這套劇本跟原素材檔有 3 個重要差別，是為了符合馥靈語感：

**1. 拿掉毒舌與批判**
原素材裡「妳根本不是在找答案」「大數據照妖鏡」「妳在無意識複製父母的悲劇」這類句子全部刪掉。那是 notebookLM 的行銷腔，不是逸君會說的話。

**2. 沒有「您不是 X，是 Y」句型**
原素材頻繁用這個否定安慰句型（例：「妳不是運氣差，妳是不敢拿回主控權」）。逸君品牌明確禁用。全部改寫。

**3. 收尾淡，不煽情**
原素材最後一封叫「深度沉睡者」、用「我們不把能量浪費在裝睡的人身上」作結，太狠。改寫成「每個人的節奏不一樣，有人剛好那段日子特別忙」，不攻擊客人，但仍保持名單清理的動作邏輯。
