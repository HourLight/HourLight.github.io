━━━ 設定 ━━━

## MailerLite 設定摘要

| 項目 | 設定值 |
|------|--------|
| Automation 名稱 | 馥靈喚回序列 v1.0 |
| 觸發器 | Subscriber is inactive for 14 days（無開信＋無點擊） |
| 寄件人名稱 | 逸君｜馥靈之鑰 |
| 寄件人信箱 | info@hourlightkey.com |
| A/B 測試 | 每封信主旨三選一 |
| 自動清理 | 30天無互動 → 低頻群組；60天 → 自動退訂 |
| 追蹤 | UTM: utm_source=mailerlite&utm_medium=email&utm_campaign=reengagement |

## 三個序列的完整流程圖

```
新訂閱者
  │
  ▼
[歡迎序列 5封 / 10天]
  │
  ├─ 有互動 ─→ newsletter_active ─→ 定期電子報
  │                                      │
  │                                      ├─ 做了測驗 ─→ [測驗跟進 3封 / 7天]
  │                                      │
  │                                      └─ 14天沒互動 ─→ [喚回序列 3封 / 16天]
  │                                                          │
  │                                                          ├─ 喚回成功 → newsletter_active
  │                                                          └─ 喚回失敗 → inactive_30d → 60天退訂
  │
  └─ 歡迎序列就沒互動 ─→ 直接進入喚回序列
```

## 標籤系統整理

| 標籤 | 用途 | 自動加/手動 |
|------|------|------------|
| welcome_sequence | 正在歡迎序列中 | 自動 |
| welcome_sequence_complete | 歡迎序列完成 | 自動 |
| quiz_complete | 完成過任一測驗 | 需技術整合 |
| quiz_followup_complete | 測驗跟進序列完成 | 自動 |
| newsletter_active | 活躍訂閱者 | 自動 |
| reengagement_complete | 喚回序列完成 | 自動 |
| inactive_30d | 30天無互動 | 自動 |
| member_free | 免費會員 | Firebase 同步 |
| member_plus | 馥靈鑰友 | Firebase 同步 |
| member_pro | 馥靈大師 | Firebase 同步 |

## 技術整合待辦（D06 技術工程部）

1. hl-email.js 訂閱時同步 MailerLite 群組 + 基礎標籤
2. 測驗完成事件（hl-tracker.js）觸發 MailerLite 標籤 quiz_complete
3. Firebase 會員等級變更時同步 MailerLite 標籤（member_free/plus/pro）
4. 各 CTA 連結加上 UTM 參數，GA4 可追蹤 email 帶來的轉化
