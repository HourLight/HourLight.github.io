📜 源頭脈絡

中國人對名字的重視，遠比「五格剖象法」古老得多。孔子說「名不正則言不順」，這句話在兩千五百年前就把名字的份量定了調。但把姓名拆成筆畫、配上數理吉凶的這套系統，歷史其實沒那麼長。

熊崎健翁與五格剖象法

1918年，日本人熊崎健翁根據中國易學的數理概念，創立了「五格剖象法」，也叫「熊崎氏姓名學」。核心邏輯是把姓名的筆畫數按照特定規則拆成五格（天格、人格、地格、外格、總格），再用81個數理的吉凶表來判斷。

這套系統在日本流行之後，1936年由台灣留日學生白玉光（後改名白惠文）帶回台灣翻譯推廣。此後傳遍韓國、東南亞、港澳和中國大陸，成為華人圈最廣泛使用的姓名分析法。

要誠實說的是：五格剖象法在學術圈爭議極大。1937年日本大學教授吉武竹雄寫了《姓名與命運》，用當時首相近衛文麿的姓名數理極凶卻平步青雲為例，直接質疑這套系統的準確性。批評者指出，五格法不分男女、不考慮八字喜用神、且以康熙字典繁體筆畫為標準但實際書寫用簡體，存在邏輯矛盾。

但它流行的原因也很實際：簡單、好學、容易程式化。一個名字輸入，幾秒鐘就出結果。這讓它成為網路時代傳播最快的姓名學工具。

五格在算什麼

以康熙字典的筆畫數為基準，把姓名拆成五個數字：

天格 = 姓氏筆畫 + 1（單姓）或兩字相加（複姓）。代表先天運，來自家族。

人格 = 姓氏最後一字 + 名字第一字的筆畫。整個姓名的核心，影響一生主運。

地格 = 名字所有字的筆畫相加。代表前半生（36歲前）的運勢。

外格 = 總格 - 人格 + 1。代表外在環境和社交運。

總格 = 姓名所有字的筆畫相加。代表後半生的運勢。

每個格的數字對應81數理表中的吉凶評價。天格、人格、地格三者的五行關係叫「三才配置」，被認為是影響健康和事業的關鍵。

康熙字典筆畫

為什麼要用康熙字典？因為五格剖象法是在繁體字環境下建立的。康熙字典（1716年完成）是清代官修的標準字書，收字47,035個，被姓名學界視為筆畫計算的標準。

但有個實際問題：很多字的康熙筆畫和日常書寫不同。比如「艸」部在康熙字典裡算6畫，但一般人寫草字頭算3或4畫。這種落差會直接影響五格的計算結果。本站使用康熙字典標準筆畫。

不只五格

中國傳統姓名學遠不止五格。更古老的方法包括：根據八字五行的缺失來取名（補五行），用字形的象形意義判斷（宋代邵康節《梅花易數》：「木瘦金方水主肥，土形敦厚背如龜」），用字音聲韻感受氣場，以及拆字法（測字術）。這些方法在中國流傳上千年，比五格法早了很多。

本站的五格分析提供的是「熊崎氏」這一套系統的計算結果。它是姓名學的一個切入角度，不是全部。

📡 科學錨點

「名字會不會影響一個人的人生？」這個問題在心理學裡有一個專門的研究領域，叫做「姓名心理學」（onomastics / name psychology）。

最被引用的現象之一是「名字字母效應」（Name Letter Effect），由比利時心理學家 Jozef Nuttin 在1985年首次發表。他發現人們對自己名字裡包含的字母有無意識的偏好，而且這種偏好會影響到生活中的選擇（比如居住城市、職業、甚至結婚對象的名字）。後續研究結果不一，有些被質疑是統計假象，但這個方向的研究至今仍在進行。

另一個有趣的發現是「名字的刻板印象效應」。2017年希伯來大學的研究發現，人們看到一張陌生人的臉時，能以高於隨機機率的準確度猜出對方的名字。研究者認為這暗示名字和外貌之間存在某種社會建構的關聯：人們會無意識地「長成」自己名字給人的印象。

五格剖象法本身沒有經過科學驗證。但「名字影響人生」這件事，心理學確實在認真研究。影響的路徑不是神秘學的「數理磁場」，而是社會心理學的「標籤效應」和「自我實現預言」。您的名字被別人叫了幾十萬次，每一次呼喚都在強化一個關於「您是誰」的暗示。

🌿 馥靈馥語

名字是聲波。每天被叫幾十次，一輩子被叫幾十萬次。那個聲音在身體裡留下的痕跡，比任何數理都實在。

人格數屬火的人，名字裡帶著一股推進的力量。玫瑰天竺葵的花香甜而不膩，像在那股衝勁裡加了一層柔軟的緩衝，讓力量到位但不傷人。

人格數屬水的人，名字裡帶著流動和適應。檀香的沉穩像在水裡放了一塊石頭，不是擋住水流，是讓流動有個可以繞的中心點。

人格數屬木的人，名字裡帶著往上長的意思。佛手柑的清亮像早上的陽光透過葉子，不是催促生長，是讓光照到該照的地方。
