用 AI 做線上預約系統
客人自己約，你不用接電話

一邊做指甲一邊接電話，你崩潰過幾次？
讓客人自己選時間，你專心做好手上的服務。

馥靈之鑰美業 AI 學苑｜王逸君

你現在的預約流程有多崩潰？

客人 LINE 你：「下週三下午有空嗎？」你放下手上的客人，拿起手機看行事曆，打字回：「三點可以喔。」她說：「那四點呢？」你又要看一次。來回三四次，十五分鐘就沒了。

更慘的是，有些客人問完就消失了。你花時間回覆，她可能只是隨便問問。而你手上的客人在等你，你卻一直低頭滑手機。

線上預約系統解決的就是這件事：客人自己看你的空檔、自己選時間、自己填資料。你只需要收到通知，確認就好。甚至不用確認，系統自動搞定。

場景：美睫師小茜的改變

小茜以前每天花快一小時在回覆預約訊息。後來她用 Calendly 做了一個預約頁面，連結放在 IG 和 LINE 的自動回覆裡。現在客人自己約好時間，她打開手機就看到排好的行程表。一個月省下來的時間，夠她多接三四個客人。

方案 A：Google Forms 快速版（10 分鐘）

最快、最簡單的方法。用 Google 表單做一個預約單，客人填好你就收到通知。缺點是不能顯示你的空檔，客人填了你可能要再確認時間。但作為起步，完全夠用。

Prompt 模板：請 AI 幫你規劃表單欄位
我是一個美甲師，想用 Google Forms 做一個線上預約表單。請幫我規劃表單欄位和設定方式。 我的服務項目： - 基礎凝膠（90分鐘）NT$1,200 - 日式手繪（120分鐘）NT$1,800 - 卸甲＋基礎保養（60分鐘）NT$600 - 足部凝膠（90分鐘）NT$1,500 營業時間：週二到週六 10:00-19:00（週日週一公休） 請包含： 1. 完整的欄位清單（姓名、電話、LINE ID、想預約的服務、偏好日期時段、備註） 2. 每個欄位建議用什麼類型（單選/複選/簡答/日期選擇） 3. 表單描述文字（要親切不要制式） 4. 自動回覆信的內容（確認收到 + 會在 2 小時內回覆確認） 5. 怎麼設定 Google Sheets 自動整理預約資料
加分技巧

在表單的最後加一個問題：「你是從哪裡知道我們的？」選項放 IG / LINE / 朋友介紹 / Google 搜尋。這樣你就知道客人從哪來，以後知道要把力氣花在哪裡。

方案 B：Calendly 進階版（30 分鐘）

Calendly 是專門做預約的工具，免費版就很夠用。它會自動顯示你的空檔，客人選時間之後直接確認，不用你來回溝通。還會自動同步到你的 Google 行事曆。

設定步驟

去 calendly.com 註冊一個帳號，用 Google 帳號登入最方便，因為它會自動讀取你 Google 行事曆的空檔。

登入之後，點「Create Event Type」。Event name 填你的服務名稱，Duration 填服務時間。在 Availability 裡面設定你的營業時間。

設定好之後，Calendly 會給你一個專屬連結。把這個連結放到你的 IG 主頁、LINE 自動回覆、個人網頁的預約按鈕裡。

Prompt 模板：用 AI 規劃 Calendly 設定
我要用 Calendly 設定線上預約系統，請幫我規劃完整的設定方式。 我的服務： - 基礎凝膠（90分鐘）NT$1,200 - 日式手繪（120分鐘）NT$1,800 - 卸甲＋基礎保養（60分鐘）NT$600 營業時間：週二到週六 10:00-19:00 每個服務之間需要 15 分鐘緩衝時間（清潔＋準備） 每天最多接 5 個客人 請告訴我： 1. Calendly 的 Event Type 怎麼設定（每個服務各一個？還是合在一起？） 2. Availability 怎麼設定才能自動排除已預約時段 3. Buffer time（緩衝時間）怎麼設定 4. 預約確認信和提醒信的內容怎麼寫（用中文，親切口吻） 5. 怎麼把 Calendly 連結嵌入到我的網頁裡 6. 免費版的限制有哪些，什麼情況需要升級付費版

方案 C：嵌入你的個人網頁

如果你已經有了個人網頁（用第一篇教的方法做的），你可以把預約功能直接嵌進去。客人在你的網頁上就能完成預約，不用跳到另一個頁面，體驗更流暢。

Prompt 模板：在網頁嵌入預約功能
這是我的個人網頁程式碼： （貼上你的 index.html 內容） 請幫我在「預約」區塊加入 Calendly 的嵌入式預約功能。 我的 Calendly 連結：（貼上你的 Calendly 連結） 要求： 1. 用 Calendly 的 inline embed 方式（不是彈出視窗） 2. 嵌入區塊高度至少 650px 3. 上面加一行文字：「選擇你想預約的服務和時間」 4. 下面加一行備註：「預約完成後會收到確認信，如需更改請提前 24 小時聯繫」 5. 手機版要自適應高度 6. 配色跟網頁整體風格一致

預約系統上線後的關鍵設定

自動回覆訊息

客人預約完之後，一定要有一封確認訊息。不管你用哪個工具，確認訊息要包含：預約的服務名稱、日期時間、地點或到店方式、注意事項（例如做指甲前不要擦護手霜）、取消或更改的方式。

預約提醒

客人很容易忘記預約。在預約前一天自動發一封提醒訊息，放客率會降很多。Calendly 有內建提醒功能，Google Forms 的話你需要手動提醒或用 Google Apps Script 自動化。

Prompt 模板：預約確認和提醒訊息
幫我寫兩封預約相關的訊息模板，用在 LINE 或 email 自動回覆。語氣親切溫暖，不要太制式。 訊息一：預約確認（客人剛預約完立刻收到） 包含：感謝預約、服務名稱、日期時間、地點、到店前注意事項、取消方式 訊息二：預約前一天提醒 包含：溫馨提醒明天有預約、時間地點、期待見面 我的店名：（你的品牌名） 地址：（你的地址） 取消政策：請提前 24 小時通知，當天取消需收取 50% 費用

處理客人的特殊需求

有些客人會在預約表單的備註欄寫：「我對某某過敏」「我想看款式再決定」「可以帶朋友一起來嗎」。這些備註你都要看，而且要在預約確認訊息裡回應。讓客人知道你有在看她寫的東西，她會對你的印象加很多分。

進階：LINE 官方帳號整合預約

很多美業人的客人都在 LINE 上。與其讓客人離開 LINE 去另一個網頁預約，不如直接在 LINE 裡面就能約。

LINE 官方帳號的「圖文選單」可以放一個預約按鈕，按下去直接跳到你的 Calendly 或 Google 表單。這樣客人從頭到尾都不用離開 LINE。

Prompt 模板：LINE 官方帳號預約整合規劃
我有一個 LINE 官方帳號，想把預約功能整合進去。請幫我規劃。 目前狀況： - 客人加我的 LINE 官方帳號後，我都是手動回覆 - 我已經有 Calendly 預約連結 - 客人人數大約每月 40-50 個預約 請幫我規劃： 1. 圖文選單怎麼設計（6 格：預約 / 服務項目 / 作品集 / 營業時間 / 常見問題 / 聯絡我們） 2. 自動回覆關鍵字設定（客人打「預約」自動回覆預約連結） 3. 加好友歡迎訊息的內容 4. 每個格子點進去之後要顯示什麼內容 請用親切口吻寫所有文字內容，不要太正式。
