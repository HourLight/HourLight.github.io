## 🔢 生命靈數計算器

### 功能說明
輸入出生日期，計算出：
- **生命靈數**（1-9）：你的核心能量
- **天賦數**：你的天生才能
- **命運數**：你的人生方向
- **靈魂數**：你的內在渴望

### 計算邏輯
- 將出生日期的所有數字相加，化為單一數字（1-9）
- 例如：1990-05-15 → 1+9+9+0+0+5+1+5=30 → 3+0=3

### 使用方式
1. 上傳到網站根目錄
2. 訪問 `https://你的網站.com/lifepath-calculator.html`
3. 或在任何頁面加上連結：
   ```html
   <a href="lifepath-calculator.html">🔢 生命靈數計算器</a>
   ```

## 🌀 瑪雅曆計算器

### 功能說明
輸入出生日期，找到你的瑪雅印記：
- **KIN數字**（1-260）：你的瑪雅生日印記
- **圖騰**（20種）：你的核心能量象徵
- **調性**（13種）：你的能量頻率
- **波符**：你的能量波動
- **顏色**：紅、白、藍、黃

### 20個瑪雅圖騰
紅龍、白風、藍夜、黃種子、紅蛇、白世界橋、藍手、黃星星、紅月、白狗、藍猴、黃人、紅天行者、白巫師、藍鷹、黃戰士、紅地球、白鏡、藍風暴、黃太陽

### 計算邏輯
- 基於1987年7月26日為KIN 1的基準日
- 計算出生日期與基準日的天數差
- 對260取餘數，得到KIN數字
- 根據KIN數字計算圖騰、調性、波符

### 使用方式
1. 上傳到網站根目錄
2. 訪問 `https://你的網站.com/maya-calculator.html`
3. 或在任何頁面加上連結：
   ```html

   ```
