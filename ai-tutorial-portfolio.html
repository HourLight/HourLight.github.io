用 AI 做專業作品集
讓客人看完就想預約

你的作品散落在手機相簿裡，沒人看得到。
一個好的作品集，比一百張 IG 貼文更有說服力。

馥靈之鑰美業 AI 學苑｜王逸君

作品集跟 IG 有什麼不同？

IG 是流動的。你今天發一張圖，三天後就被新貼文淹沒了。客人想找你之前做過的那個款式，要往上滑很久，通常就放棄了。

作品集是固定的。它是你精選過的、分好類的、有說明的作品展示。客人一進來就能看到你最拿手的東西，不用在一堆限動和日常貼文裡面自己挖。

更重要的是，IG 上你的作品跟別人的混在一起。客人滑你的圖的同時，演算法在推薦你的競爭對手。但作品集是你自己的地盤，客人的注意力百分之百在你身上。

場景：為什麼客人總是問「有沒有圖可以參考」

每個美甲師都被客人問過這句話。客人到店裡之後才開始翻手機找圖，你們一起花二十分鐘溝通款式。如果你有一個整理好的作品集，客人預約前就已經看過了，到店裡直接指著說「我要這個」。省下來的時間就是你的利潤。

先整理，再排版

在讓 AI 幫你做作品集頁面之前，你要先做一件事：從手機裡挑出你最滿意的 12 到 20 張作品照。不要全部都放，挑精華就好。寧可少但張張都好看，不要多但品質參差不齊。

分類策略

把作品分成 3 到 5 個類別。分類方式取決於你的客群在乎什麼。

如果你的客人大多是上班族，就按場合分：日常款、約會款、婚禮款、派對款。

如果你的客人喜歡追風格，就按風格分：日系清新、韓系簡約、歐美個性、經典法式。

如果你的服務項目差異大，就按服務分：凝膠美甲、手繪設計、美甲修護、足部保養。

選圖的標準

不是做得最複雜的就最好。選那些「客人看了會覺得自己也可以做的」。太誇張的款式會嚇到新客人，她們會覺得「這個老師可能只擅長很浮誇的」。日常也好看的作品，反而最容易帶來預約。

拍出好看的作品照

你不需要買單眼相機，手機就夠了。但拍法很重要。光線、角度、背景這三件事決定了一張照片是「隨手拍的紀錄」還是「專業的作品照」。

Prompt 模板：美甲作品拍照技巧
我是美甲師，想用手機拍出專業的作品照放在作品集上。請給我完整的拍照指南。 場地條件： - 我的工作桌靠窗，有自然光 - 桌面是白色的 - 手邊有一些小飾品可以當道具 請包含： 1. 光線建議（什麼時段拍最好、用不用補光、怎麼避免反光） 2. 角度建議（俯拍/側拍/特寫各適合什麼款式） 3. 手部姿勢（5 個好看的手部擺法，白話文描述我能聽懂的） 4. 背景道具建議（花、飾品、布料怎麼搭不會太雜） 5. 後製建議（手機內建修圖就好，要調哪些參數） 6. 構圖技巧（怎麼讓一張照片看起來有質感） 請用清單條列，每條建議都附一句「為什麼這樣做」的簡短理由。

用 AI 生成作品集頁面

照片整理好之後，就可以讓 AI 幫你做頁面了。跟做個人網頁一樣，一個 Prompt 搞定。

Prompt 模板：生成作品集頁面
幫我做一個美甲作品集頁面，用純 HTML + CSS + JavaScript，一個檔案。 作品分類（做成可切換的篩選按鈕）： - 全部 - 日系清新（6 張） - 簡約質感（5 張） - 特殊設計（4 張） - 足部作品（3 張） 功能需求： 1. 頂部有品牌名稱和一句話介紹 2. 篩選按鈕列（點擊切換分類，有動畫過渡） 3. 瀑布流圖片排列（手機一欄、平板兩欄、電腦三欄） 4. 點擊圖片彈出大圖（lightbox 效果） 5. 每張圖下面有款式名稱和簡短說明（一行就好） 6. 底部有「喜歡哪個款式？預約體驗」按鈕 設計要求： - 手機優先 - 米白底 #faf9f7 + 大地色系 - 圖片先用 placeholder（300x400 的灰色方塊加數字標號） - 篩選按鈕有 active 狀態（底線或填色） - 整體風格乾淨高級，不花俏 - 載入時有淡入動畫

上傳照片和替換 Placeholder

跟做網頁那篇一樣，把照片上傳到 Imgur 或其他圖床，拿到網址之後讓 AI 幫你替換。

Prompt 模板：替換作品集照片
這是我的作品集頁面程式碼： （貼上你的 portfolio.html 內容） 請幫我把 placeholder 圖片替換成真正的作品照： 日系清新： 1. https://i.imgur.com/（圖1）.jpg — 櫻花暈染 2. https://i.imgur.com/（圖2）.jpg — 透明感果凍 （以此類推） 簡約質感： 1. https://i.imgur.com/（圖3）.jpg — 裸色漸層 （以此類推） 每張圖的 alt 標籤請寫「（品牌名）美甲作品 - （款式名稱）」，有助於 SEO。

讓作品集幫你賣東西

作品集不只是「展示」，它的終極目的是讓客人預約。所以每一個環節都要設計好出路。

每張作品下面加價格區間

客人最怕的是不知道多少錢。看到好看的款式但不知道價格，很多人就不會問了。在作品下面直接寫「NT$1,200 起」，客人的心理門檻就降低了。

加一個「指定這款」的按鈕

客人看到喜歡的款式，如果要自己去找預約頁面、自己描述想要什麼，太麻煩了。直接在每張作品旁邊放一個按鈕，按下去自動帶入款式名稱跳到預約頁面。

放上最新作品的日期

如果作品集上面的照片都是半年前的，客人會覺得你是不是沒在做了。在頁面某個地方標一行「最後更新：2026 年 4 月」，讓人知道你還很活躍。

Prompt 模板：加上轉換功能
這是我的作品集頁面程式碼： （貼上程式碼） 請幫我加上以下轉換功能： 1. 每張作品圖下面加價格標籤（格式：NT$X,XXX 起） 2. 每張作品加一個「指定這款預約」小按鈕，按下去跳到預約連結，URL 後面帶上 ?style=（款式名稱） 3. 頁面頂部加「最後更新」日期 4. 底部的預約按鈕改成兩個：「LINE 諮詢」和「線上預約」 5. 加一個浮動按鈕在右下角：「立即預約」（捲動時固定顯示） 預約連結：（你的 Calendly 或預約頁面連結） LINE 連結：（你的 LINE 連結）

作品集的更新節奏

不需要每天更新，但建議每個月換一到兩張新作品。把最舊的或表現最差的換掉，放上你最近最得意的。

另一個好方法是跟著季節換。春天放粉嫩色系，夏天放透明亮色，秋天放大地暖色，冬天放深色和亮片。客人每次進來看到的東西不一樣，就會一直回來逛。

如果你做了某個比較特別的款式，客人也很滿意，可以多拍幾個角度。好的作品值得被好好展示。
