排班管理

排班表看起来简单，做起来是最花时间的苦差事。特别是人一多、班別一复杂，光是「誰可以換班」就要来回确认好几次。把格式和逻辑丟给 AI，你只需要填人名和時段。

週排班表产生器排班管理
我经营一家【美甲/美发/SPA/美容】店，有【N】位伙伴。 營業时间：【週一到週六 10:00-21:00】 班別：早班 10:00-16:00 / 晚班 15:00-21:00 / 全天班 10:00-21:00（休息 1 小時） 伙伴名单与限制： - 【小玲】：週三固定休 - 【阿珮】：只上早班 - 【小涵】：週一到週五都可以 规则： 1. 每天至少 【2】人在场 2. 每人每週休 【1-2】 天 3. 盡量不要連上 6 天 請幫我产出下週（【日期】）的排班表，用表格呈现，标示每日人力是否充足。
修改提示：把【】里的资訊換成你的实际狀況。如果有特殊节日或旺季，可以加一行「本週预计較忙的日子：週六」。
月排班表规划排班管理
我需要规划【月份】的整月排班。 店舖资訊：【美甲/美发/SPA】，營業【週一到週六】 伙伴：【列出所有名字】 每人每月休【4-6】天，国定假日【正常營業/休息】 特殊需求： - 月初月底是帳单日，至少【3】人在场 - 【伙伴名】在【日期】请假 - 週六是旺日，全员到齐 請产出整月表格，並在最下方统计每人总工時和休假天數。如果有排班冲突請标示出来。
月排班建议先产出初版，再根据伙伴反馈微调。跟 AI 说「小玲的班改一下」就好。
调班通知讯息排班管理
幫我写一段调班通知讯息，可以直接貼到 LINE 群组。 原本班表：【小玲】週三早班 调整为：【小涵】週三早班，【小玲】改到週四早班 语气要求：简潔、礼貌、不要太正式。让伙伴一看就知道改了什麼，不用回头翻班表。 最后加一行：如有问题請在今天下班前回复。
调班讯息最重要的是「誰」「什麼時候」「改成什麼」三个资訊一目了然。
加班申請与纪录表排班管理
幫我设计一个加班申請表格模板，包含以下栏位： 1. 申請人姓名 2. 申請日期 3. 加班日期与時段 4. 加班原因（简述） 5. 预计加班時數 6. 主管核准栏 另外幫我做一个月度加班统计表，栏位包含： - 伙伴姓名 / 本月加班总時數 / 加班费计算（時薪 ×【1.33 / 1.67】倍）/ 累计金额 請用表格格式呈现，方便我直接貼到 Google Sheets。
加班费倍率依劳基法：前 2 小時 1.33 倍、第 3 小時起 1.67 倍。如果你們有自己的计算方式，改括号里的數字就好。

收支记录

很多美業人記帳的方式就是看存摺尾數。不是不想記，是覺得表格很煩。用 AI 产出框架之后，每天只要花两分鐘填數字就好。你会开始看懂自己的钱到底花去哪里了。

每日营收記帳表收支记录
幫我设计一个美業店面的每日营收記帳表。 栏位需求： - 日期 / 时间 - 客人編号或暱稱（不用全名，保护隐私） - 服务项目 - 服务金额 - 付款方式（现金/轉帳/LINE Pay/信用卡） - 加購品項（如有） - 备註 底部自动加总： - 当日总营收 - 现金 vs 非现金佔比 - 服务人次 請用 Google Sheets 可以直接贴上的表格格式。一週七天做成七个分頁模板。
記帳的重点不是記得多详细，而是「每天都有記」。先从最简单的版本开始，习惯了再加栏位。
月度收支报表收支记录
根据以下资料，幫我产出【月份】的月度收支报表。 收入项目： - 服务收入：NT$【金额】 - 产品销售：NT$【金额】 - 储值卡收入：NT$【金额】 - 其他（礼券/合作）：NT$【金额】 支出项目： - 房租：NT$【金额】 - 人事成本（薪资+勞健保）：NT$【金额】 - 材料/耗材：NT$【金额】 - 水电瓦斯：NT$【金额】 - 营销费用：NT$【金额】 - 其他（雜支）：NT$【金额】 請幫我做出： 1. 收支总表（含毛利、净利、毛利率） 2. 与上月比较（如有上月數字：收入【金额】/ 支出【金额】） 3. 一段 3-5 行的分析摘要，告訴我这个月的经营重点
第一次用可以先估算，不用很精确。重要的是建立每月看一次报表的习惯。
服务成本分析收支记录
幫我分析以下服务项目的成本结构。 服务名稱：【例：日式凝胶美甲 - 基础款】 售价：NT$【1,200】 所需时间：【90】分鐘 直接成本： - 凝胶材料：约 NT$【80】 - 装饰材料（亮粉/贴纸等）：约 NT$【30】 - 一次性耗材（棉片/锉刀等）：约 NT$【20】 间接成本（月分攤）： - 月租金：NT$【金额】÷ 每月可服务人次【人次】 - 伙伴時薪：NT$【金额】× 服务時數 請幫我算出： 1. 每筆服务的总成本 2. 毛利和毛利率 3. 如果要维持 60% 毛利率，建议售价是多少 4. 每月需要接多少客人才能打平固定成本
很多人做了好几年才发现某些服务其实在虧钱。花 5 分鐘算一次，会改变你的定价策略。
服务定价计算器收支记录
我想重新檢視店里的定价。以下是我目前的服务清单： 【列出 3-5 个主要服务和目前售价】 例： 1. 基础保养 NT$800（60 分鐘） 2. 凝胶美甲 NT$1,200（90 分鐘） 3. 手足保养组合 NT$1,800（120 分鐘） 我的目标月收入：NT$【金额】 每月可工作天數：【天】 每天可服务人數：【人】 請幫我： 1. 计算目前每小時产值 2. 建议哪些服务可以调整价格 3. 设计一个阶梯式定价策略（基础/进阶/VIP） 4. 如果要涨价，给我一段跟客人说明的话术
定价不是看同行收多少就跟。你的技术、地段、服务品质都是变數。AI 幫你算完，你会更有底氣定价。

客户管理

客人来了一次就不见了，通常不是你服务不好，而是你忘了她。美業的回购率取決於你在「她离开之后」做了什麼。下面的模板幫你把客户管理变成半自动的。

客户档案模板客户管理
幫我设计一个美業客户档案模板，可以存在 Google Sheets 或 Notion。 基本资訊：暱稱 / 联络方式 / 生日（月份就好）/ 来店管道（IG/朋友介绍/Google搜寻） 服务纪录： - 日期 / 服务项目 / 金额 / 负责伙伴 - 肤质/甲型/发质备註 - 个人喜好（偏好颜色/风格/禁忌） - 特殊注意事項（过敏/不適纪录） 回购追踪： - 上次到店日 - 预计回訪时间（依服务周期自动提醒） - 累计消费金额 - VIP 等级（依消费金额分级） 請做成可以直接复制贴上 Google Sheets 的格式，第一行是标题列。
一开始不用填太多，暱稱、服务项目、日期就夠了。慢慢养成记录习惯，半年后你会发现这张表是最值钱的资产。
回购追踪与提醒讯息客户管理
我有一批客人超过【30/45/60】天沒回来了。幫我写三种不同风格的回訪讯息，可以直接貼到 LINE 发送。 客人类型 A：消费 3 次以上的常客 客人类型 B：只来过 1 次的新客 客人类型 C：之前很常来但突然消失 2 个月以上的 要求： 1. 语气温暖但不推销，像朋友传讯息 2. 不要用「好久不见」开头（太老套） 3. 可以帶一个小誘因（例如本月限定优惠/新品体验） 4. 长度控制在 3-5 行，手机看剛好不用滑 每种类型给我 2 个版本，我可以交替使用。
回訪讯息最忌諱的是制式感。如果你还記得客人的某个细节（「上次你说要去日本旅行，回来了嗎？」），加进去效果翻倍。
服务满意度问卷客户管理
幫我设计一份美業服务后的满意度问卷，用 Google Forms 的格式。 问题（6 题以内，客人才会填完）： 1. 整体服务满意度（1-5 分） 2. 环境舒适度（1-5 分） 3. 伙伴的专业程度（1-5 分） 4. 这次服务中最喜欢的部分是？（简答） 5. 有什麼可以让你下次体验更好的？（简答） 6. 你愿意推荐给朋友嗎？（愿意/考慮中/不愿意） 另外幫我写一段邀請客人填写的 LINE 讯息，语气轻松、不要太正式。讯息里要说明填完大概只要 1 分鐘。
问卷不要超过 6 题，客人的耐心大概就到这里。如果想多了解什麼，面对面聊比问卷有效。
客人生日提醒与祝福客户管理
幫我做两件事： 第一，设计一个「本月壽星」提醒表格： 栏位：客人暱稱 / 生日月份 / 上次到店 / VIP 等级 / 建议的生日礼 第二，幫我写 3 个版本的生日祝福讯息（LINE 用）： - 版本 A：纯祝福，不帶任何营销 - 版本 B：附帶生日专属优惠（【9 折/送护手霜/加送 10 分鐘按摩】） - 版本 C：VIP 客人的尊榮版（语气更私人一点） 要求：不要用「祝你生日快樂」开头，太普通了。要让客人覺得「她真的記得我」。
生日讯息提前 3-5 天发效果最好，因为客人还来得及安排到店时间。

库存管理

材料快用完了才发现要訂貨，等到貨又要一个礼拜。库存管理听起来很企業，其实就是「知道你有什麼、还剩多少、什麼時候要补」。用 AI 把模板做出来，每週花 10 分鐘更新就好。

进货采购清单库存管理
幫我设计一个美業店面的进货采购清单模板。 分类方式： - 凝胶/甲片/装饰材料（美甲类） - 洗护产品/染劑/造型品（美发类） - 精油/基底油/耗材（SPA 类） - 一次性耗材（棉片/手套/口罩等） - 清潔用品 每項栏位：品名 / 品牌 / 容量規格 / 单价 / 目前库存量 / 安全库存量 / 需訂購數量 / 供应商 / 备註 安全库存量的计算：以【2 週】用量为基準。 請用表格格式，並在最上方加一行「本次采购预估金额」自动加总。
安全库存量设 2 週用量是基本款，旺季前建议拉到 3-4 週。
月度库存盘点表库存管理
幫我设计月度库存盘点表。 栏位： - 品項名稱 - 上月结餘數量 - 本月进货數量 - 本月使用量（自动计算：上月餘 + 进货 - 现有） - 月底实际盘点數量 - 差异（盤虧/盤盈） - 备註 最下方加一个摘要区： - 本月耗材总成本 - 平均每位客人耗材成本 - 耗损率（差异量 ÷ 总量） - 需要补货的品項清单 我的店是【美甲/美发/SPA】店，约【10-15】个主要品項，請幫我列出常见品項作为范例。
盘点日固定在每月最后一个營業日，养成习惯最重要。差异超过 5% 就要找原因。
产品效期追踪提醒库存管理
幫我建立一个产品效期追踪表。 栏位：品名 / 批号 / 开封日期 / 保质期限（月） / 到期日 / 狀态（正常/即将到期/已过期）/ 处理方式 规则： - 距离到期 30 天内 = 黃色警示「即将到期」 - 已过期 = 红色「已过期，停止使用」 - 开封后保质期和未开封不同的要分开标示 另外幫我写一段每月效期检查 SOP（5 步驟以内），让新进伙伴也能照著做。
精油和保养品的开封后效期通常比标示的短很多。养成开封就写日期的习惯，用油性筆直接写在瓶身上。
耗材用量预估库存管理
根据过去【3 个月】的使用量，幫我预估下个月的耗材需求。 最近三个月数据： - 【月份 A】：服务 【N】人，凝胶用量 【N】瓶，棉片 【N】包 ... - 【月份 B】：服务 【N】人，凝胶用量 【N】瓶，棉片 【N】包 ... - 【月份 C】：服务 【N】人，凝胶用量 【N】瓶，棉片 【N】包 ... 下个月预估服务人數：【N】人（考慮到【旺季/淡季/节日】） 請幫我： 1. 计算每項耗材的「每人平均用量」 2. 预估下月总需求量 3. 列出建议采购清单（含预估金额） 4. 标示哪些品項现有库存不足
预估不用太精确，抓个大方向就好。重点是让你知道該在月初还是月中补货。

目标管理

目标不是写在年初就放著长灰塵的东西。好的目标管理是每个月回头看一次，知道自己走到哪里了、哪里需要调整。AI 可以幫你把模糊的期望变成可追踪的數字。

月度目标设定表目标管理
幫我设定【月份】的营运目标。 背景资訊： - 上月营收：NT$【金额】 - 上月服务人次：【人】 - 上月新客佔比：【%】 - 目前伙伴人數：【人】 我希望这个月： - 营收目标：NT$【金额】（成长 【%】） - 新客人數：【N】位 - 回购率提升到：【%】 請幫我： 1. 把月目标拆成每週里程碑 2. 列出达成目标需要的具体行动（每个目标 2-3 个行动） 3. 设计一个简单的追踪表格（每週填一次） 4. 如果中途落后，给我 3 个「急救方案」
目标设定的訣竅是「踮腳尖就能摸到」，太高会放棄，太低沒动力。成长 10-15% 是合理的挑战。
季度营运檢討目标管理
幫我做一份季度营运檢討报告。 Q【N】数据（【月-月】）： - 总营收：NT$【金额】（目标 NT$【金额】） - 总服务人次：【人】 - 新客 vs 回訪比例：【比例】 - 客单价平均：NT$【金额】 - 最畅销服务：【服务名】 - 最冷门服务：【服务名】 - 客诉案件：【N】件 請幫我产出： 1. 达成率总表（每个目标的实际 vs 目标） 2. 本季三个做对的事情（继续保持） 3. 本季三个需要改善的地方（下季行动） 4. 下一季的三大优先事項建议 5. 用 200 字做一段整体摘要
季度檢討最重要的不是數字好不好看，而是你有沒有从數字里看出「为什麼」。
年度经营计划目标管理
幫我擬定【年度】的美業经营计划。 现況： - 开业年资：【N】年 - 目前月均营收：NT$【金额】 - 伙伴人數：【N】人 - 主要客群：【描述】 - 最大挑战：【描述】 年度愿景（用一句话）：【例：从一人工作室扩展到三人团队，月营收稳定 15 万以上】 請幫我规划： 1. 四季主题（Q1 打基础 / Q2 冲業績 / Q3 优化 / Q4 丰收，或你建议的主题） 2. 每季 2-3 个关键目标 + 數字指标 3. 全年营销行事曆（配合节日/換季/开学季等） 4. 投资规划（设备/进修/营销的预算建议） 5. 风险预案（淡季怎麼撐、伙伴离职怎麼办）
年度计划不用写得像公司企划书，重点是让自己每个月翻开来知道「这个月該做什麼」。
每週行动追踪表目标管理
幫我设计一个每週行动追踪表，格式简单到我愿意每天花 2 分鐘填。 栏位： - 日期 - 今天最重要的 1 件事（事先填） - 完成狀态（是/否/部分） - 今日营收 - 服务人次 - 一句话心得或筆記 底部统计： - 本週完成率 - 本週总营收 - 本週服务总人次 - 下週要调整的 1 件事 請做成 7 天的表格（週一到週日），方便我每週一列印出来或貼到 Notion。
「今天最重要的 1 件事」是整张表的灵魂。不是做最多事的人贏，是做对事情的人贏。

团队管理

从一个人做到帶团队，最难的不是技术传承，而是怎麼让每个人知道「做到什麼程度算好」。绩效不是用来扣薪水的，是用来让伙伴知道自己哪里可以更好的。

伙伴绩效考核表团队管理
幫我设计一个美業伙伴的月度绩效考核表。 考核面向（各佔比）： 1. 服务品质（30%）：客诉率、客人回购率、满意度分數 2. 業績表现（30%）：个人营收、客单价、推荐成功數 3. 专业成长（20%）：新技术学习、作品产出、考證进修 4. 团队合作（20%）：出勤稳定度、同事互评、主动支援 评分方式：每項 1-5 分 总分对应： - 4.5+ 优秀（奖金/加薪候选） - 3.5-4.4 良好（维持） - 2.5-3.4 待加強（列出改善计划） - 2.5 以下 需面談 請做成表格，每个面向有具体评分项目，底部有主管评语栏和伙伴自评栏。
考核最忌諱的是「平常不说，考核才翻舊帳」。有问题当下就沟通，考核只是确认而已。
新人培训计划表团队管理
幫我设计一个新进伙伴的培训计划，为期【4 週】。 店舖类型：【美甲/美发/SPA】 新人背景：【有/无】相关经验 培训架构： 第 1 週：环境熟悉 + 基本 SOP + 观摩 第 2 週：基础操作 + 模擬练习 第 3 週：在资深伙伴陪同下接客 第 4 週：独立接客 + 反馈 每天的培训内容要包含： - 今日学习主题 - 具体练习项目 - 评量方式（观察/口試/实作） - 通过标准 另外附一份「新人手册检查清单」，确保所有 SOP 都有教到。
好的培训计划让新人知道「我现在在哪里、下一步要做什麼」，而不是「老板叫我做什麼我就做什麼」。
店内会议纪录模板团队管理
幫我设计一个店内週会纪录模板，控制在 30 分鐘内开完。 会议结构： 1. 上週回顧（5 分鐘）：上週數字 + 上週决议执行狀況 2. 本週重点（10 分鐘）：本週目标 + 排班确认 + 特殊客人提醒 3. 问题讨论（10 分鐘）：伙伴提出的问题，每人限 1 个 4. 下週预告（5 分鐘）：活动/到貨/请假 纪录格式： - 日期/时间/出席者 - 每个议题的结论和负责人 - 待办事項清单（誰/做什麼/什麼時候完成） 语气要求：简潔明快，不要会议报告的正式感。像在群组里发一段「今天开会講了这些」就好。
週会最怕变成「聊天大会」。把时间框住，超時的议题就下次再说或另外找人讨论。
一对一面談引导表团队管理
幫我设计一个跟伙伴一对一面談的引导表。每季一次，每次 20-30 分鐘。 面談问题设计（不要太正式，像聊天一样）： 工作感受： - 最近工作上覺得最有成就感的事情是什麼？ - 有沒有什麼事情让你覺得卡住的？ 成长方向： - 接下来三个月想学什麼新技术或新能力？ - 需要我提供什麼支援？ 团队与环境： - 你覺得店里有什麼可以更好的地方？ - 跟其他伙伴的合作順利嗎？ 未来规划： - 你未来想走什麼方向？（技术深耕/管理/创业） - 有什麼想跟我说但平常不方便说的？ 纪录栏位：面談日期 / 面談重点摘要 / 约定的行动项目 / 下次追踪
面談的目的不是考試，是让伙伴覺得「老板在乎我」。70% 的时间让对方说，你听就好。
