Claude Code 是什么？小白也能用吗｜一个美业老板的真心话

作者：王逸君（馥灵之钥创办人｜美业 34 年资历）｜2026.04.11

先讲结论：Claude Code 就像你请了一个住在电脑里的工程师。你跟他说你要什么，他就帮你做。不是给你建议而已，是真的动手帮你做出来。

我身边有不少美业朋友开始用 AI 工具了。写文案、做图、回客人讯息，这些大家比较熟。但很多人不知道，AI 现在已经可以帮你做「技术活」了，像是改网站、建系统、批次处理资料，这些以前一定要请工程师做的事。

这篇文章是写给不是技术背景、但想知道 Claude Code 到底能干嘛的人看的。

Claude Code 是什么

Claude 是 Anthropic 这家公司做的 AI。Claude Code 是他们推出的一个工具，让 Claude 可以直接操作你电脑上的档案。

用比喻来说：Claude.ai（就是你在网页上跟 AI 聊天那个）像是一个很聪明的顾问。你问他问题，他给你答案和建议。但他不会帮你动手。

Claude Code 像是你请了一个工程师坐在你旁边。你说「帮我把这个网页的标题改成红色」，他就直接帮你改。你说「帮我建一个预约系统」，他就真的写出程式码，建出一个可以用的系统。

差别在于：一个是用嘴巴帮你，一个是用手帮你。

跟 Claude.ai 到底差在哪里

这是很多人搞不清楚的地方。让我用最简单的方式讲。

Claude.ai 适合的事情：聊天、问问题、写文案、想点子、分析资料、翻译。你丢一段文字给它，它回一段文字给你。像传 LINE 一样。

Claude Code 适合的事情：改网站、写程式、建工具、批次处理档案、整理资料库。你告诉它你要什么结果，它直接去动你电脑上的档案。

举个例子。假设你的网站有几十页，你想把所有出现「某个旧名称」的地方统一改成新名称。如果用 Claude.ai，你要一页一页贴给它看，它告诉你哪里要改，你自己手动去改。几十页可能要改好几天。

用 Claude Code，你说一句「帮我把所有 HTML 档案里的旧名称改成新名称」。它几分钟搜完全部档案、列出要改的地方、一次改完、检查有没有改坏东西、然后自动上传到网站。

三分钟。

什么时候用 Claude Code

如果你有以下任何一种需求，Claude Code 可能帮得上你。

你想改自己的网站，但不会写程式。你想做一个线上工具（计算机、测验、预约系统），但请人太贵。你有很多档案要批次处理（改名、整理、转档）。你想把某个重复的工作自动化。

具体来说，Claude Code 能做的事包括：从零开始建一个完整的品牌官网、做各种计算工具（把不同系统整合在一起）、做互动测验系统（有计分逻辑的那种）、串接线上金流（让客人可以直接付钱）、做课程投影片（HTML 格式的教学简报）、建预约系统、做会员管理后台。

这些事如果请外面的工程师做，保守估计几十万起跳。

什么时候用 Claude.ai 就好

如果你的需求是：想一个社群文案、分析一段文字、翻译一篇文章、讨论一个商业点子、写一份报告。用 Claude.ai 就够了。

Claude.ai 的优势是快、方便、不需要设定。打开网页就能用。你不需要安装任何东西。

我自己的使用习惯是：想东西的时候开 Claude.ai，做东西的时候开 Claude Code。想的时候需要一个对话的对象，做的时候需要一个动手的伙伴。

不是工程师也能用吗

可以。但要讲实话：有一点学习曲线。

Claude Code 的操作介面是终端机（就是那个看起来很骇客的黑色视窗）。你不需要打很技术的指令，大部分时候就是用中文告诉它你要什么。但你需要会基本的档案操作：知道你的档案放在哪里、知道怎么打开终端机、知道怎么切换资料夹。

这些东西大概花一两天就能学会。YouTube 上搜「终端机基础教学」就有很多。

我妈今年 75 岁了，她会用 ChatGPT 帮她写 LINE 回覆。她不会用 Claude Code，但她证明了一件事：AI 的使用门槛，比你以为的低很多。

Claude Code 的门槛比 ChatGPT 高一点，但比学程式低非常多。你不需要会写程式，你只需要会描述你要什么。「帮我做一个页面，上面有一个表格，列出我们家的服务和价格」。这种程度的描述就够了。

多少钱

Claude Code 需要订阅 Anthropic 的 Max 方案。目前有两种：每月 100 美元和每月 200 美元。

100 美元的额度对一般使用来说够用。如果你是重度使用（像我一样每天都在改网站），200 美元比较保险。

值不值得？看你的时间值多少。

如果你原本每个月花 5,000 到 10,000 元请人改网站、做设计、处理技术问题，那 100 美元（大约 3,200 元台币）的 Claude Code 订阅费，可能比你原本的花费还少。而且它 24 小时在线，不会已读不回，不会拖工期。

如果你目前没有任何技术需求，只是想聊天和写文案，那 Claude.ai 的免费版或 Pro 版（每月 20 美元）就够了，不需要上 Claude Code。

我的真心话

用了 Claude Code 之后，我最大的感受不是「效率变高了」。是「我能做的事变多了」。

以前有很多想法，但卡在「我不会做」。想做一个心理测验？要请工程师。想在网站上加一个新功能？要请工程师。想改一下版面？还是要请工程师。每一个「想做」都要先翻译成工程语言，然后等人家排期，然后来回沟通好几次，最后做出来的可能跟你想的不一样。

现在是：我想到什么，直接跟 Claude Code 讲。它做出来的东西如果不对，我当场就说「不是，我要的是这样」。三分钟后就改好了。

这种「想到就能做到」的感觉，对一个创业者来说，比什么都值钱。

当然，Claude Code 不是万能的。它有时候会做错，有时候会理解偏差，有时候改完一个地方会不小心弄坏另一个地方。你需要学会怎么跟它沟通、怎么检查它的产出、怎么在它出错的时候导正。

但跟请真人工程师比起来，这些都是小事。至少它不会突然消失、不会谈到一半说要加价、不会在你最急的时候跟你说他在忙别的案子。

常见问题

Claude Code 跟 Claude.ai 有什么不同？

Claude.ai 是聊天介面，你问问题它回答。Claude Code 可以直接操作你电脑上的档案，帮你改程式、建网站、批次处理资料。一个用嘴巴帮你，一个用手帮你。

不是工程师也能用 Claude Code 吗？

可以，但需要学基本的终端机操作（大约一两天）。你不需要会写程式，只需要会用中文描述你要什么。「帮我做一个表格列出服务和价格」这种程度就够了。

Claude Code 要多少钱？

需要 Claude Max 方案，月费 100 美元或 200 美元。100 美元方案的额度足够一般使用。如果你原本每月花超过这个数字在技术外包上，Claude Code 可能更划算。

「工具不会取代你，但会放大你。」
