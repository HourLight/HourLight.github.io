用 AI 做线上预约系统
客人自己约，你不用接电话

一邊做指甲一邊接电话，你崩潰过几次？
让客人自己选时间，你专心做好手上的服务。

馥灵之钥美業 AI 学苑｜王逸君

你现在的预约流程有多崩潰？

客人 LINE 你：「下週三下午有空嗎？」你放下手上的客人，拿起手机看行事曆，打字回：「三点可以喔。」她说：「那四点呢？」你又要看一次。来回三四次，十五分鐘就沒了。

更慘的是，有些客人问完就消失了。你花时间回复，她可能只是随便问问。而你手上的客人在等你，你卻一直低头滑手机。

线上预约系统解決的就是这件事：客人自己看你的空档、自己选时间、自己填资料。你只需要收到通知，确认就好。甚至不用确认，系统自动搞定。

场景：美睫师小茜的改变

小茜以前每天花快一小時在回复预约讯息。后来她用 Calendly 做了一个预约页面，链接放在 IG 和 LINE 的自动回复里。现在客人自己约好时间，她打开手机就看到排好的行程表。一个月省下来的时间，夠她多接三四个客人。

方案 A：Google Forms 快速版（10 分鐘）

最快、最简单的方法。用 Google 表单做一个预约单，客人填好你就收到通知。缺点是不能显示你的空档，客人填了你可能要再确认时间。但作为起步，完全夠用。

Prompt 模板：請 AI 幫你规划表单栏位
我是一个美甲师，想用 Google Forms 做一个线上预约表单。請幫我规划表单栏位和设定方式。 我的服务项目： - 基础凝胶（90分鐘）NT$1,200 - 日式手繪（120分鐘）NT$1,800 - 卸甲＋基础保养（60分鐘）NT$600 - 足部凝胶（90分鐘）NT$1,500 營業时间：週二到週六 10:00-19:00（週日週一公休） 請包含： 1. 完整的栏位清单（姓名、电话、LINE ID、想预约的服务、偏好日期時段、备註） 2. 每个栏位建议用什麼类型（单选/复选/简答/日期选择） 3. 表单描述文字（要亲切不要制式） 4. 自动回复信的内容（确认收到 + 会在 2 小時内回复确认） 5. 怎麼设定 Google Sheets 自动整理预约资料
加分技巧

在表单的最后加一个问题：「你是从哪里知道我們的？」选项放 IG / LINE / 朋友介绍 / Google 搜寻。这样你就知道客人从哪来，以后知道要把力氣花在哪里。

方案 B：Calendly 进阶版（30 分鐘）

Calendly 是专门做预约的工具，免费版就很夠用。它会自动显示你的空档，客人选时间之后直接确认，不用你来回沟通。还会自动同步到你的 Google 行事曆。

设定步驟

去 calendly.com 注册一个账号，用 Google 账号登入最方便，因为它会自动读取你 Google 行事曆的空档。

登入之后，点「Create Event Type」。Event name 填你的服务名稱，Duration 填服务时间。在 Availability 里面设定你的營業时间。

设定好之后，Calendly 会给你一个专属链接。把这个链接放到你的 IG 主頁、LINE 自动回复、个人网页的预约按钮里。

Prompt 模板：用 AI 规划 Calendly 设定
我要用 Calendly 设定线上预约系统，請幫我规划完整的设定方式。 我的服务： - 基础凝胶（90分鐘）NT$1,200 - 日式手繪（120分鐘）NT$1,800 - 卸甲＋基础保养（60分鐘）NT$600 營業时间：週二到週六 10:00-19:00 每个服务之间需要 15 分鐘緩冲时间（清潔＋准备） 每天最多接 5 个客人 請告訴我： 1. Calendly 的 Event Type 怎麼设定（每个服务各一个？还是合在一起？） 2. Availability 怎麼设定才能自动排除已预约時段 3. Buffer time（緩冲时间）怎麼设定 4. 预约确认信和提醒信的内容怎麼写（用中文，亲切口吻） 5. 怎麼把 Calendly 链接嵌入到我的网页里 6. 免费版的限制有哪些，什麼情況需要升级付费版

方案 C：嵌入你的个人网页

如果你已经有了个人网页（用第一篇教的方法做的），你可以把预约功能直接嵌进去。客人在你的网页上就能完成预约，不用跳到另一个页面，体验更流暢。

Prompt 模板：在网页嵌入预约功能
这是我的个人网页程式碼： （贴上你的 index.html 内容） 請幫我在「预约」区块加入 Calendly 的嵌入式预约功能。 我的 Calendly 链接：（贴上你的 Calendly 链接） 要求： 1. 用 Calendly 的 inline embed 方式（不是彈出窗口） 2. 嵌入区块高度至少 650px 3. 上面加一行文字：「选择你想预约的服务和时间」 4. 下面加一行备註：「预约完成后会收到确认信，如需更改請提前 24 小時联系」 5. 手机版要自適應高度 6. 配色跟网页整体风格一致

预约系统上线后的关键设定

自动回复讯息

客人预约完之后，一定要有一封确认讯息。不管你用哪个工具，确认讯息要包含：预约的服务名稱、日期时间、地点或到店方式、注意事項（例如做指甲前不要擦护手霜）、取消或更改的方式。

预约提醒

客人很容易忘記预约。在预约前一天自动发一封提醒讯息，放客率会降很多。Calendly 有内建提醒功能，Google Forms 的话你需要手动提醒或用 Google Apps Script 自动化。

Prompt 模板：预约确认和提醒讯息
幫我写两封预约相关的讯息模板，用在 LINE 或 email 自动回复。语气亲切温暖，不要太制式。 讯息一：预约确认（客人剛预约完立刻收到） 包含：感謝预约、服务名稱、日期时间、地点、到店前注意事項、取消方式 讯息二：预约前一天提醒 包含：温馨提醒明天有预约、时间地点、期待见面 我的店名：（你的品牌名） 地址：（你的地址） 取消政策：請提前 24 小時通知，当天取消需收取 50% 费用

处理客人的特殊需求

有些客人会在预约表单的备註栏写：「我对某某过敏」「我想看款式再決定」「可以帶朋友一起来嗎」。这些备註你都要看，而且要在预约确认讯息里回應。让客人知道你有在看她写的东西，她会对你的印象加很多分。

进阶：LINE 官方账号整合预约

很多美業人的客人都在 LINE 上。与其让客人离开 LINE 去另一个网页预约，不如直接在 LINE 里面就能约。

LINE 官方账号的「图文菜单」可以放一个预约按钮，按下去直接跳到你的 Calendly 或 Google 表单。这样客人从头到尾都不用离开 LINE。

Prompt 模板：LINE 官方账号预约整合规划
我有一个 LINE 官方账号，想把预约功能整合进去。請幫我规划。 目前狀況： - 客人加我的 LINE 官方账号后，我都是手动回复 - 我已经有 Calendly 预约链接 - 客人人數大约每月 40-50 个预约 請幫我规划： 1. 图文菜单怎麼设计（6 格：预约 / 服务项目 / 作品集 / 營業时间 / 常见问题 / 联络我們） 2. 自动回复关键字设定（客人打「预约」自动回复预约链接） 3. 加好友欢迎讯息的内容 4. 每个格子点进去之后要显示什麼内容 請用亲切口吻写所有文字内容，不要太正式。
