让客人看完就想预约

作品集跟 IG 有什麼不同？

IG 是流动的。你今天发一张图，三天后就被新貼文淹沒了。客人想找你之前做过的那个款式，要往上滑很久，通常就放棄了。

作品集是固定的。它是你精选过的、分好类的、有说明的作品展示。客人一进来就能看到你最拿手的东西，不用在一堆限动和日常貼文里面自己挖。

更重要的是，IG 上你的作品跟別人的混在一起。客人滑你的图的同時，算法在推荐你的竞爭对手。但作品集是你自己的地盤，客人的注意力百分之百在你身上。

场景：为什麼客人总是问「有沒有图可以参考」

每个美甲师都被客人问过这句话。客人到店里之后才开始翻手机找图，你們一起花二十分鐘沟通款式。如果你有一个整理好的作品集，客人预约前就已经看过了，到店里直接指著说「我要这个」。省下来的时间就是你的利润。

先整理，再排版

在让 AI 幫你做作品集页面之前，你要先做一件事：从手机里挑出你最满意的 12 到 20 张作品照。不要全部都放，挑精华就好。寧可少但张张都好看，不要多但品质參差不齐。

分类策略

把作品分成 3 到 5 个类別。分类方式取決於你的客群在乎什麼。

如果你的客人大多是上班族，就按场合分：日常款、约会款、婚礼款、派对款。

如果你的客人喜欢追风格，就按风格分：日系清新、韓系简约、歐美个性、经典法式。

如果你的服务项目差异大，就按服务分：凝胶美甲、手繪设计、美甲修护、足部保养。

选图的标准

不是做得最复杂的就最好。选那些「客人看了会覺得自己也可以做的」。太誇张的款式会嚇到新客人，她們会覺得「这个老师可能只擅长很浮誇的」。日常也好看的作品，反而最容易帶来预约。

拍出好看的作品照

你不需要买单眼相机，手机就夠了。但拍法很重要。光线、角度、背景这三件事決定了一张照片是「随手拍的纪录」还是「专业的作品照」。

Prompt 模板：美甲作品拍照技巧
我是美甲师，想用手机拍出专业的作品照放在作品集上。請给我完整的拍照指南。 场地条件： - 我的工作桌靠窗，有自然光 - 桌面是白色的 - 手邊有一些小饰品可以当道具 請包含： 1. 光线建议（什麼時段拍最好、用不用补光、怎麼避免反光） 2. 角度建议（俯拍/側拍/特写各适合什麼款式） 3. 手部姿勢（5 个好看的手部擺法，白话文描述我能听懂的） 4. 背景道具建议（花、饰品、布料怎麼搭不会太雜） 5. 后製建议（手机内建修图就好，要调哪些參數） 6. 构图技巧（怎麼让一张照片看起来有质感） 請用清单条列，每条建议都附一句「为什麼这样做」的简短理由。

用 AI 生成作品集页面

照片整理好之后，就可以让 AI 幫你做页面了。跟做个人网页一样，一个 Prompt 搞定。

Prompt 模板：生成作品集页面
幫我做一个美甲作品集页面，用纯 HTML + CSS + JavaScript，一个档案。 作品分类（做成可切換的筛选按钮）： - 全部 - 日系清新（6 张） - 简约质感（5 张） - 特殊设计（4 张） - 足部作品（3 张） 功能需求： 1. 頂部有品牌名稱和一句话介绍 2. 筛选按钮列（点击切換分类，有动画过渡） 3. 瀑布流图片排列（手机一栏、平板两栏、电腦三栏） 4. 点击图片彈出大图（lightbox 效果） 5. 每张图下面有款式名稱和简短说明（一行就好） 6. 底部有「喜欢哪个款式？预约体验」按钮 设计要求： - 手机优先 - 米白底 #faf9f7 + 大地色系 - 图片先用 placeholder（300x400 的灰色方块加數字标号） - 筛选按钮有 active 狀态（底线或填色） - 整体风格乾淨高级，不花俏 - 載入時有淡入动画

上传照片和替換 Placeholder

跟做网页那篇一样，把照片上传到 Imgur 或其他图床，拿到网址之后让 AI 幫你替換。

Prompt 模板：替換作品集照片
这是我的作品集页面程式碼： （贴上你的 portfolio.html 内容） 請幫我把 placeholder 图片替換成真正的作品照： 日系清新： 1. https://i.imgur.com/（图1）.jpg — 櫻花暈染 2. https://i.imgur.com/（图2）.jpg — 透明感果凍 （以此类推） 简约质感： 1. https://i.imgur.com/（图3）.jpg — 裸色渐層 （以此类推） 每张图的 alt 标籤請写「（品牌名）美甲作品 - （款式名稱）」，有助於 SEO。

让作品集幫你賣东西

作品集不只是「展示」，它的终极目的是让客人预约。所以每一个环节都要设计好出路。

每张作品下面加价格区间

客人最怕的是不知道多少钱。看到好看的款式但不知道价格，很多人就不会问了。在作品下面直接写「NT$1,200 起」，客人的心理门槛就降低了。

加一个「指定这款」的按钮

客人看到喜欢的款式，如果要自己去找预约页面、自己描述想要什麼，太麻煩了。直接在每张作品旁邊放一个按钮，按下去自动帶入款式名稱跳到预约页面。

放上最新作品的日期

如果作品集上面的照片都是半年前的，客人会覺得你是不是沒在做了。在页面某个地方标一行「最后更新：2026 年 4 月」，让人知道你还很活躍。

Prompt 模板：加上转换功能
这是我的作品集页面程式碼： （贴上程式碼） 請幫我加上以下转换功能： 1. 每张作品图下面加价格标籤（格式：NT$X,XXX 起） 2. 每张作品加一个「指定这款预约」小按钮，按下去跳到预约链接，URL 后面帶上 ?style=（款式名稱） 3. 页面頂部加「最后更新」日期 4. 底部的预约按钮改成两个：「LINE 諮詢」和「线上预约」 5. 加一个浮动按钮在右下角：「立即预约」（捲动時固定显示） 预约链接：（你的 Calendly 或预约页面链接） LINE 链接：（你的 LINE 链接）

作品集的更新节奏

不需要每天更新，但建议每个月換一到两张新作品。把最舊的或表现最差的換掉，放上你最近最得意的。

另一个好方法是跟著季节換。春天放粉嫩色系，夏天放透明亮色，秋天放大地暖色，冬天放深色和亮片。客人每次进来看到的东西不一样，就会一直回来逛。

如果你做了某个比较特别的款式，客人也很满意，可以多拍几个角度。好的作品值得被好好展示。
