用 AI 做互动小工具
让客人在你的网页玩到不想走

你的 IG 作品很多人按讚，但沒人预约。
因为按讚只是路过，互动才是停留。

馥灵之钥美業 AI 学苑｜王逸君

为什麼需要互动工具？

你的 IG 每张图都有一两百讚，但转换成预约的有几个？可能一个月就那麼两三个。问题与其说你的作品不好，不如说客人滑过去就忘了。

互动工具的意思是，让客人在你的网页上「做一件事」。选一个颜色看適不适合自己、回答几个问题看自己适合什麼风格的美甲、输入膚色看哪个色号最显白。做完之后，她会覺得「这个美甲师好懂我」，然后自然就点预约了。

这跟心理测验的原理一样。人只要开始回答问题，就很难中途离开。而且做完之后会想分享结果，你的曝光就是这样来的。

场景：美甲师 Ami 的选色器

Ami 做了一个「你的手适合什麼颜色」的小工具放在网页上。客人选膚色色调、选平常穿衣风格、选场合，最后会配出三个推荐色。每天大概有二十几个人玩，其中五六个会直接点预约链接。比她在 IG 发限动的转换率高了三倍。

工具一：美甲风格配对器

这是最容易上手的互动工具。客人回答 4 到 5 个问题，系统根据答案配出最适合她的美甲风格。问题要有趣，不要像问卷调查，要像朋友在聊天。

Prompt 模板：美甲风格配对器
幫我做一个「你适合什麼风格的美甲」互动工具，用纯 HTML + CSS + JavaScript，一个档案。 规则： - 共 5 题选择题，每题 4 个选项 - 题目风格轻松有趣，像朋友聊天 - 每个选项对应一个风格分數（日系清新 / 韓系简约 / 歐美个性 / 经典法式） - 最后统计分數，显示最适合的风格 + 风格描述 + 推荐款式 - 结果頁要有「预约体验」按钮，链接到：（你的预约链接） - 结果頁要有「分享结果」按钮（复制结果文字） 题目方向： 1. 你今天的心情像什麼天氣？（晴天/微风/小雨/打雷） 2. 挑一个週末活动（咖啡厅看书/逛市集/夜店/野餐） 3. 你的手机桌布是什麼风格？（花草/极简/偶像/风景） 4. 朋友形容你最常用的詞？（可愛/帥氣/优雅/随性） 5. 选一个旅行目的地（京都/首爾/纽约/巴黎） 设计要求： - 手机优先，米白底 + 大地色系 - 一次只显示一题，选了自动跳下一题 - 有进度条 - 结果頁要有视觉冲击，配色要好看
进阶技巧

题目选项不要写得太明显。像「你喜欢日式美甲嗎？」这种一看就知道答案的问题沒有意义。要用间接问法，让客人透过生活喜好去对应风格，这样结果出来她才会覺得「好準」。

工具二：膚色选色器

很多客人不知道自己适合什麼颜色。冷色调的手塗暖色指甲油，做完覺得怪怪的但说不出来哪里不对。你可以做一个选色工具，让她选自己的膚色色调，系统推荐最搭的指甲油颜色。

Prompt 模板：膚色选色器
幫我做一个「什麼指甲油颜色最适合你」的选色工具，用纯 HTML + CSS + JavaScript，一个档案。 步驟流程： Step 1：选膚色色调（暖黃调 / 冷粉调 / 中性调 / 深膚色）用色块让她对比手腕内側选 Step 2：选场合（日常通勤 / 约会 / 重要场合 / 度假放松） Step 3：选偏好感覺（低调氣质 / 活潑亮眼 / 个性前卫 / 优雅经典） 结果： - 推荐 3 个指甲油色号（用色块显示）+ 每个色号的名稱和适合场景 - 加一句风格建议（例如：「你的膚色配上莓果色系会特别显白，约会的時候试试看」） - 底部有「预约体验这个颜色」按钮 设计要求： - 手机优先，极简风 - 色块要夠大，方便在手机上点选 - 结果頁的推荐色块要大且突出 - 米白底 + 大地色系 + 推荐色块用实际颜色

工具三：肤质自檢工具

这个特别适合做臉部护肤的美業人。客人自己在家用手机做肤质自檢，系统根据她的回答判断肤质类型，推荐适合的療程。重点是：她做完之后就已经被教育了，到店里不用再从头解释。

Prompt 模板：肤质自檢工具
幫我做一个「一分鐘肤质自檢」互动工具，用纯 HTML + CSS + JavaScript，一个档案。 共 6 题： 1. 早上洗完臉不擦任何东西，两小時后你的T字部位？（出油明显 / 微微出油 / 沒什麼感覺 / 感覺紧繃） 2. 你的两頰摸起来？（粗糙有顆粒 / 偶尔乾癢脫皮 / 还算滑順 / 容易泛红） 3. 換季的時候皮膚会？（瘋狂出油长痘 / 乾燥脫皮 / 沒什麼变化 / 又乾又油） 4. 你的毛孔在鼻头和臉頰看起来？（很明显 / 鼻头明显臉頰还好 / 不太明显 / 不太确定） 5. 用新的保养品皮膚容易？（长痘 / 过敏泛红 / 沒什麼反應 / 看产品而定） 6. 你覺得自己最大的皮膚困擾是？（出油和痘痘 / 乾燥和细紋 / 暗沉不均勻 / 敏感不稳定） 结果分类：油性肌 / 乾性肌 / 混合肌 / 敏感肌 每个结果包含： - 肤质说明（50字内，白话文） - 日常保养三步驟建议 - 推荐療程名稱（接你的服务项目） - 「立即预约肤质諮詢」按钮 设计：手机优先，米白底，一次一题自动跳轉，进度条

工具四：今日美甲灵感产生器

有些客人与其说不想做指甲，不如说不知道做什麼。她們会在 IG 滑半小時还是选不出来。你做一个「今日美甲灵感」产生器，她按一下按钮就随机跳出一个风格建议，附上你的作品参考。轻松好玩，而且会让她一直点。

Prompt 模板：美甲灵感产生器
幫我做一个「今日美甲灵感」随机产生器，用纯 HTML + CSS + JavaScript，一个档案。 功能： - 点击按钮，随机显示一个美甲风格建议 - 每个建议包含：风格名稱、风格描述（30字内）、推荐配色（用色块呈现3个颜色）、适合场合 - 准备 20 组不重复的风格建议 - 按钮文字：「換一个灵感」 - 底部固定一行：「喜欢这个风格？→ 预约体验」（链接到预约頁） 20 组风格范例方向： 櫻花渐層 / 焦糖奶茶 / 极光镜面 / 法式微笑线 / 莓果红酒 / 大理石紋 / 復古格紋 / 透明果凍 / 金箔点缀 / 乾燥花押花 / 貓眼极光 / 暈染水彩 / 几何色块 / 珍珠貝殼 / 啞光裸色 / 彩虹糖果 / 银河星空 / 蕾絲雕花 / 动物纹路 / 季节限定（当季主题） 设计： - 手机优先 - 卡片式设计，翻牌动画效果 - 米白底 + 大地色系 - 色块要圓形，排成一排

怎麼放到你的网页上

AI 生成的程式碼就是一个完整的 HTML 档案。你有两个选择。

第一个，如果你已经有了个人网页（按照上一篇教学做的），就在 GitHub 里新增一个档案，例如 style-match.html，把程式碼貼进去。然后在你的首頁加一个链接指向它就好。

第二个，如果你还沒有网页，直接把这个互动工具当成你的第一个页面也可以。先做一个有趣的工具让客人玩，比一个漂亮但静态的页面更有用。

转换率的关键

互动工具做好只是第一步，真正的转换来自结果頁的设计。结果頁要让客人覺得「被懂了」，然后自然看到预约按钮。不要在结果还沒出来之前就急著推销，那会让人反感。

怎麼让更多人来玩

工具做好之后，重点是让人知道它的存在。

在 IG 限动分享：「我做了一个小工具，测你适合什麼风格的美甲，链接在主頁」。这比你发作品照的互动率高得多，因为人天生喜欢跟自己有关的东西。

在 LINE 自动回复里放链接。新加你 LINE 的客人，第一条讯息就附上这个互动工具的链接。她玩完之后自然就知道你做什麼、擅长什麼。

结果頁一定要有分享按钮。客人做完测验之后想分享给朋友，你的工具就这样被传出去了。免费的口碑营销。
