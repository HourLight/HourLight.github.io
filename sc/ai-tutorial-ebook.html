用 AI 写你的第一本电子书
美業人的知识变现

你做了十几年美業，腦袋里的东西夠写三本书了。
问题不是沒东西写，而是不知道怎麼开始。

馥灵之钥美業 AI 学苑｜王逸君

为什麼美業人應該出一本书

你可能覺得出书是作家的事、是名人的事。但你想想，你每天在跟客人聊的那些，皮膚怎麼保养、指甲怎麼养护、肩頸为什麼痠、哪个精油适合什麼体质，这些全都是知识。而且是你花了好几年摸出来的真知识，不是网络上复制贴上的那种。

一本电子书可以做到三件事：让你的专业被看见、让你睡覺也在赚钱、让新客人在认识你之前就信任你。

以前出书要找出版社、要自己排版、要印刷。现在用 AI 写稿、用 Canva 排版、上架到 Readmoo 或 Amazon KDP，从头到尾不用花一毛钱，只花时间。

逸君的亲身经验

馥灵之钥创办人王逸君已经出了三本书。第一本《氣味記得我走过的路》，把 30 年芳疗经验写成散文。第二本《藏在农历生日里的秘码》，是馥灵秘码系统的完整介绍。第三本《AI 時代美業人的逆襲指南》，写给想用 AI 转型的美業人。三本书从构想到完稿，AI 都是重要的写作伙伴。

了解美業合作方案

Step 1：选一个你最会講的主题

不要想著写一本「什麼都有」的书。最好的电子书只講一件事，把那件事講透。

选主题有个简单的判断方式：你平常被客人问最多的那个问题，就是你的书。如果你是美甲师，客人老问「凝胶会不会傷甲」，那就写一本《美甲新手的 20 个真心话》。如果你是芳疗师，客人常问「精油怎麼选」，那就写《给忙碌女生的精油入门》。

1 用 AI 幫你厘清主题方向

如果你有好几个想法但不知道选哪个，把它們全部丟给 AI，让它幫你分析哪个最有市场。

Prompt 模板：主题评估
我是一位在台灣做了【N】年的【美甲师/美发师/芳疗师/SPA 美容师】。我想写一本电子书，目前有以下几个主题在考慮： 1. 【主题 A，例：美甲新手最常犯的错】 2. 【主题 B，例：居家精油保养完全指南】 3. 【主题 C，例：美業创业第一年生存手册】 我的目标读者是：【例：25-40 岁想学美甲保养的女生 / 想创业的美業新手】 請幫我分析： 1. 每个主题的市场需求（誰会买？有多大？） 2. 每个主题的竞品狀況（市面上已经有类似的书嗎？） 3. 我写哪个最有优勢（根据我的背景） 4. 最终推荐一个主题，说明理由 5. 给这个主题想 5 个书名（要吸引人但不浮誇）

Step 2：列出完整大綱

大綱就是这本书的骨架。骨架穩了，后面填肉就快了。

一本好的电子书大概 8-12 章，每章 2,000-3,000 字，整本大约 2-3 万字。不用太长，读者在手机上看的，太厚会放棄。

2 用 AI 产出书籍大綱
Prompt 模板：大綱产生
我要写一本电子书，主题是「【你选定的主题】」。 目标读者：【描述】 书的定位：【实用工具书 / 经验分享 / 入门指南 / 观念翻轉】 预计字數：约 2-3 万字（8-12 章） 语气风格：口语化、像朋友在聊天、不学术、有个人观点 請幫我列出完整大綱，包含： 1. 书名（3 个选项） 2. 副标题 3. 各章标题 + 每章 3-5 个重点（用一句话描述） 4. 前言的核心讯息（为什麼写这本书） 5. 结语的收尾方向 大綱要有节奏感：开头吸引人、中间给乾貨、后段给行动力。不要每章都是「什麼是XX」「XX的好处」这种教科书结构。
大綱确认要点

AI 产出的大綱一定要自己看过一遍。有些章节可能太理论，有些可能跟你的经验不符。大膽刪、大膽改。这是你的书，不是 AI 的书。

Step 3：逐章写作

拿到大綱之后，一章一章写。不要想著一次写完整本，每天写一章，10 天就写完了。

写作的关键是：你提供经验和观点，AI 负责组織和潤飾。你说「我有个客人怎样怎样」，AI 幫你把故事写順。你说「这个技术的重点是什麼什麼」，AI 幫你編排成读者容易吸收的结构。

3 逐章写作 Prompt
Prompt 模板：单章写作
现在幫我写这本书的第【N】章。 章节标题：「【标题】」 这章要涵盖的重点： 1. 【重点 A】 2. 【重点 B】 3. 【重点 C】 我的个人经验和素材（請融入写作中）： - 【你的真实故事、客人案例、自己踩过的坑】 - 【你对这个主题的獨特看法】 写作要求： - 字數约 2,500 字 - 语气像跟读者面对面聊天 - 每隔 3-4 段插入一个小标题，方便掃读 - 长短句交替，不要每句都很长 - 可以有一两句幽默或自嘲，但不要刻意搞笑 - 结尾给读者一个可以馬上做的小行动 注意：不要用「首先」「其次」「最后」「总之」这类过渡詞。
Prompt 模板：改写潤稿
以下是我写的初稿，請幫我改写。 【贴上你的初稿】 改写方向： 1. 保留我的观点和故事，不要改掉核心意思 2. 让句子更順暢，去掉贅字 3. 如果有太学术的地方，換成口语化的说法 4. 加入 2-3 个生活化的比喻 5. 段落太长的拆开，太短的合併 6. 确保读起来有呼吸感（长短句交替） 請给我改完的全文，不要只给修改建议。

Step 4：前言、结语和书籍简介

前言是读者打开书看的第一段文字，決定他要不要继续看下去。结语是他看完之后記住的最后一句话。书籍简介是他在书店页面決定要不要买的关键。这三个东西，花的时间應該跟写正文一样多。

4 前言与书籍简介
Prompt 模板：前言撰写
幫我写这本书的前言，800-1,000 字。 书名：「【书名】」 作者背景：我做【美甲/美发/芳疗】【N】年，【简述你的经历和轉折点】 写这本书的动机：【为什麼想写？是因为看到什麼问题？想传达什麼？】 前言要做到： 1. 第一段就抓住读者（用一个场景或问题开头，不要「大家好我是XXX」） 2. 让读者覺得「这个人懂我」 3. 誠实说出这本书能给什麼、不能给什麼 4. 结尾让人想翻到下一頁 语气：温暖但不煽情，有自信但不自大。像是一个前辈在跟你说「这条路我走过，我告訴你我看到了什麼」。
Prompt 模板：书籍简介（上架用）
幫我写这本书的书籍简介，用在 Readmoo/Amazon KDP 上架页面。 书名：「【书名】」 全书章节：【列出章节标题】 目标读者：【描述】 請写两个版本： 版本 A（200 字精简版）：用在书店页面的主要描述 版本 B（400 字完整版）：用在详细介绍区 要求： - 让沒听过你的人 10 秒内知道这本书在講什麼 - 列出 3-5 个「读完你会得到」的具体收穫 - 不要用「本书旨在」「筆者认为」这种正式用语 - 结尾用一句话製造购买冲动

Step 5：排版与封面设计

文字写完了，接下来是让它看起来像一本书。电子书的排版不用太复杂，重点是乾淨、好读、在手机上看起来舒服。

排版工具选择

Canva 是最简单的选择。它有免费的电子书模板，你只要把文字貼进去、換个颜色、选个字体就好了。如果你想更专业一点，可以用 Google Docs 排好之后导出 PDF 或 EPUB。

5 封面设计 Prompt
Prompt 模板：封面设计指导
我要用 Canva 设计电子书封面。 书名：「【书名】」 副标题：「【副标题】」 作者名：【你的名字】 书的调性：【专业温暖 / 清新活潑 / 高端质感 / 实用工具感】 目标读者偏好的视觉风格：【简约 / 花朵植物 / 几何图形 / 手繪感】 主色调偏好：【暖色系 / 冷色系 / 大地色 / 莫兰迪色】 請给我： 1. 3 个封面设计方案的文字描述（包含构图、配色、字体建议） 2. 在 Canva 搜寻模板時可以用的英文关键字（5 个） 3. 封面设计的 3 个注意事項（避免常见错誤） 4. 书名在封面上的排列建议（字级大小、位置）

Step 6：上架到平台

书写完了、排版好了，最后一步是让全世界都能买到它。两个主要平台：台灣读者用 Readmoo，海外读者用 Amazon KDP。两个都可以同時上架，不冲突。

Readmoo 上架流程

Readmoo 是台灣最大的电子书平台。到 Readmoo 的「自助出版」页面申請账号，上传 EPUB 格式的书稿，填写书籍资訊，设定价格，送审。审核通常 3-5 个工作天。分潤比例是你拿 70%，平台拿 30%。

Amazon KDP 上架流程

Amazon KDP（Kindle Direct Publishing）面向全球读者。到 kdp.amazon.com 注册账号，上传书稿（支援 EPUB、PDF、DOCX），设定价格和销售地区。定价在 USD 2.99-9.99 之间可以拿 70% 分潤，其他价格帶是 35%。

6 上架资料准备
Prompt 模板：上架资料
我的电子书要上架到 Readmoo 和 Amazon KDP，幫我准备以下资料。 书名：「【书名】」 副标题：「【副标题】」 作者：【名字】 书籍分类：【例：商業经营 / 生活风格 / 美容保养】 目标读者：【描述】 全书字數：约【N】字 定价考量：目标读者可接受的价格帶 請幫我准备： Readmoo 用： 1. 书籍简介（繁体中文，200 字 + 400 字各一版） 2. 分类建议（Readmoo 的分类选择） 3. 关键字标籤（5-7 个） 4. 建议定价（TWD） Amazon KDP 用： 1. 书籍简介（英文版 + 繁体中文版） 2. 分类建议（Amazon BISAC 分类 2 个） 3. 搜寻关键字（7 个，英文） 4. 建议定价（USD） 5. 销售地区建议

Step 7：上架之后的营销

书上架了不代表就有人会买。你需要让你的客人、追踪者、朋友都知道你出书了。

7 新书推广 Prompt
Prompt 模板：新书推广文案
我的电子书「【书名】」剛上架了。幫我写推广文案。 上架平台：【Readmoo / Amazon KDP / 两者都有】 购买链接：【链接】 定价：NT$【金额】 請幫我写以下版本： 1. IG 限時动态文字（3 张，每张 1-2 行） 2. IG 貼文（150 字以内 + 5 个 hashtag） 3. Facebook 长文（300 字，講出书背后的故事） 4. LINE 群发讯息（简短版，100 字以内） 5. 个人朋友圈/限动分享（自然不推销的版本） 语气：不要太销售感。我不是在賣书，我是在分享一个我覺得有用的东西。像朋友跟你说「欸我最近做了一个东西你看看」那种感覺。
定价建议

台灣市场的电子书甜蜜点大约是 NT$199-399。如果你的书偏实用工具型（有大量模板或方法论），可以定到 NT$349-499。如果是轻松的经验分享型，NT$199-249 比较容易让人按下购买。

可以在上架初期设定限時优惠价，例如第一週 NT$149 冲销量和评论，之后恢復原价。

准备好写你的第一本书了嗎
