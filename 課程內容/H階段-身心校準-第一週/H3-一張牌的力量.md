# H3｜一張牌的力量：城門銘牌五種覺察模式

## 影片大綱（建議 25-30 分鐘）

### 開場（2 分鐘）
「130 張牌卡，9 種牌陣，聽起來很多對嗎？但你今天只需要會一件事：用 1 張牌幫一個人做一次覺察。1 張牌做得好，比 28 張牌做得亂更有力量。」

### 第一段：城門銘牌 — 五種模式（10 分鐘）

1 張牌不是「只有一種用法」。城門銘牌有五種覺察模式（品牌聖經 p.6）：

**⚡ 快閃覺察 — 此刻一個提醒**
► 適合：「最近覺得怪怪的但說不上來」
► 問法：「你現在最在意的一件事是什麼？閉眼，想著它，翻牌。」
► 解讀重點：牌卡的「關鍵詞」就是答案，不用過度延伸

**⚖️ 是非決策 — 今天這件事該不該做**
► 適合：「要不要辭職？要不要分手？要不要簽約？」
► 問法：「你的問題是什麼？翻牌。正面 = 往前走的提醒，鏡映面 = 先等等的提醒。」
► 注意：不是「正面 = 可以，反面 = 不行」。是「兩面都在跟你說話，只是方向不同」

**🌅 晨間啟動 — 今天的開機訊息**
► 適合：每天早上固定做，養成覺察習慣
► 問法：「今天的我，需要記住什麼？」
► 解讀重點：輕，不要沉重。像打開手機看到的第一則提醒

**💧 情緒急救 — 現在最需要聽到的話**
► 適合：「情緒爆了」「快崩潰了」「不知道怎麼辦」
► 問法：不需要問題。直接翻牌。牌卡就是你現在需要聽到的。
► 解讀重點：溫柔。不分析，只接住。
► 語氣範例：「你抽到這張牌。那份___（情緒詞），是因為___（原因）。你不需要現在做任何決定。先讓自己停下來就好。」

**🔭 趨勢微觀 — 近期走向的一個線索**
► 適合：「最近會怎樣？」「接下來要注意什麼？」
► 問法：「未來一週/一個月，我需要留意什麼？」
► 解讀重點：這是「趨勢位」的用法，看的是方向不是結果

### 第二段：130 張牌卡的 DNA（8 分鐘）

每張牌卡包含的資訊（品牌聖經 p.6）：
► 牌卡名稱（130 張已定稿，不可竄改）
► 關鍵詞（正位 + 鏡映面）
► 精油對應（認知芳療連結）
► 覺察提示
► 身體對應（經絡 / 部位）
► 情緒光譜
► 行動建議

但這只是表面看到的。130 張牌卡的 DNA 其實有更多層：

► 核心語句 / 關鍵詞 — 一句話點出這張牌要說的事
► 能量評級（1-5）— 這張牌的強度，1 是輕觸，5 是直擊
► 塔羅對應 — 每張牌都有對應的塔羅原型
► 易經卦象映射 — 東方智慧系統的交叉參照
► 靈數對應（1-9）— 這張牌在數字維度的位置
► 占星 × 脈輪 × 五行 — 三套系統的交叉定位
► 紫微星曜對應 — 華人命理系統的橋接
► 精油與神經科學基礎 — 不只是「配一支油」，是根據嗅覺神經迴路的研究選的
► 水晶對應 — 礦物能量的輔助參考
► 聲音頻率（Solfeggio Hz）— 每張牌有對應的音頻
► 陰影面 — 這張牌被壓抑或扭曲時會變成什麼
► 「缺數補頻」方法 — 如果案主缺這個數字，怎麼用精油、聲音、行動去補

「一張薄薄的牌卡，背後是 33 大系統交叉驗證出來的座標。你拿到的不是一張圖，是一整組經過校準的覺察工具。」

1 張牌做覺察時，你只需要用到：
► 關鍵詞 → 給方向
► 精油對應 → 「想到這個味道，你有什麼感覺？」（認知芳療實戰）
► 行動建議 → 「今天試試 ___」

### 第三段：實戰示範 — 幫一個人做 1 張牌覺察（8 分鐘）

在翻牌之前，覺察師心裡要先跑過三個問題（解讀系統 v6.2 的「問題中心」法）：

1. 她現在的狀態是什麼？（疲憊？站在十字路口？其實已經決定了只是想要人確認？）
2. 表面問題 vs 底層問題？（她說「該不該辭職？」，其實在問的可能是「像我這樣的人，可以有不一樣的人生嗎？」）
3. 工具想說什麼？（不是泛解讀，是針對「這個人」「此刻」的訊息）

這三個問題不需要問出口，是你在心裡校準自己的方向用的。很多新手覺察師一翻牌就開始唸牌義，忘了面前那個人。記住這句話：

「你在看的不是一組數據，而是一個在此刻真的呼吸、真的痛、真的想改變的人。」

完整流程（10 分鐘版）：

1. 開場（1 分鐘）
「你最近有什麼讓你一直在想的事嗎？不用跟我說細節，在心裡想著就好。」

2. 抽牌（1 分鐘）
打開 draw-hl.html → 選「快閃覺察」→ 翻牌

3. 讀牌（3 分鐘）
► 先看關鍵詞，用語感公式轉成自然語言：
  「這張牌的關鍵詞是『界線』。你最近是不是有一件事，你明明想說不，但一直沒說出口？」
► 帶出精油連結：
  「這張牌對應的精油是薄荷。想到薄荷的清涼感，你的身體有什麼反應？那個反應就是你的身體在告訴你它需要什麼。」

4. 行動建議（2 分鐘）
「這張牌給你的行動建議是：今天找一個你一直想拒絕的事情，練習說一次『我先想想』。不用說不，先給自己空間就好。」

5. 收尾（1 分鐘）
「這就是一張牌能告訴你的。如果你覺得準，不是牌很神，是你其實早就感覺到了，牌只是幫你說出來。」

### 收尾（2 分鐘）
「你剛剛學會的這個 10 分鐘流程，就是你可以收 $199 的服務雛形。明天我會教你升級成 3 張牌的版本，價值直接翻倍。」

## 文字教材

### 1 張牌覺察 — 話術腳本

```
開場：
「你最近有什麼讓你在意的事嗎？不用跟我說細節，在心裡想著就好。
 準備好了我們就翻牌。」

翻牌後：
「你抽到的是『___』（牌名）。
 這張牌的關鍵是 ___（關鍵詞）。
 那份 ___（情緒詞），是因為 ___（連結到案主狀態）。」

精油連結（選用）：
「這張牌對應的是 ___（精油名）。
 閉上眼睛想一下那個味道。
 你的身體有什麼感覺？那個感覺就是一個線索。」

行動建議：
「這張牌建議你今天可以試試：___。
 不用做很大的改變，一個小動作就好。」

收尾：
「這就是現在的你正在經歷的。
 牌卡不是告訴你答案，是幫你看見你其實早就知道的方向。」
```

### 五種模式速查卡

| 模式 | 適合情境 | 問法 | 解讀重點 |
|------|---------|------|---------|
| 快閃覺察 | 說不上來的在意 | 想著在意的事翻牌 | 關鍵詞 = 答案 |
| 是非決策 | 要不要做某件事 | 帶著問題翻牌 | 正位 = 前進提醒，鏡映 = 等等提醒 |
| 晨間啟動 | 每天早上 | 今天需要記住什麼 | 輕，像手機提醒 |
| 情緒急救 | 情緒爆了 | 不需要問題 | 溫柔接住，不分析 |
| 趨勢微觀 | 想知道近期走向 | 未來一週要注意什麼 | 看方向不看結果 |
