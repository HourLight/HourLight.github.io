# H6｜交付你的第一份解讀報告

## 影片大綱（建議 25 分鐘）

### 開場（2 分鐘）
「昨天你做了第一次付費覺察。今天要學的是怎麼把那次覺察變成一份案主可以反覆看、會截圖分享的『解讀報告』。報告做得好，案主會把它分享給朋友——那就是你的免費行銷。」

### 第一段：解讀報告的三層結構（10 分鐘）

品牌聖經的解讀結構：第一層溫暖接住，第二層高 EQ 毒舌說真話。

加上第三層：行動指令。

**第一層：你被看見了（情緒命中）**
► 開頭直接點出案主的核心狀態
► 用語感公式：「那份 ___，是因為 ___。」
► 案主看到會覺得：「她真的在講我。」

**第二層：但你可能沒注意到（洞察翻轉）**
► 提供案主「自己看不到的角度」
► 品牌聖經的雙層語感：溫柔接住之後，用不刺但準的方式說真話
► 案主看到會覺得：「啊，原來是這樣。」

**第三層：所以你現在可以做這件事（行動指令）**
► 具體、可執行、今天就能做
► 不是「多愛自己一點」，是「今天晚上洗澡的時候，閉眼 30 秒，什麼都不想。」
► 案主看到會覺得：「好，我可以試試。」

### 第二段：報告格式模板（8 分鐘）

```

覺察師：___（你的名字）

第二張：___（牌名）— 現在的狀態
第三張：___（牌名）— 未來的方向

【整體覺察】
（第一層：情緒命中）

【精油對應】
```

### 補充：報告品質標準 — 案主付了錢，報告要值那個錢

解讀報告是付費內容。案主花了 $199 或 $599，她拿到的那份報告必須讓她覺得「這比我花的錢值得」。不是用華麗辭藻堆起來的那種值得，是「每一句話都在講我，而且我知道接下來要做什麼」的那種值得。

「3 張牌報告標準（300-500 字）」

必須包含五個區塊，一個都不能少：
► 牌卡組合：三張牌的名稱 + 各自對應的位置（不是列出來就好，要用一句話串起來。例如：「過去是 XX，帶著 ___ 的氣息走到現在的 XX，而未來指向 XX。」）
► 整體覺察：兩層結構。第一層接住她的感受，第二層說那個她沒看見的角度。不超過 150 字，精準比長度重要。
► 精油對應：三張牌各自對應的精油，加上一句認知芳療建議。不需要她手邊有精油，想到那個味道就夠了。
► 行動建議：今天就能做的一件具體的事。不是「對自己好一點」，是「今天睡前花 5 分鐘，把你最想對那個人說但一直沒說的話寫在手機備忘錄裡。不用傳，寫下來就好。」
► 收尾金句：馥靈馥語（有香氣的文字），15 字以內，命中她的狀態，值得截圖。

「7 張牌報告標準（800-1500 字）」

7 張牌是 H.O.U.R. 完整解讀，報告結構要對應四個階段：
► H 身心校準：前兩張牌的解讀，聚焦在案主現在的身心狀態。她的身體在說什麼？情緒卡在哪裡？
► O 智慧辨識：第三、四張牌，聚焦在她的思維模式。她是怎麼想的？那個想法是真的嗎？
► U 潛能解鎖：第五、六張牌，聚焦在她還沒動用的資源。她有什麼是自己沒發現的？
► R 行動進化：第七張牌，聚焦在接下來的行動。不只是一件事，是一份 7 天行動藍圖。
► 精油處方：7 張牌對應的精油組合，寫成一個「本週精油節奏」。例如：「早上用 ___（清醒），下午用 ___（穩定），睡前用 ___（沉澱）。手邊沒有的，想到那個味道就好。」
► 每份報告最後都用馥靈馥語收尾。有香氣的文字，不是雞湯，是她看完會想深呼吸的那種句子。

「通用原則」

► 報告裡不能出現「你的命很好」「你前世怎樣」「宇宙會回應你」這類話。品牌禁忌，也是專業底線。
► 每一段都要有案主的具體狀況。如果你寫出來的報告把名字換掉可以給任何人用，那就是不及格。她花錢買的是「專屬於她的覺察」，不是罐頭內容。
► 行動建議不能抽象。可以驗證的標準：案主看完知不知道「今天回家第一件事做什麼」。如果她還要再想一下才知道，你的建議不夠具體。

### 第三段：報告交付方式（5 分鐘）

► LINE 文字訊息（最方便，案主可以隨時回看）
► 複製到 LINE 直接貼（格式會保留）
► 如果想更專業：截圖做成圖片（用 Canva 套品牌模板）
► 交付後一句話：「報告看完如果有看不懂的地方，可以延伸提問 3 次，不另收費。」

交付後追蹤：
► 3 天後主動問一句：「上次建議的那件事你有試嗎？感覺如何？」
► 這一句話 = 回頭客的起點

### 收尾（2 分鐘）
「你現在手上有了：秘碼 + 1 張牌 + 3 張牌 + 對談話術 + 報告模板。這五樣東西加起來，就是一個完整的 $199 覺察服務。明天是第一週的最後一天，我們來複盤，看看你這一週到底做了什麼、賺了多少、學到了什麼。」

## 文字教材

### 解讀報告常見錯誤

| 錯誤 | 為什麼不好 | 怎麼改 |
|------|-----------|--------|
| 照背牌義 | 像教科書不像人話 | 用語感公式轉成自然語言 |
| 太長 | 案主看不完 | 3 張牌報告控制在 500 字以內 |
| 沒有行動建議 | 看完不知道要幹嘛 | 每份報告必須有「今天就做」 |
| 用「你不是XX是XX」句型 | 品牌禁忌 | 直接描述狀態，不否定人 |
| 用雞湯收尾 | 「宇宙會回應你的」 | 給具體動作，不給空話 |

### 讓案主截圖分享的秘訣

報告裡一定要有「一句話送給你」收尾，這句話要：
► 短（15 字以內）
► 準（命中案主的狀態）
► 美（值得截圖）

範例：
► 「停下來不是放棄，是讓自己看清楚。」
► 「你已經比你以為的走得更遠了。」
► 「答案不在外面，在你停下來的那個瞬間。」

## 今天就做

1. ✅ 用模板寫出昨天那位案主的完整解讀報告
2. ✅ 透過 LINE 交付給案主
3. ✅ 交付後發一句：「看完如果有不懂的，可以問我 3 次哦。」
4. ✅ 請案主給你一句回饋 + 評分（1-10 分）
5. ✅ 把回饋截圖保存 → 這是你的見證庫
