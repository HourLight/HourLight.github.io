# H4｜三張牌快速覺察：城堡三塔實戰

## 影片大綱（建議 30 分鐘）

### 開場（2 分鐘）
「1 張牌是快閃。3 張牌是故事。昨天你學會了幫人做一次快閃覺察，今天要升級——用 3 張牌看見一個人的過去、現在、未來。這也是你第一個可以收 $199 的服務。」

### 第一段：城堡三塔 — 五種解讀方式（10 分鐘）

3 張牌叫「城堡三塔」，不是只有「過去、現在、未來」一種擺法（品牌聖經 p.6）：

**1. 時間軸（最常用）**
► 第一張 = 過去的影響
► 第二張 = 現在的狀態
► 第三張 = 未來的方向
► 適合：「我想知道這件事的來龍去脈」

**2. 問題解析**
► 第一張 = 問題的根源
► 第二張 = 現在的障礙
► 第三張 = 解決的方向
► 適合：「我知道有問題但不知道卡在哪」

**3. 決策輔助**
► 第一張 = 選 A 的能量
► 第二張 = 選 B 的能量
► 第三張 = 建議的方向
► 適合：「A 還是 B？」

**4. 感情專用**
► 第一張 = 我的感受
► 第二張 = 對方的感受
► 第三張 = 這段關係的方向
► 適合：感情問題（最熱門的主題）

**5. 真偽辨別**
► 第一張 = 表面看到的
► 第二張 = 底下藏著的
► 第三張 = 真正需要面對的
► 適合：「我覺得有什麼不對勁」

### 補充：城堡三塔五種模式的選用時機

案主的問題千百種，但翻譯過來大概就五類。你選對模式，解讀的精準度會直接跳一個層級。

► 案主說「我想知道為什麼會變成這樣」→ 時間軸。她在找脈絡，想理解事情的因果。三張牌排出來就是一條線：這件事從哪裡來、走到哪裡、要往哪裡去。

► 案主說「我知道有問題但不知道卡在哪」→ 問題解析。第一張翻出來的不是她以為的問題，是真正的根源。第二張是她現在卡住的點。第三張不是答案，是方向。你的工作是把這三張串成「原來你卡在這裡，出口在那邊」的故事。

► 案主問「A 還是 B」→ 決策輔助。注意：第三張不叫「宇宙建議」，叫「這兩條路之外你沒看到的東西」。有時候第三張牌翻出來，案主會突然意識到她其實不想選 A 也不想選 B，她想要的是 C。那就對了。

► 案主帶感情問題來 → 感情模式。第一張是她自己的狀態，不是她以為的狀態。第二張是對方的狀態，通常會讓案主沉默三秒。第三張是這段關係的走向。記住：不預測結果，只翻譯現況。「你們現在的位置在這裡，接下來會往那個方向移動」就夠了。

► 案主說「我覺得哪裡怪怪的」→ 真偽辨別。這是最刺激的模式。第一張是她看到的表面，第二張是她隱約感覺到但不敢確認的東西，第三張是「你真正需要面對的是什麼」。用這個模式的時候，語氣要特別穩。因為第二張翻開來，案主很可能已經在心裡說「果然」了。

### 補充：五類問題的解讀協議

不是所有問題都用同一種方式回應。解讀之前先判斷：她到底需要什麼。

「1. 她需要被聽見」
有些人來做覺察，不是要答案，是要一個人認真聽她講。你會認出來，因為她說了很多、很長、一直在解釋。這種時候不要急著給解法。讓牌說完它要說的，你就安靜地接著。等她停下來，問一句：「你想動的第一步是什麼？」。如果她說不知道，那就陪她再看一遍牌，慢慢來。

「2. 她需要方向」
她知道自己不開心，但不知道該往哪走。這種時候牌卡給的方向要落地。不要停在「你需要更勇敢」這種層級，要給實驗性微任務：「接下來 7 天，當 XX 發生時，試著用 OO 回應。每天晚上花 3 分鐘寫下你的感覺。」7 天後她會自己看到答案。

「3. 她需要決定」
A 或 B，她來找你其實不是要你幫她選。她要的是有人幫她看清楚兩條路的質地。你的工作是翻譯，不是選擇：「走這條路，你會經歷的感覺比較像 ___；走那條路，你會經歷的感覺比較像 ___。」說完，安靜。讓她自己決定。她會的。

「4. 她需要解決問題」
有些問題不需要溫柔迂迴。她問「我該不該辭職」，牌卡翻出來很明顯在說「你待太久了」，那就直說不繞彎：「牌卡反映的是你在這個位置已經學完你要學的了。」不需要加很多鋪墊。清楚就是溫柔。

「5. 收尾原則」
不管前面哪一類，每次解讀結束至少要帶走一個「向外行動」：打一通電話、做一個小選擇、對一個人說一句話。覺察如果只停在腦子裡，三天就會忘。帶進真實世界的那一步，才是真正的覺察。

### 第二段：三張牌解讀的核心技巧（8 分鐘）

**技巧 1：先看整體，再看細節**
「三張牌攤開，先不要看每一張的意思。先感覺整體氛圍：亮的多還是暗的多？動的多還是靜的多？這個整體感就是第一層訊息。」

**技巧 2：牌跟牌之間的關係比單張更重要**
「第一張跟第二張如果完全相反，代表案主正在經歷一個轉折。如果三張都很相似，代表一個很明確的訊號在重複強調。」

**技巧 3：用品牌語感串成故事**
不是：「第一張是 XX，代表 XX。第二張是 XX，代表 XX。」
而是：「你過去經歷了一段 ___，那個經驗讓你現在 ___。但牌卡在說的是，接下來你可以 ___。」

**技巧 4：認知芳療串連**
「三張牌對應三種精油。把三個味道串在一起：過去是薰衣草（安撫），現在是薄荷（清醒），未來是檀香（沉穩）——你的身體正在從安撫走向清醒，最後會抵達沉穩。」

### 第三段：實戰示範 — 完整 3 張牌覺察流程（10 分鐘）

**1. 開場確認（2 分鐘）**
「你想問什麼方向的問題？感情、事業、人際、還是整體狀態？」
→ 根據回答選用五種擺法之一

**2. 抽牌（1 分鐘）**
draw-hl.html → 選 3 張 → 翻牌

**3. 整體觀感（1 分鐘）**
「三張牌攤開，我先說一個整體感覺：___」

**4. 逐張解讀（4 分鐘）**
每張用語感公式：「這張牌是 ___。那份 ___，是因為 ___。」
三張之間串成故事線

**5. 精油連結 + 行動建議（2 分鐘）**
「三張牌的精油分別是 ___、___、___。如果你手邊有 ___，今天可以聞一下。沒有也沒關係，想到那個味道就好。」
「這三張牌合起來建議你的一件事是：___。」
