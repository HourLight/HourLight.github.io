# H4｜三張牌快速覺察：城堡三塔實戰

## 影片大綱（建議 30 分鐘）

### 開場（2 分鐘）
「1 張牌是快閃。3 張牌是故事。昨天你學會了幫人做一次快閃覺察，今天要升級——用 3 張牌看見一個人的過去、現在、未來。這也是你第一個可以收 $199 的服務。」

### 第一段：城堡三塔 — 五種解讀方式（10 分鐘）

3 張牌叫「城堡三塔」，不是只有「過去、現在、未來」一種擺法（品牌聖經 p.6）：

**1. 時間軸（最常用）**
► 第一張 = 過去的影響
► 第二張 = 現在的狀態
► 第三張 = 未來的方向
► 適合：「我想知道這件事的來龍去脈」

**2. 問題解析**
► 第一張 = 問題的根源
► 第二張 = 現在的障礙
► 第三張 = 解決的方向
► 適合：「我知道有問題但不知道卡在哪」

**3. 決策輔助**
► 第一張 = 選 A 的能量
► 第二張 = 選 B 的能量
► 第三張 = 建議的方向
► 適合：「A 還是 B？」

**4. 感情專用**
► 第一張 = 我的感受
► 第二張 = 對方的感受
► 第三張 = 這段關係的方向
► 適合：感情問題（最熱門的主題）

**5. 真偽辨別**
► 第一張 = 表面看到的
► 第二張 = 底下藏著的
► 第三張 = 真正需要面對的
► 適合：「我覺得有什麼不對勁」

### 第二段：三張牌解讀的核心技巧（8 分鐘）

**技巧 1：先看整體，再看細節**
「三張牌攤開，先不要看每一張的意思。先感覺整體氛圍：亮的多還是暗的多？動的多還是靜的多？這個整體感就是第一層訊息。」

**技巧 2：牌跟牌之間的關係比單張更重要**
「第一張跟第二張如果完全相反，代表案主正在經歷一個轉折。如果三張都很相似，代表一個很明確的訊號在重複強調。」

**技巧 3：用品牌語感串成故事**
不是：「第一張是 XX，代表 XX。第二張是 XX，代表 XX。」
而是：「你過去經歷了一段 ___，那個經驗讓你現在 ___。但牌卡在說的是，接下來你可以 ___。」

**技巧 4：認知芳療串連**
「三張牌對應三種精油。把三個味道串在一起：過去是薰衣草（安撫），現在是薄荷（清醒），未來是檀香（沉穩）——你的身體正在從安撫走向清醒，最後會抵達沉穩。」

### 第三段：實戰示範 — 完整 3 張牌覺察流程（10 分鐘）

**1. 開場確認（2 分鐘）**
「你想問什麼方向的問題？感情、事業、人際、還是整體狀態？」
→ 根據回答選用五種擺法之一

**2. 抽牌（1 分鐘）**
draw-hl.html → 選 3 張 → 翻牌

**3. 整體觀感（1 分鐘）**
「三張牌攤開，我先說一個整體感覺：___」

**4. 逐張解讀（4 分鐘）**
每張用語感公式：「這張牌是 ___。那份 ___，是因為 ___。」
三張之間串成故事線

**5. 精油連結 + 行動建議（2 分鐘）**
「三張牌的精油分別是 ___、___、___。如果你手邊有 ___，今天可以聞一下。沒有也沒關係，想到那個味道就好。」
「這三張牌合起來建議你的一件事是：___。」

