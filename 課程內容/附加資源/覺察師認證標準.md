# 覺察師認證標準｜馥靈之鑰 Hour Light

## 認證等級

### Level 1：馥靈覺察旅人
► 條件：完成 H+O 階段（前兩週）
► 權益：可做 1-7 張牌覺察（$199-$599）
► 無年費

### Level 2：馥靈覺察師（認證）
► 條件：完成全部 H.O.U.R. 四階段 + 提交 5 份報告 + 5 份見證 + 面談通過
► 權益：可做 1-28 張全牌陣 + 出現在平台夥伴列表 + 專屬預約頁
► 年費 $39,800（含認證培訓）+ 月費 $999

### Level 3：馥靈覺察導師
► 條件：累計 50 個案主 + 帶出 3 個 Level 2 覺察師
► 權益：可開班授課 + 培訓新覺察師 + 課程分潤
► 年費 $39,800 + 月費 $999

### Level 4：馥靈傳承者
► 條件：累計 200 個案主 + 帶出 10 個覺察師 + 年度 MVP
► 權益：品牌共創 + 地區授權 + 高階策略輔導
► 面議

► 可使用「馥靈之鑰認證覺察師」頭銜

► 遵守醫療邊界（不取代心理諮商/醫療）
