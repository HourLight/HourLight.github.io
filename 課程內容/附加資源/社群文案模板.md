# 社群文案模板（覺察師通用版）

## Threads / X 短文（100 字以內）

### 模板 1：日常覺察
「你有沒有那種感覺，每天都很忙但說不上來在忙什麼？那不是因為事情多。是因為你沒有停下來問自己：這些裡面，哪一件其實可以不做？」

### 模板 2：牌卡觸發
「今天抽到一張牌，關鍵詞是『界線』。想到最近一直說好的那件事。不是因為想幫忙，是因為怕拒絕之後對方不喜歡我了。原來界線不是推開別人，是先把自己接住。」

### 模板 3：案主故事（匿名化）
「一位案主跟我說：『我明明知道該放手了，為什麼就是做不到？』抽到的牌告訴她：不是做不到，是你還沒有允許自己難過。放手之前要先讓自己哭完。」

## IG 輪播（5 頁）

P1：你最近是不是覺得 ___？（鉤子）
P2：【牌卡圖】___（牌名）
P3：那份 ___，是因為 ___（語感解讀）
P4：今天試試 ___（行動建議）
P5：想做更深的覺察？連結在自介 ✦

## FB / LINE 長文（300-500 字）

「前幾天一位朋友問我：『你覺得人為什麼會一直重複同樣的事情？明明知道不對了，還是在做。』

我幫她抽了三張牌。

第一張是 ___，代表她過去的 ___。
第二張是 ___，說的是她現在 ___。
第三張是 ___，方向很清楚：___。

三張牌攤開，她安靜了好久。然後說了一句：『我知道了。』

覺察不是告訴你答案。覺察是讓你看見，你其實早就知道的答案。

如果你也想試試，在下面留言『我想抽一張』，我幫你做一次免費快閃覺察。」

## 「我成為覺察師的 N 天」分享模板

「成為覺察師第 ___ 天。

這週做了 ___ 次覺察，遇到了 ___ 個很有意思的案主。

最讓我印象深刻的一次：___（匿名化的案主故事）

他/她說的那句話我到現在還記得：『___』

我才知道，覺察師做的不是告訴別人答案，是陪他們看見自己一直在逃避的那個問題。

然後他們會自己找到出路。

如果你也對覺察有興趣，歡迎來聊。」
