# U1｜LINE 官方帳號設定：你的數位門面

## 影片大綱（建議 20 分鐘）

### 核心概念
「沒有 LINE 官方帳號的覺察師 = 沒有門面的店。案主找不到你、聯繫不到你、信任建不起來。」

### 操作教學

「1. 申請 LINE 官方帳號（5 分鐘操作）」
► LINE Official Account Manager → 建立帳號
► 帳號名稱：你的品牌名（如「xxx 覺察工作室」）
► 頭像：你的照片或品牌 LOGO
► 背景資料：一句話說明你做什麼

「2. 自動回覆設定」
► 歡迎訊息：「你好！感謝加入 ___。想做覺察體驗？直接說一句你最近在意的事就好。」
► 圖文選單：參考馥靈之鑰的 LINE 圖文選單（6 格版），改成你的服務

「3. 圖文選單（Rich Menu）設計」

圖文選單是客人打開你 LINE 帳號時，畫面最下方那一排按鈕。你可以把它想成你的「數位店門口招牌」，客人不用打字、不用翻訊息，點一下就到對的地方。

► 6 格完整版（建議覺察師用這個）：

| 左上 | 中上 | 右上 |
|------|------|------|
| 線上抽牌 | 預約覺察 | 會員中心 |
| 心理測驗 | 課程資訊 | 聯絡我們 |

► 每一格的連結怎麼設：
  → 線上抽牌：hourlightkey.com/draw-hl.html（客人自己抽，1 張免費）
  → 預約覺察：hourlightkey.com/booking.html（直接選服務、選時間、付款）
  → 會員中心：hourlightkey.com/member-dashboard.html
  → 心理測驗：hourlightkey.com/quiz-hub.html（100+ 項免費測驗，做完會想做更多）
  → 課程資訊：你的課程頁連結
  → 聯絡我們：直接觸發 LINE 聊天

► 3 格精簡版（剛開始可以先用這個）：線上抽牌 / 預約覺察 / 聯繫我

► 圖文選單製作：用 Canva 做圖（1200×810px）→ LINE Official Account Manager 上傳 → 設定每格的連結動作

「4. LINE Messaging API 自動通知」

進階功能：當客人透過 booking.html 完成預約後，系統可以自動透過 LINE Messaging API 發送確認訊息。

► 客人端收到：「已收到您的預約！___（服務名稱），___（日期時間）。覺察師會在 24 小時內確認。」
► 覺察師端收到：「新預約通知！___（客人暱稱）預約了___（服務），時間___。」
► 這個功能需要申請 LINE Messaging API（免費額度每月 500 則），在 LINE Developers Console 設定

### 經營日常
► 每週一發一則「本週能量提醒」
► 案主做完覺察後的跟進訊息
► 不群發廣告（會被封鎖）
