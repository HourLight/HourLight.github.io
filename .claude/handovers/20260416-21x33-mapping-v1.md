## 一、21 張牌位置結構

21 張抽牌 = 1 核心牌 + 4 區塊 × 5 張（L.I.G.H.T. 第二軌）

### 核心牌（第 0 張）— 誓約位
- 整篇祈禱文的總基調
- 對應：流年數 + 核心命理主軸
- 輸出：今日主題標題 + 收尾金句選擇

### L.I.G.H.T. 第二軌定義（5 位牌的語意）

| 位 | 代號 | 語意 | 祈禱文作用 |
|---|---|---|---|
| L | 根源 | 這件事的底層土壤 | 事件緣起 |
| I | 信念 | 內在核心信念 | 宣告起句 |
| G | 行為 | 日常展現的模式 | 行動動詞 |
| H | 覺察 | 自我觀照的能力 | 轉折體認 |
| T | 禮物 | 完成後獲得的能量 | 成就宣告 |

## 三、六段祈禱文骨架

### 1. 今日主題標題
- 來源：核心牌（第 0 張）+ 流年主軸
- 格式：「YYYY/MM/DD｜（核心牌名稱）之日｜（流年能量一句話）」
- 範例：「2026/04/16｜天使之城之日｜轉化通道全開的榮耀主日」

### 2. 晨起宣言（身份）
- 來源：O 區塊第 10 張（T 禮物位）+ 身份層命理
- 作用：一起床念的身份定錨
- 結構：「我是＿＿＿（身份指認 × 命理元素）。我允許＿＿＿（今日展現）。」
- 3-5 句

### 3. 身體宣言
- 來源：H 區塊完整 5 張（第 1-5 張）+ 身體層命理
- 作用：身體覺察 + 能量到位
- 結構：L（根源）→ I（信念）→ G（行為）→ H（覺察）→ T（禮物）五步
- 每步結合身體某部位 + 感受描述
- 5-7 句

### 4. 豐盛宣言（含自訂願望）
- 來源：U 區塊完整 5 張（第 11-15 張）+ 豐盛層命理 + **使用者自訂願望注入**
- 作用：豐盛允許 + 具體許願
- 結構：L（根源豐盛基礎）→ I（配得信念）→ G（具體行動）→ H（自我察覺）→ T（豐盛禮物）
- **第三句（G 行為位）嵌入使用者自訂願望**（例：「我允許＿＿＿（使用者原話）自然發生」）
- 7-9 句

### 5. 流年／年度宣言
- 來源：R 區塊第 16 張（L 根源位）+ 第 20 張（T 禮物位）+ 流年數 + 年度命理
- 作用：年度定錨 + 長線視野
- 結構：今年的主題 → 能量流向 → 最終成就
- 4-6 句

### 6. 使命宣言 + 收尾
- 來源：R 區塊第 18 張（G 行為位）+ 核心牌 + 使命層命理
- 作用：使命點亮 + 收束回歸
- 結構：使命確認 → 行動承諾 → 收尾
- **固定收尾句**：「我準備好了。我值得。我允許。如是臨在，圓滿豐盛。」
- 3-4 句（不含收尾）

## 四、AI Prompt 架構（Claude Sonnet 4.6）

### System Message（絕不暴露給前端）
```
你是馥靈之鑰的祈禱文生成器「馥靈馥語」。
依照使用者的 21 張牌卡 + 33 套命理資料 + 自訂願望，
產出六段吸引力法則祈禱文。

禁忌：
- 不使用雙破折號、粗體符號、療癒/治癒/調頻/對頻/啟數/王座塔
- 不使用「你不是 XXX，是 XXX」句型
- 不醫療宣稱
- 每段長短句交叉，留一點呼吸廢話

格式鐵律：
- 六段結構按下方順序，每段之間空一行
- 對外用「您」
- 收尾句固定：「我準備好了。我值得。我允許。如是臨在，圓滿豐盛。」
```

### User Message（結構化輸入）
```
【核心牌】牌名 + 解讀編號
【H 區塊（1-5）】五張牌編號 + 身體層命理濃縮
【O 區塊（6-10）】五張牌編號 + 身份層命理濃縮
【U 區塊（11-15）】五張牌編號 + 豐盛層命理濃縮
【R 區塊（16-20）】五張牌編號 + 使命層命理濃縮
【自訂願望】使用者原話
【流年】YYYY + 流年主軸一句
【星座／紫微／八字主軸】濃縮一段（50-100 字）

請依照六段祈禱文骨架輸出。
```

### 安全鐵律（交接文 L66-71）
- ❌ 130 張牌卡完整 DNA 絕不進前端 → 只傳編號
- ❌ System Prompt 絕不進前端
- ❌ 33 套命理的完整計算邏輯絕不進前端 → 只傳濃縮結果
- ✅ 前端只顯示最終祈禱文黑盒子輸出
