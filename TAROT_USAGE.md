# 塔羅抽牌功能使用說明

## 📋 概覽

馥靈之鑰塔羅抽牌功能提供兩個版本：
1. **獨立頁面版** (`tarot-draw.html`) - 可直接訪問的完整頁面
2. **嵌入模組版** (`tarot-widget.html`) - 可插入任何現有頁面的小區塊

---

## ✨ 功能特色

- ✅ **78張完整塔羅牌**（22張大阿爾克那 + 56張小阿爾克那）
- ✅ **簡單線條圖示**：每張牌都有對應的emoji圖示
- ✅ **支援正逆位**：隨機產生，逆位的牌會翻轉180度
- ✅ **可選擇張數**：1、3、5、7、9、12張
- ✅ **翻牌動畫**：卡片會依序翻開，有視覺趣味
- ✅ **一鍵複製**：點擊後自動複製所有結果，格式化好可直接分享
- ✅ **小區塊設計**：不佔大版面，響應式設計
- ✅ **台灣品牌識別**：🇹🇼 源自台灣｜馥靈之鑰

---

## 🚀 使用方法

### 方法1：獨立頁面版

**檔案：** `tarot-draw.html`

**使用方式：**
1. 直接上傳到網站根目錄
2. 訪問 `https://你的網站.com/tarot-draw.html`
3. 或在任何頁面加上連結：
   ```html
   <a href="tarot-draw.html">🎴 塔羅抽牌</a>
   ```

**適用場景：**
- 想要一個專門的塔羅抽牌頁面
- 需要分享給他人的獨立連結
- 想要完整的視覺呈現

---

### 方法2：嵌入模組版

**檔案：** `tarot-widget.html`

**使用方式：**
1. 打開 `tarot-widget.html`，複製所有內容
2. 在你想要插入的頁面中，找到合適的位置（例如首頁的某個區塊）
3. 直接貼上複製的內容
4. 保存並刷新頁面

**範例：插入到首頁**
```html
<!-- 你的首頁其他內容 -->

<!-- 塔羅抽牌模組 -->
<style>
  /* 這裡貼上 tarot-widget.html 中的 <style> 內容 */
</style>

<div class="tarot-widget">
  <!-- 這裡貼上 tarot-widget.html 中的 HTML 內容 -->
</div>

<script>
  /* 這裡貼上 tarot-widget.html 中的 <script> 內容 */
</script>

<!-- 你的首頁其他內容 -->
```

**適用場景：**
- 想要在首頁或其他頁面加上塔羅抽牌功能
- 不想要額外的頁面跳轉
- 希望訪客可以直接在當前頁面體驗

- 卡片翻開：3D翻轉動畫
- 依序出現：每張卡片間隔0.08-0.1秒
- Hover效果：微微上浮

## 🔧 自訂調整

如果你想要調整樣式，可以修改以下部分：

### 調整顏色
在 `<style>` 區塊中找到：
```css
:root{
  --milk:#f9f5f0;      /* 背景色 */
  --cocoa:#6b4f3e;     /* 主要文字色 */
  --rose:#cda86e;      /* 主要按鈕色 */
  --rose-dark:#a88257; /* 按鈕深色 */
}
```

### 調整卡片大小
在 `<style>` 區塊中找到：
```css
.cards-container{
  grid-template-columns:repeat(auto-fit,minmax(140px,1fr));
  /* 把 140px 改成你想要的最小寬度 */
}
```

### 調整動畫速度
在 `<script>` 區塊中找到：
```javascript
cardEl.style.animationDelay = `${index*0.1}s`;
// 把 0.1 改成你想要的間隔時間（秒）
```

## ⚠️ 注意事項

1. **瀏覽器相容性**：需要支援 `navigator.clipboard` API 的現代瀏覽器
2. **HTTPS要求**：複製功能在HTTPS環境下才能正常運作
3. **樣式衝突**：如果嵌入到現有頁面，可能需要調整CSS避免樣式衝突
4. **JavaScript衝突**：嵌入版已使用IIFE包裝，避免變數污染
