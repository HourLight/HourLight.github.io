# 馥靈之鑰：專屬牌陣系統與 L.I.G.H.T. 結構整合

## 牌陣設計核心理念
馥靈之鑰的牌陣系統完全摒棄傳統占卜的「預測未來」模式，轉而聚焦於「情緒覺察與自我梳理」。牌陣的設計緊密結合 H.O.U.R. 四階段方法論與「內在城堡」的隱喻，引導使用者從看見問題、辨識情緒，到解鎖潛能並採取行動。

**牌陣數量規範**：僅使用 1張、3張、7張、12張、15張、21張（無5張與9張）。

---

## 一、1張牌陣：每日覺察（Daily Check-in）
**適用場景**：每日早晨的自我對焦、突發情緒的快速梳理。
**抽牌方式**：從108張精油卡中抽取1張。
**解讀焦點**：
- **H.O.U.R. 提問**：這張牌的植物智慧，今天想提醒我注意內在城堡的哪個房間？
- **行動建議**：根據抽到的精油卡，進行簡單的嗅吸或呼吸練習。

## 二、3張牌陣：H.O.U.R. 核心梳理（The Core Alignment Spread）
**適用場景**：單一事件的情緒卡關、需要快速釐清現狀。
**抽牌方式**：1張覺察指引牌（原型卡） + 2張精油卡。
**佈局**：由左至右橫向排列。

| 位置 | 牌卡類型 | 牌位名稱 | 內在城堡隱喻 | 解讀焦點 |
| :--- | :--- | :--- | :--- | :--- |
| 1 | 覺察指引牌 | **O (Own) 智慧辨識** | 覺察廳 | 這是你目前面臨的核心系統動力或盲點。 |
| 2 | 精油卡 | **U (Unlock) 潛能解鎖** | 解鎖密室 | 這是幫助你解開這個結的植物智慧與資源。 |
| 3 | 精油卡 | **R (Rise) 行動進化** | 王座塔 | 這是你接下來需要採取的具體行動或態度。 |

## 三、7張牌陣：內在城堡探索（The Inner Castle Spread）
**適用場景**：反覆出現的情緒模式、人際關係衝突的深層梳理。
**抽牌方式**：2張覺察指引牌 + 5張精油卡。
**佈局**：城堡形狀（底層3張為地基，中層3張為塔樓，頂層1張為旗幟）。

| 位置 | 牌卡類型 | 牌位名稱 | 解讀焦點 |
| :--- | :--- | :--- | :--- |
| 1-3 (底層) | 精油卡 | **H (Heal) 身心校準** | 你的身體與情緒目前承載了什麼？哪裡需要被安撫？ |
| 4-5 (中層) | 覺察指引牌 | **O (Own) 智慧辨識** | 這些情緒背後，隱藏著什麼樣的家族動力或舊有模式？ |
| 6 (中層) | 精油卡 | **U (Unlock) 潛能解鎖** | 什麼資源可以幫助你打破這個模式？ |
| 7 (頂層) | 精油卡 | **R (Rise) 行動進化** | 當你拿回城堡的主導權，你的下一步是什麼？ |

## 四、12張牌陣：生命週期與12宮分佈（The 12 Houses Spread）
**適用場景**：年度回顧、結合「馥靈秘碼」的深度個人化解讀。
**抽牌方式**：12張精油卡（可搭配馥靈秘碼的12宮分佈圖使用）。
**佈局**：時鐘圓形排列（1-12點鐘方向），對應馥靈秘碼的12宮。

| 位置 | 對應宮位 | 解讀焦點 |
| :--- | :--- | :--- |
| 1-3 | 自我、價值、溝通 | 你如何看待自己？你如何表達你的價值？ |
| 4-6 | 根源、創造、日常 | 你的原生家庭印記？你如何在日常中展現創造力？ |
| 7-9 | 關係、深層、信念 | 你在關係中的投射？你最深層的恐懼與信念是什麼？ |
| 10-12 | 社會、群體、潛意識 | 你在社會中的位置？什麼在潛意識中默默影響你？ |

## 五、15張牌陣：代際創傷梳理（The Generational Healing Spread）
**適用場景**：處理原生家庭議題、深層的代際情緒傳遞。
**抽牌方式**：5張覺察指引牌 + 10張精油卡。
**佈局**：三層階梯式（底層5張，中層5張，頂層5張）。

| 位置 | 牌位名稱 | 牌卡組合 | 解讀焦點 |
| :--- | :--- | :--- | :--- |
| 1-5 (底層) | **過去的印記** | 2張指引牌 + 3張精油卡 | 家族系統中未被看見的痛，以及它如何影響你的身體與情緒。 |
| 6-10 (中層) | **現在的牽絆** | 2張指引牌 + 3張精油卡 | 你出於盲目的愛承擔了什麼？你正在重複誰的劇本？ |
| 11-15 (頂層) | **未來的轉化** | 1張指引牌 + 4張精油卡 | 如何將責任交還？你需要什麼植物資源來建立健康的界線？ |

## 六、21張牌陣：L.I.G.H.T. 完整生命力評估（The L.I.G.H.T. Blueprint Spread）
**適用場景**：馥靈覺察師帶領的「深度覺醒」或「三次轉化」高階一對一服務。
**抽牌方式**：7張覺察指引牌 + 14張精油卡。
**佈局**：五層金字塔結構，完美對應 L.I.G.H.T. 系統。

| 層級 | L.I.G.H.T. 維度 | 牌卡數量 | 系統排列含義與解讀焦點 |
| :--- | :--- | :--- | :--- |
| **L** | **Love (愛之殿) / 根源** | 6張 (2指引+4精油) | **系統的根源與愛的流動**：檢視你與生命來源（父母/祖先）的連結。愛在哪裡中斷了？ |
| **I** | **Intuition (直覺閣) / 信念** | 5張 (2指引+3精油) | **無意識的忠誠與信念**：你承接了什麼不屬於你的信念？你的直覺試圖告訴你什麼真相？ |
| **G** | **Ground (磐石廳) / 行為** | 4張 (1指引+3精油) | **日常行為與界線**：這些信念如何顯化在你的日常行為中？你的內在城堡是否穩固？ |
| **H** | **Harmony (和諧苑) / 覺察** | 3張 (1指引+2精油) | **看見盲點與重新校準**：當你看見這一切，你如何與自己、與系統達成和解？ |
| **T** | **Transform (蛻變室) / 禮物** | 3張 (1指引+2精油) | **轉化與行動進化**：帶著祖先的祝福，你將如何運用這些植物資源，走向自己的生命道路？ |

## 牌陣解讀的合規與實務原則

1. **嚴守禁忌詞規範**：解讀過程中，覺察師**絕對不可**使用「療癒、治療、調頻、靈魂藍圖」等詞彙。必須使用「梳理、覺察、重新對焦、內在節奏」。
2. **不預測，只覺察**：牌陣不是用來算命或預測未來，而是「看見自己」的工具。
3. **行動導向**：每一次的牌陣解讀，最後都必須落實到具體的行動（如：使用某張精油卡對應的精油進行呼吸練習）。
4. **AI 輔助解讀**：在7張以上的複雜牌陣中，覺察師可將抽出的牌卡輸入「AI智慧模組」，由AI產出參考架構，但覺察師必須親自引導個案進行深度梳理。
