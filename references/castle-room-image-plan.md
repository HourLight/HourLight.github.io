# 城堡房間圖片升級規劃書

## 設計原則

**風格：** 溫暖手繪水彩風，偏奶油色調。不是寫實 3D，不是像素風，是有溫度的插畫感。
**視角：** 統一為「房間一角」的 3/4 視角，像坐在房間角落往裡看。
**光線：** lv1 微光（窗邊一道光）→ lv4 溫暖滿溢（整個空間發光）。
**情緒：** lv1 安靜、等待、有一點孤獨但有希望 → lv4 被愛、完整、像回家的感覺。
**色調：** 以各房間的主題色為基礎，整體偏暖。不要冷色調。
**尺寸：** 1024×576（16:9 橫幅），WebP 格式。

## 升級邏輯

- **lv1**（0 件傢具）：空房間，只有一個代表性物件。安靜、等待。
- **lv2**（1 件傢具）：多了合成的傢具，空間開始有生氣。
- **lv3**（3 件傢具）：更多物件，光線更亮，有被照顧的感覺。
- **lv4**（5 件或傳說傢具）：完整、華麗、溫暖飽滿，像自己的家。

## H.O.U.R. 四主房

### 1. 鏡之廳（mirror）— H 身心校準
**主題：** 誠實面對自己
**主色調：** 琥珀金 + 柔白
**代表物件：** 一面鏡子

| 等級 | 畫面描述 |
|------|---------|
| lv1 | 石牆角落，一面蒙塵的橢圓古鏡掛在牆上。鏡面模糊，映不出清楚的影子。窗邊一道微光灑進來，照在鏡框邊緣。地板是舊木板，空曠安靜。氛圍：還沒準備好看見自己，但光已經在等了。 |
| lv2 | 鏡子被擦亮了，能看見模糊的輪廓。鏡子旁邊多了一盞自省燈（🪔），暖黃色的小火焰。光線比 lv1 亮一些。牆角可能放了芳療靠枕（🛋️）。氛圍：開始願意靠近鏡子了。 |
| lv3 | 鏡面清晰，映照出窗外的光。牆上有了壁燈，地上鋪了一小塊地毯。自省燈的光更亮，靠枕旁邊有一本打開的日記。空間變溫暖了。氛圍：已經習慣每天來看一眼了。 |
| lv4 | 鏡子散發柔和金光，整個角落被溫暖的光包圍。牆上有精緻的壁飾，地毯鋪滿，自省燈變成了一盞華麗的燈台。鏡中映出美麗的光芒（不是人影，是光）。氛圍：你看見的是完整的自己。 |

### 2. 價值寶庫（treasure）— O 智慧辨識
**主題：** 看見自己的價值
**主色調：** 深金 + 酒紅
**代表物件：** 一個舊木箱

| 等級 | 畫面描述 |
|------|---------|
| lv1 | 昏暗的石室角落，一個舊木箱半開著，裡面空的。牆上掛著一把生鏽的鑰匙。只有縫隙裡透進一點光。氛圍：寶藏在哪？也許你自己就是。 |
| lv2 | 木箱裡開始有了一些發光的東西（像水晶碎片）。牆上的鑰匙擦亮了。旁邊放了一個小燈座。光線稍亮。氛圍：開始發現一些被忽略的好東西。 |
| lv3 | 木箱裡的寶石更多了，散發柔和的光。牆上掛了幾幅小畫。角落有一張墊了靠枕的矮凳。光線溫暖。氛圍：原來我一直都有這些，只是沒看見。 |
| lv4 | 整個角落像被金光浸泡。木箱變成了精緻的寶箱，寶石溢出來。牆上的畫變成了華麗的掛毯。地上有柔軟的毛毯。氛圍：富足不是擁有多少，是知道自己值得。 |

### 3. 解鎖密室（key）— U 潛能解鎖
**主題：** 打開那扇一直不敢開的門
**主色調：** 靛藍 + 銀白
**代表物件：** 一扇鎖著的門

| 等級 | 畫面描述 |
|------|---------|
| lv1 | 走廊盡頭，一扇厚重的木門。門上有一把大鎖，鏽跡斑斑。門縫下透出微微的藍光。牆壁是冷色的石頭。氛圍：你知道門後面有東西，但你還沒有鑰匙。 |
| lv2 | 門上的鎖開始鬆動，縫隙裡的光更亮了。門旁邊掛了一盞燈。牆上有一些刻痕（像是有人試過很多次）。氛圍：快了，再一點點。 |
| lv3 | 門半開了，藍白色的光灑出來。門框上開始長出藤蔓（生機）。旁邊有一張小桌，上面放著幾把鑰匙。氛圍：門開了，但你還在門口猶豫。 |
| lv4 | 門完全打開，光芒從門後傾瀉而出。整個走廊被照亮。門框上開滿了花。門後看不清楚，但光是美的。氛圍：你走進去了。門後不是答案，是更大的自己。 |

### 4. 啟程塔（throne）— R 行動進化
**主題：** 從知道到做到
**主色調：** 紫紅 + 金
**代表物件：** 一張空椅子

| 等級 | 畫面描述 |
|------|---------|
| lv1 | 塔頂的圓形房間，中央有一張簡單的木椅。窗戶很大，可以看到遠方。風吹進來。椅子空著。氛圍：位子一直都在，只是你還沒坐上去。 |
| lv2 | 椅子上多了一個墊子。窗台上有一盞燈。遠方的景色更清晰了。氛圍：開始覺得，也許我可以坐在這裡。 |
| lv3 | 椅子變得更舒適，旁邊有一張小桌放著茶杯。窗邊有簾子隨風飄動。遠方有朝陽。氛圍：坐在這裡，看得很遠，心很定。 |
| lv4 | 椅子變成了一個溫暖的王座（不是華麗的那種，是舒服的那種）。整個塔頂被金色晨光填滿。遠方風景壯麗。氛圍：你不是在等什麼，你已經在路上了。 |

## 注意事項

- 同一間房 lv1-lv4 要像同一個角落的時間推移，不是四個不同的房間
- 傢具出現的位置要固定（lv2 加的東西，lv3/lv4 還要在）
- 不要有人物出現
- 不要有文字
- 光線是最重要的情緒工具：lv1 微光 → lv4 溫暖滿溢
