核心概念

「不是每天發文，是每篇都有人想截圖分享。」

七大平台差異化策略

每個平台的人不一樣，看東西的方式不一樣，你說話的方式也要不一樣。不是把同一篇文案貼到七個地方，那叫複製貼上，不叫經營。

  • Threads / X：100 字以內的覺察吐槽（每天 1 篇，像內心 OS）

→ 短、狠、有共鳴。像朋友突然在群組丟了一句話，你截圖存下來的那種。

  • IG：輪播圖（每週 2 篇，牌卡解讀 + 覺察語句）

→ 視覺優先。第一頁決定他會不會滑，最後一頁決定他會不會追蹤你。

  • FB：長文（每週 1 篇，朋友的故事 + 深度分析，300-500 字）

→ FB 的人願意讀長文，但你要用「說故事」的方式寫，不是「寫報告」的方式。

  • LINE 官方帳號：每週覺察提醒（固定每週一）

→ 私密感最強。像朋友傳訊息給你，不是品牌在廣播。

  • 小紅書：圖文筆記（牌卡美圖 + 300 字覺察心得）

→ 搜尋導向。標題要有關鍵字，內容要有乾貨。

  • YouTube / Podcast：深度內容（10-30 分鐘）

→ 信任感最強。聽過你聲音的人，轉化率是只看文字的 3 倍。

  • TikTok / Reels：15-60 秒短影片

→ 不露臉也可以。牌卡翻轉 + 字卡 + 背景音樂，剪映 3 分鐘搞定。

內容模板

Threads / X 模板（100 字以內）

範例 1：

「你有沒有那種感覺，每天都很忙但說不上來在忙什麼？那不是因為事情多。是因為你沒有停下來問自己：這些裡面，哪一件其實可以不做？」

範例 2（牌卡觸發型）：

「今天抽到『放手』。不是叫你放棄，是叫你鬆開那個已經不在你手裡的東西。你握著一個空的拳頭，手會痠，但什麼都拿不到。」

範例 3（匿名案例型）：

「一個客人跟我說她最近一直夢到在找東西。我問她：『你現實中是不是也一直在找一個答案？』她愣了三秒。有時候你找的不是答案，是允許自己不知道的勇氣。」

IG 輪播模板（5 頁）

P1（鉤子）：一句話問題，讓人停下來（「你最近是不是覺得___？」）

P2（牌卡）：牌卡圖 + 這張牌的關鍵訊息（一個詞或一句話）

P3（解讀）：語感解讀（「那份___，是因為___」） → 用品牌核心語感公式

P4（行動）：今天可以做的一件小事（具體、可執行、不費力）

P5（CTA）：「想做更深的覺察？連結在自介」或「留言『我想抽一張』，我私訊你」

FB / LINE 長文模板（300-500 字）

結構：朋友的故事 → 3 張牌覺察 → 洞察 → CTA

「前幾天一個朋友跟我說，她最近跟老公吵架，不是因為什麼大事，就是覺得自己不被看見。

我幫她抽了 3 張牌。第一張是___，第二張是___，第三張是___。

有意思的是，三張牌都在講同一件事：___。

她聽完沈默了一下，然後說：『我一直以為是他的問題。』

有時候覺察不是找到答案，是發現你問錯了問題。

如果你也有一件事想看清楚，留言『我想抽一張』，我會私訊你一張牌的快閃覺察，免費的。」

「我成為覺察師的第 N 天」系列模板

這個系列很適合剛開始的覺察師。真實、不完美、有溫度。

  • Day 1：「今天正式開始了。說不緊張是騙人的。但我想起自己第一次被牌卡說中的那個瞬間，覺得如果我能讓別人也有那個瞬間，就值得了。」
  • Day 7：「第一個客人的反饋來了。她說：『你怎麼知道的？』我沒有知道。牌卡知道，我只是翻譯。」
  • Day 30：「一個月了。做了___次覺察，收入___元。數字不大，但每次客人說『謝謝你看見我』的時候，我知道這條路是對的。」
