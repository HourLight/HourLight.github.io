第一段：三套數字系統（8 分鐘）

品牌聖經的鐵則（不可混用）：

  • ①馥靈秘碼（農曆生日） → 逸君原創 IP → 看「內在設定」
  • ②三角生命密碼（西曆生日） → 市面既有，不是我們的 IP
  • ③馥靈三角秘碼（身分證字號） → 逸君全新自創 → 看「外在使命」

口訣：農曆是秘碼、三角非自創、身分證是三角秘碼

三套系統各自對應什麼能量？

  • 西曆 = 社會身份能量：你在現代社會系統裡的定位。學校、職場、身分證、護照，全部用西曆。這套數字反映的是「別人眼中的你」和「你在社會裡扮演的角色」。
  • 農曆 = 情緒潮汐能量：農曆是月亮週期曆法。月球引力影響潮汐是物理學事實，人體 70% 是水也是生理學事實。月亮拉動海水，也拉動你身體裡的水。女性月經週期平均 28 天，跟 29.5 天的月亮週期幾乎重疊。所以農曆生日算出來的數字，反映的是你的情緒節奏、內在潮汐、身體跟月亮的對應關係。
  • 出生時辰 = 身體潛能能量：中醫的「子午流注」理論，12 個時辰各對應一條經絡。你在哪個時辰出生，那條經絡的能量就是你的起始設定。

子午流注十二時辰經絡對應：

子時 23:00-01:00 膽經 決斷力 / 勇氣
丑時 01:00-03:00 肝經 調節 / 創造力
寅時 03:00-05:00 肺經 呼吸 / 釋放

舉個例子：逸君戌時出生，對應心包經，天生的潛能在「保護與敞開」之間。心包經的人，很會幫別人建立安全感，但自己有時候保護過頭，反而把真正想靠近的人擋在外面。你看她做的品牌，不就是這件事嗎？幫每個人找到保護自己跟對世界敞開之間的平衡點。

完整公式：完整的你 = 三維時間座標（內在設定）＋ 馥靈三角秘碼（外在使命）

三維時間座標看的是你「本來是誰」，馥靈三角秘碼看的是你「來做什麼」。兩套加在一起，才是完整的人生導航地圖。

為什麼要用農曆？

「傳統西方靈數只用國曆，但華人的情緒記憶、家族節奏、潛意識模式跟農曆更深層連結。逢年過節你記的是農曆還是國曆？清明、中秋、過年，你的身體記住的是農曆的節奏。」

這就是全球獨創的地方：用農曆生日算生命靈數。

第二段：四主數深度拆解（12 分鐘）

帶學員到 fuling-mima.html 重新看自己的結果：

**H數（癒數）— 你怎麼安頓自己**

  • 對應 1 宮勇氣殿
  • 核心提問：「我夠好嗎？」
  • 高 H 數的人：善於自我修復，但可能過度內省
  • 低 H 數的人：行動力強，但容易忽略自己的感受
  • 覺察師要注意的：案主的 H 數 → 決定你用什麼語氣開場

**O數（識數）— 你怎麼理解世界**

  • 對應 7 宮心門宮
  • 核心提問：「我值得被愛嗎？」
  • 高 O 數：直覺強，容易「感覺到」但說不出來
  • 低 O 數：邏輯清晰，但可能忽略感覺層的訊號
  • 覺察師要注意的：O 數高的案主要多用「感受」語言，低的要多用「分析」語言

**U數（鑰數）— 你的潛能在哪**

  • 對應 10 宮志氣殿
  • 核心提問：「我配擁有嗎？」
  • 這個數字跟「值得感」直接相關
  • 很多案主卡住的根源在這裡：不是能力不夠，是覺得自己不配

**R數（行數）— 你會怎麼行動**

  • 對應 12 宮平衡室
  • 核心提問：「我為何而來？」
  • 注意：「啟數」已廢止，統一叫「行數」
  • 這個數字決定案主「聽完覺察之後會不會真的去做」

第三段：L.I.G.H.T. 五能量指數（5 分鐘）

第一軌（日常五力）：

  • L 愛之殿 → 愛與被愛的能力
  • I 直覺閣 → 直覺敏銳度
  • G 磐石廳 → 穩定度與安全感
  • H 和諧苑 → 人際和諧
  • T 蛻變室 → 轉化與突破的能力

覺察師實戰用法：

「五力像一張雷達圖。哪個特別低，就是案主現在最需要被看見的地方。不要急著補，先讓他看見。」

第四段：實戰 — 幫自己做一份迷你秘碼報告（8 分鐘）

帶學員寫出自己的迷你報告，格式：

【我的馥靈秘碼】
H數 ___（癒數）：我安頓自己的方式是 ___
O數 ___（識數）：我理解世界的方式是 ___
U數 ___（鑰數）：我最大的潛能在 ___
R數 ___（行數）：我的行動模式是 ___

五力中最強的是 ___，最需要覺察的是 ___

一句話座標：我是一個 ___（H 特質）的人，用 ___（O 特質）的方式理解世界，
潛能在 ___（U 特質），行動時 ___（R 特質）。

覺察師不需要精通全部 33 套。你需要的是：

  • 八字基礎（案主最常問）
  • 馥靈秘碼（你的核心工具）
  • 西洋占星基本概念（太陽星座 + 月亮星座）
  • 人類圖基本概念（類型 + 策略）

其他 29 套，交給 destiny-engine.html 自動算。你的工作是「翻譯」，不是「計算」。

覺察師的語感練習

品牌聲音的核心（品牌聖經 p.2-3）：

  • 口氣是淡的。不用力、不激動、不推銷
  • 長句和短句要交叉
  • 廢話要留一點（真人說話不是只說重點）
  • 比喻從生活裡撿（手機當機、Google Maps 帶錯路）

禁用詞（永遠不能用）：

  • 「你不是XXX，是XXX」— 全面禁止
  • 「首先」「其次」「最後」「總之」
  • 療癒 → 用「舒緩」；頻率 → 用「節奏」；靈魂 → 用「內在」
