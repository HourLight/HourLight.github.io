兩條高速公路：皮膚 + 鼻子

精油進入身體，主要走兩條路。不是隨便抹了就直接跑到你的肝臟，它有自己的路線圖。

第一條：經皮吸收 — 塗抹、按摩、泡澡

  • 精油分子很小（分子量通常低於 500 道爾頓），可以穿透皮膚
  • 經過皮膚三層 → 進入微血管 → 隨血液循環全身
  • 大約 20-70 分鐘可以在血液中偵測到精油成分

第二條：嗅覺吸收 — 擴香、直接嗅聞、蒸氣吸入

  • 氣味分子 → 鼻腔頂端嗅覺上皮 → 嗅覺神經 → 直達大腦邊緣系統
  • 速度非常快，幾秒鐘內就能影響情緒和自律神經

還有第三條比較少用的：口服。但 NAHA 不建議一般芳療師執行口服，那是臨床芳療師的範疇。初階考試記住這點就好。

皮膚吸收：三層關卡

你的皮膚不是一片保鮮膜，它是三層結構，每層都有自己的工作：

1. 表皮（Epidermis）

  • 最外面那層，像城牆一樣擋住外來物質
  • 最外面的角質層（stratum corneum）是精油要突破的第一關
  • 角質層是死掉的細胞 + 脂質排列成的磚牆結構，精油因為是脂溶性的，所以可以「溶進去」

2. 真皮（Dermis）

  • 有血管、淋巴管、神經末梢、毛囊、汗腺
  • 精油穿過表皮後在這裡被微血管吸收，進入血液循環

3. 皮下組織（Hypodermis）

  • 脂肪層，保溫和緩衝用的
  • 精油通常不需要到這麼深，在真皮層就已經被吸收了

為什麼需要基底油？

精油可以穿透皮膚，那為什麼不直接抹原液？

因為精油濃度太高了。就像你不會直接喝一整瓶濃縮果汁，你會加水稀釋。

基底油（carrier oil）的三個功能：

  • 稀釋 — 降低刺激性，保護皮膚
  • 延展 — 讓你可以在更大面積的皮膚上按摩
  • 減緩揮發 — 精油不會一下子全蒸發掉，有更多時間被皮膚吸收

有些人說「基底油會阻擋精油吸收」，這是錯的。基底油本身也是脂溶性的，它跟精油一起慢慢滲透，反而讓吸收更均勻。

「基底油不是稀釋劑，是保護劑。它讓精油用一個皮膚可以接受的速度慢慢進去，而不是一次全衝進來把你的皮膚嚇到。」

鼻子那條路：嗅覺 → 大腦

這條路線是芳療最神奇的部分，因為它快到你來不及思考。

路線圖：

  • 氣味分子飄進鼻腔
  • → 到達鼻腔最頂端的嗅覺上皮（只有一張郵票大小）
  • → 嗅覺受體神經元被觸發
  • → 信號傳到嗅球（olfactory bulb）
  • → 直接進入邊緣系統（limbic system）

邊緣系統裡面有兩個關鍵角色：

  • 杏仁核（amygdala） — 管情緒反應：害怕、開心、噁心、安心
  • 海馬迴（hippocampus） — 管記憶：這個味道讓你想起誰？什麼場景？

嗅覺是人類五感中唯一不經過丘腦（thalamus）中繼站的感官。視覺、聽覺、觸覺都要先經過丘腦「翻譯」再送到大腦皮質處理。但嗅覺直接衝進邊緣系統，所以你聞到一個味道的情緒反應，比你「理解」那是什麼味道還快。

聞到味道為什麼會想起某個人？

你有沒有這種經驗：走在路上突然聞到一個味道，瞬間被拉回十年前的某個場景？不是「想起來」，是整個人被丟回去，連當時的感覺都原封不動。

這就是嗅覺記憶的威力。因為嗅覺信號直接進入管情緒和記憶的邊緣系統，所以氣味和情緒記憶的連結特別深、特別牢固。

「這也是認知芳療的核心概念。你不需要手邊有精油，光是『想到』薄荷的味道，大腦就已經在啟動那個沁涼的神經迴路了。想到酸梅，嘴裡會自動分泌唾液。你的身體不區分『真的聞到』和『想到那個味道』，它都會回應。」

這也是為什麼馥靈之鑰的牌卡系統可以在線上運作。牌卡上寫著精油名稱，你看到「薰衣草」三個字，你的嗅覺記憶就已經被觸發了。覺察在那個瞬間就真實發生了，不需要精油瓶子在你面前。

人體八大系統：精油能幫什麼？

精油進入身體後，會跟著血液循環跑到各個系統。不同的精油成分對不同系統有不同的親和力。

精油怎麼離開身體？

精油不會永遠留在你體內。它們被代謝後，會透過四個出口離開：

  • 肝臟 — 主要代謝器官。精油成分在肝臟被分解成水溶性代謝物
  • 腎臟 — 代謝後的產物隨尿液排出（這是最主要的排出途徑）
  • 肺臟 — 一部分揮發性成分會隨呼氣排出（你有沒有吃大蒜後呼吸有味道？精油也一樣）
  • 皮膚 — 少量隨汗液排出

代謝時間：大多數精油成分在 24-72 小時內會被完全代謝排出。

「這也是為什麼肝腎功能不好的人使用精油要特別謹慎，因為代謝變慢，等於精油在體內停留的時間更長、濃度更高。不是不能用，是要減量、拉長間隔。」

考試重點提醒

NAHA 初階考試在這個單元，你需要記住：

  • ✓ 兩條主要吸收途徑：經皮吸收和嗅覺吸收
  • ✓ 皮膚三層名稱：表皮、真皮、皮下組織
  • ✓ 嗅覺路徑：鼻腔 → 嗅覺上皮 → 嗅球 → 邊緣系統（杏仁核 + 海馬迴）
  • ✓ 嗅覺是唯一不經過丘腦的感官 — 這是考試最愛出的送分題
  • ✓ 精油代謝途徑：肝臟代謝 → 腎臟（尿液）排出為主
  • ✓ 基底油的功能：稀釋、延展、減緩揮發
  • ✓ 八大系統各對應哪些精油（至少記 2 個代表精油）
「記住嗅覺那條路線，不只是為了考試。當你理解為什麼聞一個味道就能改變心情，你才能跟案主解釋：這不是安慰劑效應，這是你的大腦真的在回應。」
