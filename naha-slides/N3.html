N3｜20 種核心精油：記住它們像記住 20 個朋友

每支精油都有自己的個性、脾氣、和最擅長的事
馥靈之鑰 Hour Light ✦ 讀懂自己，活對人生

怎麼認識 20 個朋友

背精油資料很痛苦，跟背單字一樣。但如果你把每支精油當成一個人，記住她的個性、專長、還有她的地雷，就容易很多。

接下來的 20 張投影片，每一支精油我都會給你：

  • 她的「身分證」— 拉丁學名、萃取部位、萃取方法
  • 她的「關鍵成分」— 知道裡面有什麼，才知道為什麼有效
  • 她最擅長的三件事
  • 她的地雷（安全注意事項）
  • 一句話記住她

不用一次全記住。先挑 5 支你最有感覺的，把它們記熟。然後每週加 3 支。一個多月你就全部認識了。

「認識精油跟認識人一樣 — 不是讀完履歷就算認識。你要真的跟她相處過，聞過她、用過她，才算。」

檸檬 Lemon

Citrus limon

科屬：芸香科

萃取：冷壓法 / 果皮

主要成分：檸檬烯 Limonene（60-70%）

氣味：明亮、乾淨、清新，像剛切開的檸檬片

三大用途：

  • 提振精神、提升專注力
  • 淨化空氣、居家清潔
  • 支持消化系統舒適

安全：光敏性 — 塗抹後 12 小時避免日曬

一句話：「早晨第一杯咖啡旁邊的那道陽光。」

甜橙 Sweet Orange

Citrus sinensis

科屬：芸香科

萃取：冷壓法 / 果皮

主要成分：檸檬烯 Limonene（90%+）

氣味：甜暖、圓潤、讓人想微笑的味道

三大用途：

  • 舒緩緊張情緒、幫助放鬆
  • 支持消化、緩解腹部不適感
  • 兒童友善精油首選

安全：光敏性風險低（比檸檬低很多），但仍建議注意。保質期短，約 6-12 個月

一句話：「那個不管什麼場合都能讓氣氛變好的朋友。」

佛手柑 Bergamot

Citrus bergamia

科屬：芸香科

萃取：冷壓法 / 果皮

主要成分：乙酸沉香酯 Linalyl Acetate、檸檬烯 Limonene、呋喃香豆素

氣味：柑橘但不甜膩，帶一點花香的優雅感。伯爵茶的味道就是她

三大用途：

  • 舒緩焦慮與低落情緒（柑橘類裡情緒支持最強的）
  • 支持皮膚油脂平衡
  • 幫助入睡前放鬆

安全：高光敏性（呋喃香豆素） — 塗抹後 12-18 小時避免日曬。可選無呋喃香豆素版本（FCF）

一句話：「穿著高跟鞋的柑橘，優雅但有態度。」

檸檬香茅 Lemongrass

Cymbopogon citratus

科屬：禾本科

萃取：蒸餾法 / 葉

主要成分：檸檬醛 Citral（70-80%）

氣味：強烈的檸檬草香，帶一點泥土味。泰式料理的香氣你就知道了

三大用途：

  • 驅蟲（蚊子很討厭這個味道）
  • 舒緩肌肉緊繃不適
  • 淨化空間、去除異味

安全：皮膚刺激性高，務必稀釋（建議 1% 以下）。避免用於敏感肌膚或幼兒

一句話：「夏天廚房裡的除蟲大隊長。」

真正薰衣草 Lavender

Lavandula angustifolia

科屬：唇形科

萃取：蒸餾法 / 花

主要成分：沉香醇 Linalool（25-45%）、乙酸沉香酯 Linalyl Acetate（25-47%）

氣味：花香帶草本，柔和、乾淨，像剛洗好的被單曬了太陽

三大用途：

  • 助眠、舒緩壓力（研究最多的精油）
  • 支持皮膚修護（Gattefossé 的那隻手）
  • 舒緩頭部不適

安全：少數可以低濃度直接接觸皮膚的精油之一。極少過敏案例，但仍建議稀釋。注意不要跟穗花薰衣草（L. latifolia）混淆

一句話：「芳療界的萬用瑞士刀，什麼都能做一點。」

羅馬洋甘菊 Roman Chamomile

Chamaemelum nobile（舊名 Anthemis nobilis）

科屬：菊科

萃取：蒸餾法 / 花

主要成分：酯類（歐白芷酸異丁酯等，80%+）

氣味：蘋果般的甜香，溫暖、柔軟。有人說像秋天午後的蘋果園

三大用途：

  • 舒緩焦慮與失眠（比薰衣草更「安定」）
  • 支持敏感肌膚舒適
  • 兒童與嬰幼兒友善（稀釋後）

安全：非常溫和。菊科過敏者少數可能反應，但機率極低。價格偏高（產油率低）

一句話：「那個說話很輕、但每次都說到你心裡的朋友。」

快樂鼠尾草 Clary Sage

Salvia sclarea

科屬：唇形科

萃取：蒸餾法 / 花、葉

主要成分：乙酸沉香酯 Linalyl Acetate（高達 75%）、沉香醇 Linalool

氣味：草本帶堅果味，溫暖微甜，有人覺得像麝香

三大用途：

  • 舒緩女性週期不適
  • 放鬆神經緊繃、支持情緒平穩
  • 支持頭皮與毛髮健康

安全：孕婦避免使用。不要跟酒精同時使用（可能增強酒醉感）。注意不要跟「鼠尾草」（Salvia officinalis）搞混 — 那支含大量側柏酮，毒性高很多

一句話：「女生那幾天最想打電話的閨蜜。」

天竺葵 Geranium

Pelargonium graveolens

科屬：牻牛兒苗科

萃取：蒸餾法 / 葉、莖

主要成分：香茅醇 Citronellol、甲酸香茅酯 Citronellyl Formate、香葉醇 Geraniol

氣味：玫瑰感但帶草本，比玫瑰清爽，不那麼甜

三大用途：

  • 支持皮膚油水平衡（油性和乾性都適用）
  • 舒緩情緒波動
  • 驅蟲（跟檸檬香茅一樣有效）

安全：整體溫和。極少數人對香葉醇過敏。常被當作「平價玫瑰替代品」，但她有自己的個性，不只是替身

一句話：「買不起玫瑰的時候找她，但她不是替身，是另一種美。」

依蘭 Ylang Ylang

Cananga odorata

科屬：番荔枝科

萃取：蒸餾法 / 花（分四級：Extra, I, II, III）

主要成分：沉香醇 Linalool、乙酸芐酯 Benzyl Acetate、金合歡烯 Farnesene

氣味：濃郁的熱帶花香，甜到有點膩。有人愛死有人受不了，沒有中間值

三大用途：

  • 舒緩緊張情緒、支持心血管放鬆
  • 護髮（印尼傳統用法）
  • 增添浪漫氛圍（知名催情精油）

安全：用量要少，過量會頭痛或反胃。建議先從 0.5% 開始。敏感肌膚稀釋使用

一句話：「約會之夜的秘密武器，但記得少量就好。」

玫瑰 Rose

Rosa damascena（大馬士革玫瑰）

科屬：薔薇科

萃取：蒸餾法（精油）或溶劑萃取（原精）/ 花瓣

主要成分：香茅醇 Citronellol（35-55%）、香葉醇 Geraniol、橙花醇 Nerol

氣味：深邃、多層次的花香。蒸餾版比較清透，原精版比較濃甜。約 3000-5000 公斤花瓣才出 1 公斤精油

三大用途：

  • 支持情緒療癒（失落、悲傷、心碎）
  • 支持肌膚抗老化、保濕
  • 舒緩女性週期和更年期不適

安全：非常溫和，但孕婦前三個月避免。價格極高（純精油約 NT$3,000-8,000/ml）。買到便宜的 99% 是假的

一句話：「精油界的愛馬仕，貴是有道理的。」

薄荷 Peppermint

Mentha × piperita

科屬：唇形科

萃取：蒸餾法 / 葉

主要成分：薄荷醇 Menthol（30-50%）、薄荷酮 Menthone

氣味：清涼、穿透力強，像嚼了一片口香糖再深呼吸

三大用途：

  • 提神醒腦、支持專注力
  • 舒緩頭部不適（太陽穴塗抹）
  • 支持呼吸道暢通

安全：6 歲以下兒童禁用（薄荷醇可能引起呼吸抑制）。不要用在臉部大面積塗抹。不要跟順勢療法同時使用（可能干擾）。哺乳期避免（可能減少泌乳）

一句話：「下午三點在辦公室快睡著時，她會把你搖醒。」

尤加利 Eucalyptus

Eucalyptus globulus（藍膠尤加利）

科屬：桃金孃科

萃取：蒸餾法 / 葉

主要成分：1,8-桉油醇 1,8-Cineole（60-85%）

氣味：清新、穿透、有一點樟腦味。感冒時聞到會覺得鼻子突然通了

三大用途：

  • 支持呼吸道舒暢（鼻塞、咳嗽期間蒸氣吸入）
  • 舒緩肌肉關節不適
  • 淨化空氣、抗菌

安全：10 歲以下兒童建議用澳洲尤加利（E. radiata）替代，1,8-桉油醇含量較低、較溫和。不可口服。高濃度可能刺激皮膚

一句話：「感冒季節的護衛隊隊長。」

茶樹 Tea Tree

Melaleuca alternifolia

科屬：桃金孃科

萃取：蒸餾法 / 葉

主要成分：萜品烯-4-醇 Terpinen-4-ol（30-48%）、γ-萜品烯

氣味：藥草味、乾淨、有一點辛辣。不是那種「好聞」的油，但非常實用

三大用途：

  • 支持皮膚淨化（痘痘、小傷口清潔）
  • 抗菌、抗黴（居家清潔好幫手）
  • 支持免疫系統

安全：少數可低濃度點塗的精油之一，但建議還是稀釋。貓狗敏感 — 有寵物的家庭擴香要注意通風。氧化後刺激性增加，過期的直接丟掉

一句話：「不好看但超級能幹的工具人朋友。」

迷迭香 Rosemary

Rosmarinus officinalis（或 Salvia rosmarinus）

科屬：唇形科

萃取：蒸餾法 / 葉、花

主要成分：依化學品種不同 — 桉油醇型（1,8-Cineole）/ 樟腦型（Camphor）/ 馬鞭草酮型（Verbenone）

氣味：草本、清新、略帶樟腦的穿透感。烤肉時加的那個香料就是她

三大用途：

  • 提升記憶力與專注力（研究顯示嗅聞可改善短期記憶）
  • 支持頭皮健康、促進髮絲豐盈
  • 舒緩肌肉疲勞

安全：癲癇患者禁用（樟腦成分可能誘發）。孕婦避免。高血壓患者謹慎使用。考試常考三種化學品種的差異

一句話：「考試前聞一聞，大腦會感謝你。」

乳香 Frankincense

Boswellia carterii（或 B. sacra）

科屬：橄欖科

萃取：蒸餾法 / 樹脂

主要成分：α-蒎烯 α-Pinene、檸檬烯 Limonene、乙酸辛酯

氣味：溫暖的樹脂香，帶一點柑橘和辛香。深呼吸一口會自動想閉眼

三大用途：

  • 支持冥想與靜心（數千年的寺院傳統）
  • 支持肌膚修護與抗老化
  • 舒緩呼吸道不適

安全：非常溫和，幾乎沒有已知禁忌。東方三賢士帶給耶穌的禮物之一。價格中高，品質差異大

一句話：「聞了會自動安靜下來的那種存在感。」

沒藥 Myrrh（補充精油）

Commiphora myrrha

科屬：橄欖科

萃取：蒸餾法 / 樹脂

主要成分：倍半萜烯類（呋喃桉二烯 Furanoeudesma-1,3-diene 等）

氣味：深沉、溫暖、帶苦味的樹脂香。比乳香更沉、更接地

三大用途：

  • 支持口腔健康（漱口水配方常見）
  • 支持皮膚修護（乾裂、龜裂肌膚）
  • 支持情緒沉澱（跟乳香搭配是經典組合）

安全：孕婦禁用（可能促進子宮收縮）。質地黏稠、流動性差，正常現象。羅馬士兵上戰場帶的就是她

一句話：「乳香的沉默搭檔，一個向上一個向下，合在一起最完整。」

絲柏 Cypress

Cupressus sempervirens

科屬：柏科

萃取：蒸餾法 / 葉、嫩枝

主要成分：α-蒎烯 α-Pinene、δ-3-蒈烯 δ-3-Carene

氣味：清新的木質香，像走進一片針葉林。乾淨、挺拔

三大用途：

  • 支持循環系統舒適（腿部沉重感）
  • 收斂、調節油脂分泌
  • 支持呼吸道暢通

安全：整體溫和。孕婦避免。不要跟藍絲柏（Callitris intratropica）搞混，那是完全不同的品種

一句話：「森林裡那棵永遠站得最直的樹。」

雪松 Cedarwood

Cedrus atlantica（大西洋雪松）

科屬：松科

萃取：蒸餾法 / 木頭

主要成分：大西洋酮 Atlantone、雪松烯 Cedrene

氣味：溫暖、乾燥的木頭香。像打開一個老木箱，或走進日式木屋

三大用途：

  • 助眠、支持深度放鬆
  • 支持頭皮健康、舒緩頭皮屑
  • 驅蟲（衣櫃防蛀的天然選擇）

安全：孕婦避免。注意品種：大西洋雪松（Cedrus atlantica）跟維吉尼亞雪松（Juniperus virginiana）是不同科的植物，成分差異大

一句話：「冬天夜晚窩在木頭小屋裡的安心感。」

甜馬鬱蘭 Sweet Marjoram

Origanum majorana

科屬：唇形科

萃取：蒸餾法 / 葉

主要成分：萜品烯-4-醇 Terpinen-4-ol、γ-萜品烯、沉香醇 Linalool

氣味：溫暖的草本香，帶一點甜和辛。像廚房裡正在燉湯的味道

三大用途：

  • 舒緩肌肉痙攣與緊繃
  • 支持消化系統舒適
  • 助眠、舒緩神經緊張

安全：孕婦避免。不要跟野馬鬱蘭 / 牛至（Origanum vulgare）搞混 — 牛至精油的酚含量極高，刺激性很強

一句話：「像一碗熱湯一樣，從裡面暖到外面。」

肉桂 Cinnamon

Cinnamomum verum（錫蘭肉桂）/ C. cassia（中國肉桂）

科屬：樟科

萃取：蒸餾法 / 樹皮（Bark）或葉（Leaf）

主要成分：肉桂皮 → 肉桂醛 Cinnamaldehyde（高達 90%）；肉桂葉 → 丁香酚 Eugenol（70-85%）

氣味：溫暖、辛甜、讓人想到聖誕節和蘋果派

三大用途：

  • 溫暖身體、支持循環
  • 抗菌力強（居家淨化擴香）
  • 提振低落情緒

安全：皮膚刺激性極高 — 務必高度稀釋（0.5% 以下）。肉桂皮精油比肉桂葉精油刺激性更強。孕婦禁用。不建議泡澡使用。抗凝血藥物使用者避免

一句話：「脾氣火爆但心地善良，保持距離就是最好的相處方式。」

薑 Ginger

Zingiber officinale

科屬：薑科

萃取：蒸餾法 / 根莖

主要成分：薑烯 Zingiberene、β-倍半水芹烯 β-Sesquiphellandrene

氣味：溫暖、辛辣、帶一點檸檬調。比鮮薑柔和很多，不嗆

三大用途：

  • 舒緩噁心感（暈車暈船、孕吐）
  • 溫暖身體、舒緩寒冷不適
  • 舒緩肌肉與關節僵硬

安全：整體安全。高濃度可能刺激敏感皮膚，建議稀釋至 1% 以下。抗凝血藥物使用者注意。是少數孕婦可以謹慎使用的精油（低濃度舒緩孕吐）

一句話：「冬天泡完腳那種從腳底暖上來的感覺，就是她。」
