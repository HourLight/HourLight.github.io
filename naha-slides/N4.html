為什麼要學化學？

先深呼吸。這堂不是要你回到高中化學課那個噩夢。

學精油化學只有一個目的：搞懂這瓶油「為什麼」有效，以及「什麼時候」不該用。

你去藥局買感冒藥，藥盒背面會寫成分。你不需要當藥劑師，但你至少要知道哪些成分會讓你想睡覺、哪些不能跟酒一起吃。精油也一樣。

「知道成分，不是為了變成化學家，是為了不害到人。」

NAHA 考試會考官能基的安全性和適用情境，不會叫你畫分子結構圖，放心。

異戊二烯：精油世界的樂高積木

精油的化學成分，幾乎都是用一個叫「異戊二烯（isoprene）」的小單位拼出來的。

你可以把它想成樂高的基礎磚塊。一塊磚 = 5 個碳原子（C5）。

  • 2 塊磚拼在一起（C10） → 單萜類
  • 3 塊磚拼在一起（C15） → 倍半萜類
  • 4 塊磚拼在一起（C20） → 雙萜類（精油裡比較少見）

磚塊越少，分子越小、越輕、揮發越快。磚塊越多，分子越大、越沉、留香越久。

然後在這些磚塊上面接不同的「零件」（官能基），就會產生不同的功能和安全性。接了 -OH 變成醇，接了 =O 變成醛或酮，接了 -O- 變成氧化物。

「骨架決定大小，零件決定個性。」

單萜烯（C10）：衝第一的小個子

2 塊樂高磚，分子最小、最輕、跑最快。打開瓶蓋你最先聞到的，通常就是它。

代表精油：

  • 檸檬（d-limonene）、甜橙、葡萄柚 — 柑橘家族幾乎都是
  • 黑胡椒、絲柏、杜松漿果
  • 乳香裡也有大量單萜烯（α-pinene）

特性：

  • 提振、淨化空間、促進循環
  • 抗菌效果中等，但擴香效果好

要注意的事：

「單萜烯氧化很快。開封後 6-12 個月就開始變質，變質後會刺激皮膚。你家那瓶開了兩年的檸檬精油？拿來擴香就好，不要再塗皮膚了。」

倍半萜烯（C15）：沉穩的大哥

3 塊樂高磚，分子比較大，揮發慢，留香久。性格也比較沉穩，不會一開瓶就衝出來。

代表精油：

  • 依蘭依蘭、薑、廣藿香、岩蘭草
  • 德國洋甘菊（天藍烴 chamazulene，藍色的就是它）
  • 檀香裡也有倍半萜醇（santalol）

特性：

  • 抗發炎、抗過敏、鎮靜
  • 比單萜烯穩定，不容易氧化
  • 很多「後調」香氣就是倍半萜在撐場

你有沒有聞過一瓶精油，剛打開覺得味道很淡，但抹在手腕上半小時後反而越來越香？那通常就是倍半萜在慢慢釋放。它不急，因為它知道自己會留到最後。

單萜醇（-OH）：溫柔的抗菌王

在單萜骨架上接了一個 -OH（氫氧基），性格就變溫柔了。像是一個很會照顧人的護理師，抗菌能力強，但對皮膚很友善。

代表精油：

  • 茶樹（萜品烯-4-醇 terpinen-4-ol）
  • 真正薰衣草（沉香醇 linalool）
  • 天竺葵（香茅醇 citronellol）
  • 玫瑰草、花梨木

特性：

  • 抗菌、抗病毒、抗真菌 — 三棲選手
  • 皮膚刺激性低，兒童和老人都相對安全
  • 免疫調節、溫和提振
「如果你只能記一個官能基，記單萜醇。它是芳療師最常用、最安全、最百搭的化學家族。茶樹和薰衣草能成為國民精油，不是沒有原因的。」

酯類：安撫系擔當

酯 = 酸 + 醇反應的產物。名字裡通常有「某某酸某某酯」，聽起來很複雜，但你只需要記住一件事：酯類 = 放鬆。

代表精油：

  • 真正薰衣草（乙酸沉香酯 linalyl acetate）— 含量越高越放鬆
  • 快樂鼠尾草（乙酸沉香酯也很高）
  • 羅馬洋甘菊（酯類含量超過 80%，放鬆之王）
  • 佛手柑、苦橙葉

特性：

  • 抗痙攣、止痛、鎮靜、抗發炎
  • 皮膚刺激性非常低
  • 聞起來通常很舒服，帶果香或花香

想像一下，你加班到晚上十點回到家，點了一盞薰衣草擴香。那個讓你肩膀慢慢放下來的感覺，有很大一部分是乙酸沉香酯在幫你的副交感神經按下「可以休息了」的按鈕。

醛類：穿透力強的雙面刃

醛類的味道很「衝」，穿透力強，你一聞就知道是它。有點像那種講話很直的朋友，說的都對，但有時候會刺到你。

代表精油：

  • 檸檬香茅（香茅醛 citronellal、橙花醛 neral）
  • 山雞椒（檸檬醛 citral 含量很高）
  • 香蜂草（melissa）

特性：

  • 抗真菌效果強（香港腳那種）
  • 消炎、降血壓
  • 驅蟲效果好（檸檬香茅防蚊就是這個原理）

要注意的事：

「醛類對皮膚和黏膜有刺激性。用在皮膚上一定要充分稀釋（1% 以下），敏感肌更要小心。擴香沒問題，但別拿純的檸檬香茅往身上抹。」

酮類：威力強大但要小心

酮類是精油化學裡的「高濃度藥物」。效果很強，但劑量錯了就會出事。就像止痛藥，吃一顆救你，吃一把害你。

代表精油：

  • 薄荷（薄荷酮 menthone）
  • 迷迭香 CT 樟腦型（樟腦 camphor）
  • 大西洋雪松（大西洋酮 atlantone）
  • 鼠尾草（側柏酮 thujone — 這個最危險）

特性：

  • 化痰、溶解黏液 — 鼻塞時超有感
  • 促進細胞再生、傷口癒合
  • 抗凝血

危險性：

「高劑量酮類有神經毒性，可能引發癲癇。孕婦、嬰幼兒、癲癇患者絕對避免使用高酮類精油。這是 NAHA 考試必考的紅線。」

氧化物：呼吸道的好朋友

氧化物家族裡最出名的成員就是 1,8-桉油醇（1,8-cineole）。你感冒鼻塞的時候聞尤加利會覺得鼻子瞬間通了，就是它的功勞。

代表精油：

  • 尤加利（澳洲尤加利 1,8-cineole 含量可達 85%）
  • 茶樹（也含一些）
  • 迷迭香 CT 桉油醇型
  • 綠花白千層、月桂

特性：

  • 祛痰、化黏液、打開呼吸道
  • 輕度抗菌、抗病毒
  • 提振精神、幫助專注

注意：2 歲以下幼兒避免使用含高量 1,8-cineole 的精油。小小孩的呼吸道很敏感，太強的氧化物可能讓他們呼吸急促。這不是嚇你，是基本功。

酚類：最強抗菌但最刺激

如果單萜醇是溫柔的護理師，酚類就是動手術的外科醫生。殺菌力最強，但也最傷組織。

代表精油：

  • 丁香（丁香酚 eugenol）
  • 百里香 CT 百里酚型（thymol）
  • 肉桂皮（肉桂醛其實是醛類，但肉桂葉含丁香酚）
  • 野馬鬱蘭 / 奧勒岡（香荊芥酚 carvacrol）

特性：

  • 最強的抗菌、抗病毒、抗真菌能力
  • 免疫刺激、暖身

危險性：

「酚類對皮膚和肝臟都有刺激性。塗抹一定要高度稀釋（0.5-1%），不能長期連續使用。丁香精油直接滴在牙齦上止牙痛？可以急用一次，但不能天天這樣搞。」

官能基 → 安全性 → 稀釋比例

安全性由高到低：

  • 酯類 — 最溫和，幾乎不刺激
  • 單萜醇 — 溫和，兒童老人可用
  • 氧化物 — 中等，注意幼兒
  • 單萜烯 — 注意氧化變質
  • 倍半萜烯 — 通常安全
  • 醛類 — 需充分稀釋
  • 酮類 — 高劑量有神經毒性
  • 酚類 — 最刺激，短期低劑量

速記口訣：

「酯醇溫柔隨便用，氧萜中間看情況，醛酮酚類要小心。」

考試核心邏輯：

越溫和的 → 稀釋比例可以高一點（2-3%）

越刺激的 → 稀釋比例必須低（0.5-1%）

有毒性疑慮的 → 特定族群完全避免

考試重點提醒

NAHA 初階考試在化學這塊，不會叫你畫分子式。但你一定要知道：

  • ✓ 每個官能基的主要功效和安全注意事項
  • ✓ 酮類的神經毒性、酚類的皮膚刺激性 — 必考紅線
  • ✓ 單萜烯氧化後會怎樣（皮膚刺激、不能塗抹）
  • ✓ 至少能舉出每個官能基 2-3 個代表精油
  • ✓ 異戊二烯是基本結構單位（C5），單萜 C10、倍半萜 C15
「化學不是背的，是理解的。你只要記住：骨架決定大小，零件決定個性，個性決定安全規則。這樣就夠了。」
